Reward Points Activation

Reward Points Activation in EquiBillBook Settings allows you to enable and configure the reward points system for customer loyalty programs. This feature helps you incentivize repeat customers and build customer loyalty through points-based rewards. This guide covers how to activate and set up the reward points system.

What is Reward Points System?

The Reward Points System enables:

  • Points Earning: Customers earn points on purchases
  • Points Redemption: Customers redeem points for discounts or rewards
  • Points Tracking: Track customer points balance and history
  • Loyalty Programs: Build customer loyalty through rewards
  • Points Expiry: Configure points expiration rules
  • Points Reports: Analyze points earning and redemption

Accessing Reward Points Settings

To activate the reward points system:

  1. Navigate to Settings from the main menu
  2. Click on Reward Points Settings or find it in Other Settings
  3. Look for "Reward Points Activation" or "Enable Reward Points"
  4. Toggle the activation switch

Activation Steps

Step 1: Check Feature Availability

Before activation, ensure:

  • Your subscription plan includes Reward Points feature
  • You have administrative privileges
  • Sales module is activated (points are earned on sales)

Step 2: Activate Reward Points

To activate:

  1. Go to Settings → Reward Points Settings
  2. Find the "Reward Points System" section
  3. Toggle "Enable Reward Points" to ON
  4. Click "Save" or "Update Settings"
  5. Wait for confirmation that the system has been activated

Step 3: Verify Activation

After activation, verify that:

  • Reward Points Settings page shows all configuration options
  • Points earning rules can be configured
  • Points redemption rules are available
  • Customer points balance can be viewed

Initial Configuration

Basic Settings

After activation, configure basic settings:

  • Points Name: What to call points (e.g., "Points", "Rewards", "Stars")
  • Points Symbol: Symbol or abbreviation for points
  • Points Currency: Currency for points calculations
  • Decimal Places: Number of decimal places for points

Points Account

  • Points Account: Select account for tracking points liability
  • Account Type: Liability or equity account for points
  • Auto-reconciliation: Automatically reconcile points transactions

Post-Activation Setup

1. Configure Points Earning Rules

  • Set how customers earn points
  • Define earning rates (points per currency unit)
  • Configure earning conditions
  • Set minimum purchase requirements

2. Configure Points Redemption Rules

  • Set redemption rates (points per currency unit)
  • Define minimum redemption amounts
  • Configure maximum redemption limits
  • Set redemption conditions

3. Configure Points Expiry

  • Enable or disable points expiration
  • Set expiry periods
  • Configure expiry rules
  • Set expiry notifications

4. Test the System

  • Create test customer
  • Process test sale to earn points
  • Test points redemption
  • Verify points balance updates

Deactivating Reward Points

If you need to deactivate the reward points system:

  1. Go to Settings → Reward Points Settings
  2. Toggle "Enable Reward Points" to OFF
  3. Confirm deactivation when prompted
  4. Note: Existing points balances will be preserved

Warning: Deactivating the system will:

  • Stop new points from being earned
  • Prevent points redemption (unless you allow it)
  • Hide points-related features from customers
  • Preserve existing points balances and history

Best Practices

  • Activate Before Sales: Activate before processing sales to start earning points
  • Clear Communication: Clearly communicate points program to customers
  • Test Thoroughly: Test earning and redemption before going live
  • Set Reasonable Rates: Set earning and redemption rates that are sustainable
  • Monitor Usage: Monitor points usage and adjust rules as needed
  • Customer Education: Educate customers about the rewards program

Troubleshooting

Reward Points option not visible?

  • Check if your subscription plan includes Reward Points feature
  • Verify you have administrative access
  • Contact EquiBillBook support if the option should be available

System activated but points not earning?

  • Verify points earning rules are configured
  • Check if sales are being processed correctly
  • Ensure customers are eligible for points
  • Review points earning conditions

Cannot activate system?

  • Verify your subscription plan includes Reward Points
  • Check for any pending payments or subscription issues
  • Ensure Sales module is activated
  • Contact support for assistance
