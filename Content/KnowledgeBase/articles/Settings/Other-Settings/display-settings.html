Display Settings

Display settings allow you to customize how information is displayed throughout EquiBillBook. These settings control the visual presentation of data, reports, and interface elements to match your preferences and improve readability.

What are Display Settings?

Display settings include:

  • Table Display: How data tables are displayed
  • Report Formatting: Formatting options for reports
  • Number Formatting: How numbers are displayed
  • Date Display: Date format and display options
  • Column Visibility: Which columns to show by default
  • Row Height: Row spacing and density
  • Color Coding: Color schemes for status and categories

Accessing Display Settings

To configure display settings:

  1. Navigate to Settings → Other Settings → Display Settings
  2. You'll find various display customization options
  3. Configure settings based on your viewing preferences

Key Configuration Areas

1. Table Display Settings

Configure how data tables appear:

  • Default Columns: Which columns to show by default
  • Column Width: Default column widths
  • Row Height: Compact, normal, or comfortable spacing
  • Alternating Row Colors: Enable zebra striping for readability
  • Show Row Numbers: Display row numbers
  • Sticky Headers: Keep column headers visible while scrolling
  • Sort Indicators: Show sort direction indicators

2. Report Display Settings

Formatting for reports:

  • Report Layout: Portrait or landscape orientation
  • Page Size: A4, Letter, Legal, or custom
  • Margins: Page margin settings
  • Header/Footer: Show or hide headers and footers
  • Grid Lines: Show or hide grid lines
  • Font Size: Default font size for reports
  • Color vs Grayscale: Color or black and white printing

3. Number Formatting

How numbers are displayed:

  • Decimal Places: Number of decimal places (0, 2, or custom)
  • Thousand Separator: Comma, period, space, or none
  • Negative Numbers: How to display negative numbers (parentheses, minus sign)
  • Zero Display: How to display zero values (0, -, blank)
  • Percentage Format: How percentages are displayed
  • Currency Display: Currency symbol position and format

4. Date Display Options

Date formatting and display:

  • Date Format: DD/MM/YYYY, MM/DD/YYYY, or custom
  • Time Display: Show or hide time with dates
  • Relative Dates: Show relative dates (Today, Yesterday, etc.)
  • Date Range Display: How date ranges are shown
  • Week Display: Show week numbers or week names

5. Color and Theme Settings

Visual appearance customization:

  • Status Colors: Colors for different statuses
  • Category Colors: Color coding for categories
  • Priority Colors: Colors for priority levels
  • Theme: Light, dark, or custom theme
  • Accent Color: Primary accent color
  • High Contrast: Enable high contrast mode

6. Column Management

Control column visibility and order:

  • Default Visible Columns: Columns shown by default
  • Column Order: Default column order
  • Column Grouping: Group related columns
  • Frozen Columns: Columns that stay visible when scrolling
  • Column Widths: Default and minimum column widths

Step-by-Step Configuration

Step 1: Configure Table Display

  1. Navigate to Display Settings → Table Display
  2. Set default columns to show
  3. Choose row height (compact, normal, comfortable)
  4. Enable alternating row colors for better readability
  5. Configure sticky headers if needed
  6. Set column width preferences

Step 2: Configure Report Display

  1. Go to Display Settings → Report Display
  2. Set default report layout (portrait or landscape)
  3. Choose page size (A4, Letter, etc.)
  4. Configure margins
  5. Set header and footer display options
  6. Choose font size and color options

Step 3: Configure Number Formatting

  1. Navigate to Display Settings → Number Formatting
  2. Set decimal places (usually 2 for currency)
  3. Choose thousand separator
  4. Configure negative number display
  5. Set zero value display
  6. Configure percentage format

Step 4: Configure Date Display

  1. Go to Display Settings → Date Display
  2. Set date format preference
  3. Configure time display options
  4. Enable relative dates if preferred
  5. Set date range display format

Step 5: Configure Colors and Themes

  1. Navigate to Display Settings → Colors and Themes
  2. Choose theme (light or dark)
  3. Configure status colors
  4. Set category color coding
  5. Choose accent color
  6. Enable high contrast if needed

Step 6: Configure Column Management

  1. Go to Display Settings → Column Management
  2. Set default visible columns
  3. Configure column order
  4. Set frozen columns if needed
  5. Configure column widths

Step 7: Save Settings

Click "Update" or "Save" to apply your display settings.

How Display Settings Affect Your Business

Readability

  • Proper formatting improves data readability
  • Color coding helps identify status and categories quickly
  • Appropriate row height reduces eye strain
  • Clear number formatting prevents misinterpretation

Efficiency

  • Default columns show most important information
  • Sticky headers keep context while scrolling
  • Color coding enables quick visual scanning
  • Optimized layout reduces scrolling and searching

Professional Appearance

  • Consistent formatting across reports
  • Professional color schemes
  • Clean, organized data presentation
  • Brand-consistent appearance

Best Practices

  • Consistency: Use consistent formatting across all views
  • Readability: Prioritize readability over density
  • Color Coding: Use colors consistently for status and categories
  • User Preferences: Consider team preferences for display
  • Accessibility: Ensure settings support accessibility needs
  • Print Optimization: Configure settings for both screen and print
  • Regular Review: Periodically review and optimize display settings

Troubleshooting

Display settings not applying?

  • Refresh the page
  • Clear browser cache
  • Verify settings are saved
  • Check if user-specific settings override general settings
  • Log out and log back in

Numbers displaying incorrectly?

  • Verify number formatting settings
  • Check decimal places configuration
  • Verify thousand separator setting
  • Review currency format settings

Columns not showing as expected?

  • Check default column visibility settings
  • Verify column order configuration
  • Reset to default columns if needed
  • Check if columns are hidden by user preference

Colors not displaying correctly?

  • Verify color settings are saved
  • Check theme selection
  • Clear browser cache
  • Verify browser supports color settings

Important Notes

  • Display settings affect how data is shown but not how it's stored
  • Some settings may be user-specific rather than system-wide
  • Print settings may differ from screen display settings
  • Export formats may use different formatting
  • Test display settings before applying organization-wide
  • Consider accessibility when configuring colors and themes
