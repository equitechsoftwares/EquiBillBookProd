Print Settings

Print settings allow you to configure how documents, reports, and invoices are printed from EquiBillBook. Proper print configuration ensures professional-looking printed documents that meet your business requirements.

What are Print Settings?

Print settings include:

  • Page Setup: Page size, orientation, and margins
  • Print Quality: Resolution and quality settings
  • Header and Footer: Custom headers and footers
  • Print Options: What to include in prints
  • Printer Selection: Default printer and printer settings
  • Document Formatting: Font, spacing, and layout options
  • Color Settings: Color or grayscale printing

Accessing Print Settings

To configure print settings:

  1. Navigate to Settings → Other Settings → Print Settings
  2. You'll find various print configuration options
  3. Configure settings based on your printing requirements

Key Configuration Areas

1. Page Setup

Basic page configuration:

  • Page Size: A4, Letter, Legal, A3, or custom size
  • Orientation: Portrait or Landscape
  • Margins: Top, bottom, left, and right margins
  • Page Breaks: Automatic or manual page breaks
  • Scaling: Fit to page or actual size

2. Print Quality

Print resolution and quality:

  • Print Quality: Draft, Normal, or High quality
  • Resolution: DPI (dots per inch) setting
  • Color Mode: Color, Grayscale, or Black & White
  • Print Speed: Balance between speed and quality

3. Header and Footer

Custom headers and footers:

  • Show Header: Enable or disable header
  • Header Content: Company name, logo, date, page number
  • Show Footer: Enable or disable footer
  • Footer Content: Page numbers, date, company info, disclaimers
  • Header/Footer Font: Font size and style
  • Header/Footer Position: Left, center, or right alignment

4. Print Options

What to include in prints:

  • Print Background: Include background colors and images
  • Print Images: Include images and logos
  • Print Grid Lines: Show grid lines in tables
  • Print Comments: Include comments and notes
  • Print Formulas: Show formulas instead of values
  • Print Headings: Include row and column headings

5. Document Formatting

Font and layout options:

  • Font Family: Default font for printed documents
  • Font Size: Default font size
  • Line Spacing: Spacing between lines
  • Table Formatting: How tables are formatted
  • Border Settings: Border styles and widths

6. Printer Settings

Default printer configuration:

  • Default Printer: Select default printer
  • Printer Properties: Printer-specific settings
  • Paper Source: Default paper tray
  • Duplex Printing: Enable double-sided printing
  • Collate: Collate multiple copies

Step-by-Step Configuration

Step 1: Configure Page Setup

  1. Navigate to Print Settings → Page Setup
  2. Select page size (A4 is most common)
  3. Choose orientation (Portrait for most documents, Landscape for wide reports)
  4. Set margins (standard: 1 inch or 2.5 cm)
  5. Configure page break settings
  6. Set scaling options

Step 2: Configure Print Quality

  1. Go to Print Settings → Print Quality
  2. Select print quality (Normal is recommended for most documents)
  3. Set resolution if available
  4. Choose color mode (Color for invoices, Grayscale for reports)
  5. Configure print speed if option available

Step 3: Configure Headers and Footers

  1. Navigate to Print Settings → Header and Footer
  2. Enable header if needed
  3. Configure header content (company name, logo, date)
  4. Enable footer if needed
  5. Configure footer content (page numbers, disclaimers)
  6. Set header/footer alignment and font

Step 4: Configure Print Options

  1. Go to Print Settings → Print Options
  2. Enable or disable background printing
  3. Configure image printing
  4. Set grid line printing for tables
  5. Configure comments and notes printing
  6. Set heading printing options

Step 5: Configure Document Formatting

  1. Navigate to Print Settings → Document Formatting
  2. Select default font family
  3. Set default font size
  4. Configure line spacing
  5. Set table formatting preferences
  6. Configure border settings

Step 6: Configure Printer Settings

  1. Go to Print Settings → Printer Settings
  2. Select default printer
  3. Configure printer properties
  4. Set default paper source
  5. Enable duplex printing if needed
  6. Configure collation for multiple copies

Step 7: Save Settings

Click "Update" or "Save" to apply your print settings.

Print Settings for Different Document Types

Invoices and Bills

  • Page Size: A4 or Letter
  • Orientation: Portrait
  • Color: Color (for logos and branding)
  • Quality: High (for professional appearance)
  • Header: Company logo and details
  • Footer: Terms and conditions, page numbers

Reports

  • Page Size: A4 or Letter
  • Orientation: Portrait or Landscape (depending on report width)
  • Color: Grayscale or Black & White (to save ink)
  • Quality: Normal
  • Header: Report title and date
  • Footer: Page numbers, generated date

Labels

  • Page Size: Custom (label sheet size)
  • Orientation: Portrait
  • Color: Color or Grayscale
  • Quality: High
  • Margins: Minimal
  • Scaling: Fit to label size

Best Practices

  • Standard Sizes: Use standard page sizes (A4, Letter) for compatibility
  • Margins: Use adequate margins for professional appearance
  • Color Usage: Use color for important documents, grayscale for reports
  • Headers/Footers: Include essential information in headers and footers
  • Test Prints: Test print settings before printing large batches
  • Printer Compatibility: Ensure settings work with your printer
  • Paper Conservation: Use grayscale and duplex printing when possible

Troubleshooting

Documents not printing correctly?

  • Verify page size matches paper in printer
  • Check orientation setting (portrait vs landscape)
  • Verify margins are within printable area
  • Check printer settings match configured settings
  • Test with a simple document first

Print quality is poor?

  • Increase print quality setting
  • Check printer resolution settings
  • Verify printer has sufficient ink/toner
  • Check printer driver is up to date
  • Clean printer heads if applicable

Headers/footers not appearing?

  • Verify headers/footers are enabled
  • Check header/footer content is configured
  • Verify margins allow space for headers/footers
  • Check print preview before printing

Colors not printing?

  • Verify color mode is set to Color
  • Check printer has color ink/toner
  • Verify printer supports color printing
  • Check printer driver color settings

Important Notes

  • Print settings may vary by document type
  • Some settings are printer-specific
  • Always preview before printing important documents
  • Test settings with sample prints
  • Save paper by using grayscale and duplex printing when appropriate
  • Keep printer drivers updated for best results
