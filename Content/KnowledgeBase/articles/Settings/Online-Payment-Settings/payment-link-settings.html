Payment Link Settings

Payment link settings in EquiBillBook allow you to configure how payment links are generated, shared, and managed. This guide explains how to set up and customize payment link settings effectively.

What are Payment Link Settings?

Payment link settings control various aspects of payment links, including link generation, expiration, customization, sharing options, and security. These settings help you create professional, secure, and user-friendly payment links for collecting payments from customers.

Accessing Payment Link Settings

To access payment link settings:

  1. Navigate to Settings → Online Payment Settings → Payment Link Settings
  2. View current payment link configuration
  3. Configure link settings
  4. Save changes

Payment Link Generation

Configure how payment links are generated:

  • Link Format: Customize link URL format
  • Link Prefix: Set custom prefix for links
  • Link Length: Configure link length
  • Link Uniqueness: Ensure each link is unique
  • Link Encoding: Configure link encoding

Payment Link Expiration

Configure payment link expiration:

  • Default Expiry: Set default expiration time (hours, days)
  • Expiry Options: Provide expiry options when creating links
  • No Expiry: Allow links without expiration
  • Expiry Reminders: Send reminders before link expires
  • Expired Link Handling: What happens when link expires

Payment Link Customization

Customize payment link appearance:

  • Branding: Add logo and colors to payment page
  • Custom Domain: Use custom domain for payment links
  • Page Title: Customize payment page title
  • Page Description: Add description to payment page
  • Custom CSS: Add custom styling (if supported)

Payment Link Content

Configure payment link content:

  • Invoice Details: Show invoice details on payment page
  • Customer Information: Display customer information
  • Payment Amount: Show payment amount clearly
  • Due Date: Display due date
  • Payment Methods: Show available payment methods
  • Terms and Conditions: Include terms and conditions

Payment Link Sharing

Configure payment link sharing options:

  • Email Sharing: Send links via email
  • SMS Sharing: Send links via SMS
  • WhatsApp Sharing: Send links via WhatsApp
  • Copy Link: Allow copying payment link
  • QR Code: Generate QR code for payment link
  • Social Sharing: Share on social media (if applicable)

Payment Link Security

Configure security settings for payment links:

  • Link Encryption: Encrypt payment link data
  • Access Control: Restrict link access
  • Password Protection: Require password to access link
  • IP Restrictions: Restrict access by IP address
  • Single Use: Allow link to be used only once
  • Link Validation: Validate link before processing payment

Payment Link Notifications

Configure notifications for payment links:

  • Link Generated: Notify when link is created
  • Link Opened: Notify when customer opens link
  • Link Expiring: Send reminder before expiration
  • Link Expired: Notify when link expires
  • Payment Received: Notify when payment is made
  • Payment Failed: Notify on payment failure

Payment Link Analytics

Configure analytics for payment links:

  • Track Views: Track how many times link is viewed
  • Track Clicks: Track link clicks
  • Track Conversions: Track payment conversions
  • Geographic Tracking: Track link access by location
  • Device Tracking: Track device types used
  • Analytics Reports: Generate analytics reports

Payment Link Templates

Create payment link templates:

  • Template Creation: Create reusable link templates
  • Template Variables: Use variables in templates
  • Template Selection: Choose template when creating link
  • Default Template: Set default template
  • Template Customization: Customize template appearance

Payment Link Redirects

Configure redirect URLs after payment:

  • Success URL: Redirect after successful payment
  • Failure URL: Redirect after failed payment
  • Cancel URL: Redirect if payment is cancelled
  • Custom Messages: Show custom messages on redirect
  • Redirect Delay: Set delay before redirect

Payment Link Management

Manage payment links:

  • Link List: View all payment links
  • Link Search: Search for specific links
  • Link Filtering: Filter links by status, date, etc.
  • Link Deactivation: Deactivate links manually
  • Link Deletion: Delete unused links
  • Bulk Operations: Perform bulk actions on links

Payment Link Best Practices

  • Set Appropriate Expiry: Set expiry based on payment urgency
  • Customize Branding: Add your branding for professional appearance
  • Clear Instructions: Provide clear payment instructions
  • Multiple Sharing Options: Offer various ways to share links
  • Track Performance: Monitor link performance and conversions
  • Secure Links: Implement security measures
  • Test Links: Test payment links before sharing
  • Monitor Expiry: Send reminders before links expire

Payment Link Troubleshooting

Common issues and solutions:

  • Link Not Working: Check link is active and not expired
  • Link Expired: Generate new link or extend expiry
  • Payment Not Processing: Verify gateway configuration
  • Link Not Accessible: Check security settings and restrictions
  • QR Code Not Working: Verify QR code generation

Properly configured payment link settings help you create professional, secure, and effective payment links that improve payment collection and customer experience. Configure settings according to your business needs and test thoroughly.
