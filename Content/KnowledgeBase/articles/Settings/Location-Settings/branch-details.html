Branch Details

Branch Details in EquiBillBook provide comprehensive information about each branch. This guide will help you understand, view, and manage branch details effectively.

Understanding Branch Details

Branch details include all information about a branch:

  • Basic information (name, code, type)
  • Address and contact information
  • GST registration details
  • Financial settings
  • Operational settings
  • User access and permissions
  • Statistics and performance data

Accessing Branch Details

To view branch details:

  1. Navigate to Settings → Location Settings → Branch Management
  2. You'll see a list of all branches
  3. Click on a branch name or "View Details" button
  4. Branch details page will open

Branch Details Sections

1. Basic Information

This section shows basic branch information:

Displayed Information

  • Branch Name: Name of the branch
  • Branch Code: Unique identifier code
  • Branch Type: Main branch, Sub-branch, Store, etc.
  • Status: Active or Inactive
  • Parent Branch: Parent branch (if sub-branch)
  • Created Date: When branch was created
  • Last Updated: Last modification date

2. Address Information

Complete address details of the branch:

Address Details

  • Complete Address: Full address with all lines
  • City: City name
  • State: State/Union Territory
  • Pincode: Postal code
  • Country: Country
  • Map Location: Location on map (if configured)

3. Contact Information

Contact details for the branch:

Contact Details

  • Phone Number: Primary contact number
  • Alternate Phone: Secondary contact
  • Email: Branch email address
  • Website: Branch website
  • Contact Person: Primary contact person

4. GST Information

GST registration details (if branch has separate GST):

GST Details

  • GST Registration Number: Branch GSTIN
  • GST Registration Date: Registration date
  • GST Status: Active/Inactive
  • GST Type: Regular, Composition, etc.
  • Place of Business: Place of business for GST

5. Financial Settings

Financial configuration for the branch:

Financial Details

  • Separate Books: Whether separate books are maintained
  • Financial Year: Branch financial year
  • Currency: Branch currency
  • Opening Balance: Opening balance
  • Current Balance: Current financial balance

6. Operational Settings

Operational configuration:

Sales Settings

  • Sales Enabled: Whether sales are enabled
  • Invoice Prefix: Branch invoice prefix
  • Sales Configuration: Sales-specific settings

Purchase Settings

  • Purchases Enabled: Whether purchases are enabled
  • Bill Prefix: Branch bill prefix
  • Purchase Configuration: Purchase-specific settings

Inventory Settings

  • Stock Tracking: Whether stock tracking is enabled
  • Low Stock Alerts: Low stock alert configuration
  • Transfer Settings: Stock transfer settings

7. User Access

Users assigned to the branch:

User Information

  • Branch Manager: Assigned branch manager
  • Total Users: Number of users assigned
  • User List: List of all assigned users
  • Permissions: User permissions for the branch

8. Statistics

Performance and activity statistics:

Branch Statistics

  • Total Sales: Total sales amount
  • Total Purchases: Total purchase amount
  • Current Stock Value: Value of current stock
  • Total Transactions: Number of transactions
  • Active Customers: Number of active customers
  • Active Suppliers: Number of active suppliers

Editing Branch Details

To modify branch details:

Access Edit Mode

  1. Go to branch details page
  2. Click "Edit" button
  3. Edit form will open
  4. Modify the required information
  5. Click "Save" to save changes

What Can Be Edited

  • Branch name (with restrictions)
  • Address information
  • Contact information
  • GST details
  • Operational settings
  • Financial settings
  • User assignments

What Cannot Be Edited

  • Branch code (usually fixed after creation)
  • Created date
  • Some historical data

Viewing Branch Reports

From branch details, you can access branch-specific reports:

Available Reports

  • Branch Sales Report: Sales for this branch
  • Branch Purchase Report: Purchases for this branch
  • Branch Stock Report: Stock levels at this branch
  • Branch Profit Report: Profitability of this branch
  • Branch Transaction Report: All transactions for this branch

Branch Status Management

You can change branch status from details page:

Activating Branch

  1. Go to branch details
  2. Click "Activate" button
  3. Branch status changes to Active
  4. Branch can now process transactions

Deactivating Branch

  1. Go to branch details
  2. Click "Deactivate" button
  3. Confirm deactivation
  4. Branch status changes to Inactive
  5. Branch cannot process new transactions

Exporting Branch Details

You can export branch information:

Export Options

  • Export to PDF: Generate PDF of branch details
  • Export to Excel: Export data to Excel
  • Print: Print branch details

Best Practices

  • Regular Review: Periodically review branch details for accuracy
  • Update Information: Keep branch information up to date
  • Verify Settings: Verify operational settings are correct
  • Monitor Statistics: Regularly monitor branch statistics
  • Check Users: Verify user assignments are appropriate
  • Document Changes: Document any changes made to branch details

Common Operations

Quick Actions

From branch details page, you can:

  • Edit branch information
  • Activate/Deactivate branch
  • View branch reports
  • Manage users
  • Configure settings
  • Export information
