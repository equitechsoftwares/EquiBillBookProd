Operating Hours

Operating Hours settings in Restaurant Settings allow you to configure your restaurant's operating schedule, including default hours for each day, special hours for holidays, break times, and date overrides. This guide covers all operating hours configuration options.

Accessing Operating Hours Settings

To configure operating hours:

  1. Navigate to Settings â†’ Restaurant Settings
  2. Click on "Operating Hours" or "Hours Configuration"
  3. Configure hours for each day of the week

Default Operating Hours

Setting Hours for Each Day

Configure standard operating hours for each day of the week:

  • Monday: Set opening and closing time
  • Tuesday: Set opening and closing time
  • Wednesday: Set opening and closing time
  • Thursday: Set opening and closing time
  • Friday: Set opening and closing time
  • Saturday: Set opening and closing time
  • Sunday: Set opening and closing time

Operating Hours Configuration

For each day, configure:

  • Open/Closed: Mark day as open or closed
  • Opening Time: When restaurant opens (e.g., 11:00 AM)
  • Closing Time: When restaurant closes (e.g., 11:00 PM)
  • 24-Hour: Mark as open 24 hours
  • Copy to Other Days: Copy hours to multiple days at once

Multiple Time Slots

Split Hours

Configure multiple operating periods in a day:

  • Lunch Hours: First operating period (e.g., 11:00 AM - 3:00 PM)
  • Dinner Hours: Second operating period (e.g., 6:00 PM - 11:00 PM)
  • Multiple Periods: Add more time slots as needed
  • Gap Between Periods: Break time between operating periods

Use Cases

  • Restaurants with lunch and dinner service
  • Businesses with afternoon break
  • Extended hours with mid-day closure

Break Times

Break Configuration

Set up break periods during operating hours:

  • Enable Breaks: Allow break times during operating hours
  • Break Start Time: When break period begins
  • Break End Time: When break period ends
  • Break Days: Days when breaks apply
  • Multiple Breaks: Configure multiple break periods per day

Break Behavior

  • No Bookings During Break: Prevent bookings during break time
  • Allow Existing Bookings: Allow bookings that span break time
  • Break Notifications: Notify customers about break times

Special Hours Configuration

Holiday Hours

Set different hours for holidays:

  • Holiday List: Define holidays and special dates
  • Holiday Hours: Set operating hours for each holiday
  • Closed on Holidays: Mark restaurant as closed on holidays
  • Extended Holiday Hours: Set longer hours for holidays

Peak Season Hours

  • Peak Season Dates: Define peak season period
  • Extended Hours: Longer operating hours during peak season
  • Special Rules: Different booking rules during peak season

Date Overrides

Creating Date Overrides

Set specific dates with different operating hours:

  • Override Date: Select specific date
  • Custom Hours: Set different hours for that date
  • Closed Date: Mark specific date as closed
  • Special Event Hours: Extended hours for special events

Date Override Types

  • Single Date: Override for one specific date
  • Date Range: Override for date range
  • Recurring Override: Override that repeats (e.g., every first Monday)

Managing Overrides

  • View All Overrides: See all date overrides in calendar view
  • Edit Override: Modify existing date override
  • Delete Override: Remove date override
  • Copy Override: Copy override to another date

Time Zone Configuration

Time Zone Settings

  • Restaurant Time Zone: Set time zone for your restaurant
  • Display Format: 12-hour or 24-hour time format
  • Daylight Saving: Automatic adjustment for daylight saving time

Operating Hours Display

Public Display

  • Show on Website: Display operating hours on public booking portal
  • Show on Booking Page: Display hours on booking interface
  • Hours Format: How hours are displayed to customers

Staff Display

  • Dashboard Display: Show operating hours on restaurant dashboard
  • Booking Calendar: Highlight operating hours on calendar
  • Hours Reminder: Remind staff of operating hours

Booking Restrictions Based on Hours

Booking Availability

  • Only During Hours: Allow bookings only during operating hours
  • Buffer Before Opening: Allow bookings slightly before opening
  • Buffer After Closing: Allow bookings that end slightly after closing
  • Strict Hours: Enforce strict operating hours for bookings

Last Booking Time

  • Last Booking Before Closing: Latest time when bookings can be made
  • Minimum Duration: Ensure bookings don't extend beyond closing
  • Closing Time Buffer: Time buffer before closing for last bookings

Saving Operating Hours

After configuring operating hours:

  1. Review hours for all days
  2. Check date overrides and special hours
  3. Verify break times are correctly set
  4. Click "Save" or "Update Hours"
  5. Hours will apply immediately to booking availability

Best Practices

  • Accurate Hours: Ensure operating hours match actual restaurant schedule
  • Regular Updates: Update hours when schedule changes
  • Holiday Planning: Set holiday hours well in advance
  • Clear Communication: Display hours clearly to customers
  • Staff Awareness: Ensure staff know operating hours
  • Seasonal Adjustments: Update hours for seasonal changes
  • Special Events: Plan date overrides for special events early

Related Settings

Operating hours work together with:

  • Booking Settings: Booking availability is limited by operating hours
  • Date Overrides: Special dates with different hours
  • Public Booking Settings: Hours affect public booking availability
  • Notification Settings: Notify customers about hours and closures

Troubleshooting

Bookings allowed outside operating hours?

  • Check if "Strict Hours" enforcement is enabled
  • Verify operating hours are correctly set
  • Check for date overrides that might affect the date
  • Review booking buffer settings

Hours not displaying correctly?

  • Check time zone settings
  • Verify time format (12-hour vs 24-hour)
  • Clear browser cache
  • Check for conflicting date overrides

Proper operating hours configuration ensures accurate booking availability and smooth restaurant operations!
