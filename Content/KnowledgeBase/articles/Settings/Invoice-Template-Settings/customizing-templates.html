Customizing Templates

Customizing templates in EquiBillBook allows you to personalize the appearance and layout of your invoices, quotations, and other documents to match your brand and business needs.

Accessing Template Customization

To customize a template:

  1. Navigate to Settings â†’ Invoice Template Settings
  2. Select the invoice type
  3. Find the template you want to customize
  4. Click "Customize" or "Edit"

Template Customization Options

1. Layout Customization

Control the overall structure:

Layout Style
  • Standard: Traditional invoice layout
  • Compact: Condensed layout for space efficiency
  • Detailed: Comprehensive layout with more information
Section Arrangement
  • Reorder sections (Header, Company Details, Customer Details, Items, Totals, Footer)
  • Show/hide sections
  • Adjust section spacing

2. Color Customization

Customize the color scheme:

Primary Colors
  • Primary Color: Main brand color
  • Secondary Color: Accent color
  • Header Text Color: Text color in header
  • Body Background Color: Background color
  • Body Text Color: Main text color
Table Colors
  • Border Color: Border color for tables
  • Table Header Color: Header row background
  • Total Row Color: Total row highlight
  • Due Row Color: Due amount highlight

3. Logo and Branding

Add your business identity:

  • Logo: Upload company logo
  • Logo Position: Left, Center, or Right
  • Logo Size: Adjust logo dimensions
  • Company Details: Show/hide company information

4. Field Customization

Control which fields appear:

Invoice Details
  • Invoice Number
  • Invoice Date
  • Due Date
  • Reference Number
  • Payment Terms
Customer Information
  • Customer Name
  • Billing Address
  • Shipping Address
  • Contact Information
  • Tax Information
Item Details
  • Item Name/Description
  • Quantity
  • Unit Price
  • Amount
  • Tax Information
  • Item Images
  • Barcode
Totals Section
  • Subtotal
  • Discount
  • Tax Breakdown
  • Shipping Charges
  • Grand Total
  • Amount Due

5. Additional Elements

Add extra features:

  • QR Code: Add QR code for payments or verification
  • QR Code Position: Where to place QR code
  • Tax Breakdown: Show detailed tax information
  • Payment Information: Display payment details
  • Footer Note: Add custom footer text
  • Terms and Conditions: Include terms

Step-by-Step Customization

Step 1: Open Template Editor

  1. Go to Invoice Template Settings
  2. Select template to customize
  3. Click "Customize"

Step 2: Configure Layout

  • Select layout style
  • Arrange sections
  • Set section visibility

Step 3: Set Colors

  • Choose primary and secondary colors
  • Set text colors
  • Configure table colors
  • Use color picker for exact colors

Step 4: Add Logo

  • Upload logo image
  • Set logo position
  • Adjust logo size
  • Preview logo placement

Step 5: Configure Fields

  • Show/hide fields
  • Set field labels
  • Arrange field order
  • Configure field visibility

Step 6: Add Additional Elements

  • Add QR code if needed
  • Configure footer note
  • Set up terms and conditions
  • Add payment information

Step 7: Preview Template

  • Use preview with sample data
  • Check all sections
  • Verify colors and layout
  • Test on different screen sizes

    • Step 8: Save Customization

    1. Review all changes
    2. Click "Save" or "Update"
    3. Customizations will be applied

    Troubleshooting

    Colors not applying correctly?

    • Verify color codes are correct
    • Check color format (hex, RGB)
    • Clear browser cache
    • Preview again

    Logo not displaying?

    • Check logo file format (PNG, JPG)
    • Verify file size is within limits
    • Check logo upload was successful
    • Verify logo position settings

    Fields not showing/hiding correctly?

    • Check field visibility settings
    • Verify field configuration
    • Check conditional display rules
    • Review template settings
