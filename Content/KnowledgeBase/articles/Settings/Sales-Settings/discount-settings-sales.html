Discount Settings

Discount Settings for Sales in EquiBillBook allow you to configure discount types, default discounts, special discounts, and customer group discounts. This guide covers all discount-related settings for sales transactions.

Accessing Discount Settings

To configure discount settings:

  1. Navigate to Settings → Sales Settings → Discount Settings
  2. Configure discount types, rates, and application rules
  3. Click "Update" or "Save" to apply changes

Discount Type Configuration

Configure available discount types:

Discount Types

  • Percentage Discount: Discount as percentage of item price
  • Fixed Amount Discount: Fixed amount discount
  • Quantity-based Discount: Discount based on quantity purchased
  • Volume Discount: Discount based on order value

Default Discount Type

  • Set default discount type for new transactions
  • Can be overridden per transaction
  • Affects discount calculation method

Default Discount Settings

Configure default discount values:

Default Discount Rate

  • Default Percentage: Set default discount percentage
  • Default Amount: Set default discount amount
  • Apply to All Items: Apply default discount to all items
  • Apply Per Item: Apply discount individually per item

Discount Application

  • Item Level: Apply discount at item level
  • Invoice Level: Apply discount at invoice level
  • Both: Allow both item and invoice level discounts

Special Discount Settings

Configure special discount options:

Enable Special Discount

  • Allow Special Discount: Enable special discount functionality
  • Special Discount Limit: Set maximum special discount percentage
  • Approval Required: Require approval for special discounts
  • Special Discount Account: Select account for special discounts

Special Discount Rules

  • Minimum Amount: Set minimum order amount for special discount
  • Maximum Amount: Set maximum discount amount
  • Valid Period: Set validity period for special discounts

Customer Group Discounts

Configure discounts based on customer groups:

Group-based Pricing

  • Enable Customer Groups: Enable customer group functionality
  • Group Discounts: Set discounts for specific customer groups
  • Group Pricing: Set different prices for customer groups

Group Discount Configuration

  • Create discount rules for each customer group
  • Set discount percentage or amount per group
  • Apply discounts automatically based on customer group

Quantity-based Discounts

Configure discounts based on quantity:

  • Enable Quantity Discount: Enable quantity-based discounts
  • Discount Tiers: Set discount tiers based on quantity
  • Example: Buy 10+ get 5% off, Buy 50+ get 10% off

Volume Discounts

Configure discounts based on order value:

  • Enable Volume Discount: Enable volume-based discounts
  • Discount Tiers: Set discount tiers based on order value
  • Example: Orders above ₹10,000 get 5% off, above ₹50,000 get 10% off

Discount Account Configuration

Configure accounts for discount recording:

  • Discount Account: Select account for regular discounts
  • Special Discount Account: Select account for special discounts
  • Discount Allocation: Configure how discounts are allocated

Discount Limits and Restrictions

Configure discount limits and restrictions:

Discount Limits

  • Maximum Discount: Set maximum discount percentage allowed
  • Minimum Order Value: Set minimum order value for discounts
  • Discount Approval: Require approval for discounts above limit

Discount Restrictions

  • Item Restrictions: Restrict discounts on specific items
  • Category Restrictions: Restrict discounts on specific categories
  • Customer Restrictions: Restrict discounts for specific customers

Discount Display Settings

Configure how discounts are displayed:

  • Show Discount on Invoice: Display discount on invoices
  • Show Discount Breakdown: Show detailed discount breakdown
  • Discount Label: Customize discount label/description

Best Practices

  • Clear Rules: Define clear discount rules and policies
  • Approval Process: Set approval process for high discounts
  • Account Setup: Properly configure discount accounts
  • Documentation: Document discount policies
  • Review Regularly: Regularly review discount effectiveness

Common Scenarios

Retail Business

  • Simple percentage discounts
  • Seasonal discount campaigns
  • Loyalty discounts

Wholesale Business

  • Customer group discounts
  • Volume-based discounts
  • Quantity-based discounts

What's Next?

After configuring Discount Settings, you should:

  1. Configure Tax Settings for Sales
  2. Set up Customer Groups (if applicable)
  3. Test discount application
  4. Review discount accounts
  5. Train staff on discount policies

Proper Discount Settings help you manage pricing strategies and customer incentives effectively.
