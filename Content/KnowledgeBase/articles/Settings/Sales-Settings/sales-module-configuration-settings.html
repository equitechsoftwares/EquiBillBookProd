Sales Module Configuration

The Sales Module Configuration in EquiBillBook Settings allows you to activate and configure the core sales module features. This guide will help you understand how to enable and set up the sales module for your business operations.

Accessing Sales Module Configuration

To configure the Sales Module:

  1. Navigate to Settings → Sales Settings → Sales Module Configuration
  2. You'll see options to enable or disable various sales features
  3. Configure the settings according to your business needs
  4. Click "Update" or "Save" to apply changes

Module Activation

The Sales Module Configuration allows you to control which sales features are available in your system:

Core Sales Features

  • Sales Invoice: Enable or disable sales invoice creation
  • Sales Quotation: Enable or disable quotation functionality
  • Sales Order: Enable or disable order management
  • Sales Proforma: Enable or disable proforma invoice creation
  • Delivery Challan: Enable or disable delivery challan functionality
  • POS (Point of Sale): Enable or disable POS operations
  • Recurring Sales: Enable or disable recurring invoice generation
  • Sales Returns: Enable or disable sales return functionality
  • Payment Links: Enable or disable payment link generation

Feature Enablement

You can selectively enable features based on your business model:

For Retail Businesses

  • Enable POS for quick transactions
  • Enable Sales Invoice for billing
  • Disable Quotations and Orders if not needed

For Wholesale Businesses

  • Enable Quotations for customer proposals
  • Enable Orders for order management
  • Enable Delivery Challan for delivery tracking
  • Enable Sales Invoice for final billing

For Service Businesses

  • Enable Sales Invoice for service billing
  • Enable Recurring Sales for subscription services
  • Enable Payment Links for online payments
  • Disable stock-related features if not applicable

Module Integration

The Sales Module integrates with other modules:

  • Inventory Module: For stock tracking and management
  • Accounts Module: For accounting entries and financial records
  • Customer Module: For customer information and credit limits
  • Tax Module: For tax calculations and compliance

Best Practices

  • Start Simple: Enable only the features you currently need
  • Plan Ahead: Consider future business needs when configuring
  • Review Regularly: Periodically review and adjust module settings
  • Test Changes: Test new features before full implementation
  • Train Users: Ensure your team understands enabled features

What's Next?

After configuring the Sales Module, you should:

  1. Configure Invoice Settings
  2. Set up Quotation Settings
  3. Configure Order Settings
  4. Set up POS Settings (if applicable)
  5. Configure Payment Settings
  6. Set up Discount Settings
  7. Configure Tax Settings for Sales

Proper Sales Module Configuration is the foundation for efficient sales operations in EquiBillBook.
