Payment Settings

Payment Settings for Sales in EquiBillBook allow you to configure payment methods, payment terms, online payment integration, and payment processing for sales transactions. This guide covers all payment-related settings.

Accessing Payment Settings

To configure payment settings:

  1. Navigate to Settings → Sales Settings → Payment Settings
  2. Configure payment methods, terms, and processing options
  3. Click "Update" or "Save" to apply changes

Payment Methods Configuration

Configure accepted payment methods for sales:

Available Payment Methods

  • Cash: Enable cash payments
  • Bank Transfer: Enable bank transfer payments
  • Cheque: Enable cheque payments
  • Credit Card: Enable credit card payments
  • Debit Card: Enable debit card payments
  • UPI: Enable UPI payments
  • Digital Wallet: Enable digital wallet payments
  • Credit: Enable credit sales (payment on account)

Payment Method Settings

  • Default Method: Set default payment method
  • Required Method: Make payment method selection mandatory
  • Multiple Methods: Allow multiple payment methods per transaction

Payment Terms Configuration

Configure payment terms for sales transactions:

Default Payment Terms

  • Due Days: Set default number of days for payment due
  • Due Date Calculation: Configure how due dates are calculated
  • Payment Term Required: Make payment term selection mandatory

Custom Payment Terms

  • Create Terms: Create custom payment terms
  • Term Examples: Net 15, Net 30, Net 60, Due on Receipt
  • Term Description: Add descriptions for payment terms

Online Payment Integration

Configure online payment gateway integration:

Payment Gateway Setup

  • Enable Online Payment: Activate online payment functionality
  • Payment Gateway: Select payment gateway provider
  • Gateway Configuration: Configure gateway credentials and settings
  • Test Mode: Enable test mode for testing

Payment Link Settings

  • Enable Payment Links: Allow generating payment links
  • Link Expiry: Set expiry period for payment links
  • Link Sharing: Configure how links are shared
  • Auto Email: Automatically email payment links

Partial Payment Settings

Configure partial payment options:

  • Allow Partial Payment: Enable partial payments
  • Minimum Payment: Set minimum payment amount
  • Payment Allocation: Configure how payments are allocated
  • Outstanding Tracking: Track outstanding amounts

Advance Payment Settings

Configure advance payment handling:

  • Allow Advance Payment: Enable advance payments
  • Advance Allocation: Configure advance payment allocation
  • Advance Account: Select account for advance payments
  • Advance Refund: Configure advance refund process

Payment Receipt Settings

Configure payment receipt generation:

  • Auto Generate Receipt: Automatically generate payment receipts
  • Receipt Numbering: Configure receipt numbering
  • Receipt Template: Select receipt template
  • Auto Print Receipt: Automatically print payment receipts
  • Email Receipt: Automatically email payment receipts

Payment Reconciliation

Configure payment reconciliation settings:

  • Auto Reconciliation: Enable automatic payment reconciliation
  • Reconciliation Rules: Define reconciliation matching rules
  • Manual Reconciliation: Allow manual reconciliation
  • Reconciliation Reports: Generate reconciliation reports

Payment Reminders

Configure payment reminder settings:

  • Enable Reminders: Activate payment reminder system
  • Reminder Schedule: Set reminder schedule (before due, after due)
  • Reminder Methods: Configure reminder methods (email, SMS, WhatsApp)
  • Reminder Templates: Customize reminder messages

Refund Settings

Configure refund processing:

  • Allow Refunds: Enable refund functionality
  • Refund Approval: Require approval for refunds
  • Refund Methods: Configure refund methods
  • Refund Account: Select account for refunds

Payment Account Configuration

Configure accounts for payment processing:

  • Cash Account: Select account for cash payments
  • Bank Account: Select account for bank payments
  • Credit Account: Select account for credit sales
  • Payment Account: Configure payment accounts

Best Practices

  • Payment Terms: Set clear and consistent payment terms
  • Online Payment: Enable online payment for faster collections
  • Reminders: Set up automated payment reminders
  • Reconciliation: Regularly reconcile payments
  • Security: Ensure secure payment processing

Common Scenarios

Retail Business

  • Immediate payment (cash/card)
  • Multiple payment methods
  • Online payment integration

B2B Business

  • Credit terms (Net 30, Net 60)
  • Payment reminders
  • Payment reconciliation

What's Next?

After configuring Payment Settings, you should:

  1. Configure Discount Settings
  2. Set up Tax Settings for Sales
  3. Test payment processing
  4. Set up payment reminders
  5. Configure payment gateway (if applicable)

Proper Payment Settings ensure smooth payment processing and efficient collection management.
