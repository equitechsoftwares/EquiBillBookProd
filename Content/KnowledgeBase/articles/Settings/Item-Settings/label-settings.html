Label Settings

Label settings in EquiBillBook allow you to configure how item labels are generated, formatted, and printed. These settings control label appearance, included information, and printing behavior for inventory management and product identification.

Accessing Label Settings

To configure label settings:

  1. Navigate to Settings â†’ Item Settings
  2. Go to Label Settings or Print Label Settings
  3. Configure various label-related options
  4. Click "Update" to save changes

Label Printing Enable/Disable

Control label printing functionality:

  • Enable Print Label: Enable or disable the label printing feature
  • When enabled, you can generate and print labels for items
  • When disabled, label printing options are hidden
  • Useful for businesses that don't require label printing

Default Label Template

Configure default label template:

  • Default Template: Select the default template for label generation
  • Templates define label layout and content
  • Can be changed per label generation if needed
  • Consistent templates ensure uniform labeling

Default Label Dimensions

Set default label size:

  • Width: Default label width (inches, mm, or cm)
  • Height: Default label height
  • Standard Sizes: Choose common sizes (1x1, 2x1, 4x2 inches)
  • Dimensions can be customized per label generation

Default Font Sizes

Configure default font sizes for label elements:

  • Branch Name Size: Font size for location name
  • Product Name Size: Font size for item name
  • Variation Size: Font size for variation details
  • Price Size: Font size for price information
  • Date Size: Font size for packaging date

Default Field Visibility

Control which fields appear by default:

  • Show Branch Name: Display or hide branch name
  • Show Product Variation: Show variation when available
  • Show Packaging Date: Display packaging/manufacturing date
  • Show Price: Include or exclude price
  • Show Barcode: Display barcode on labels

Barcode Settings

Configure barcode generation:

  • Barcode Type: Default barcode format (Code 128, EAN-13, UPC)
  • Barcode Height: Default vertical size
  • Barcode Width: Default horizontal size
  • Show Barcode Value: Display numbers below barcode
  • Barcode Position: Default position on label

Price Display Settings

Configure price display:

  • Price Format: Ex-tax or inc-tax by default
  • Currency Symbol: Show or hide currency symbol
  • Price Label: Default text before price
  • Decimal Places: Number of decimal places

Text Formatting Settings

Configure text appearance:

  • Font Family: Default font type
  • Font Weight: Default font weight (bold, normal, light)
  • Text Alignment: Default alignment (left, center, right)
  • Line Height: Default spacing between lines

Label Layout Settings

Configure default layout:

  • Element Order: Default sequence of fields
  • Vertical Spacing: Default space between elements
  • Margins: Default margins (top, bottom, left, right)
  • Layout Style: Default arrangement style

Print Settings

Configure printing behavior:

  • Auto Print: Automatically print after generation
  • Print Preview: Show preview before printing
  • Printer Selection: Default printer for labels
  • Copies: Default number of copies per label

Label Quantity Settings

Configure default quantities:

  • Default Labels per Item: Default number of labels to generate
  • Bulk Generation: Enable batch label generation
  • Quantity Limits: Maximum labels per generation

Integration Settings

Configure label integration:

  • Barcode Scanner Integration: Enable barcode scanning for label selection
  • Export Options: Enable label export formats
  • Template Sharing: Share templates across branches

Label Settings Best Practices

  • Set appropriate default dimensions for your label stock
  • Choose readable font sizes for your label size
  • Include only essential information to avoid clutter
  • Test settings with sample labels before bulk printing
  • Maintain consistency across all labels
  • Regularly review and update settings as needed

Settings Maintenance

  • Review settings periodically
  • Update when label requirements change
  • Test settings after system updates
  • Document custom settings for reference
  • Backup settings configurations

Label settings in EquiBillBook provide comprehensive control over how labels are generated and printed, ensuring professional and consistent labeling for your inventory management needs.
