Catalogue Items Management

Managing items in your public catalogue allows you to control which products are displayed, update item information, and organize your catalogue effectively. This guide covers all aspects of catalogue items management.

Understanding Catalogue Item Selection

When creating or editing a catalogue, you can choose which items to include based on the catalogue type:

  • All Items: Automatically includes all active items from your inventory
  • Selected Items: Manually select specific items to include
  • By Category: Include all items from selected categories
  • By Brand: Include all items from selected brands

Selecting Items for Catalogue

To select items for your catalogue:

  1. Open your catalogue for editing
  2. Choose Selected Items as catalogue type
  3. Click on Select Items or Add Items
  4. Browse or search for items
  5. Select items you want to include
  6. Save your selections

Selecting by Category

To include items by category:

  1. Choose By Category as catalogue type
  2. Click on Select Categories
  3. Browse category hierarchy
  4. Select categories to include
  5. All items from selected categories will be included

Selecting by Brand

To include items by brand:

  1. Choose By Brand as catalogue type
  2. Click on Select Brands
  3. Browse available brands
  4. Select brands to include
  5. All items from selected brands will be included

Item Information Display

Configure what information is shown for each item:

  • Item Name: Product name
  • Item Code/SKU: Product code (if enabled)
  • Description: Product description
  • Images: Product images (if enabled)
  • Price: Selling price (if enabled)
  • MRP: Maximum Retail Price (if enabled)
  • Discount: Discount information (if enabled)
  • Stock Status: Availability status (if enabled)
  • Category: Item category
  • Brand: Item brand

Managing Item Display Order

Control how items appear in your catalogue:

  • Default Sort Order: Set default sorting (Popularity, Price, Newest)
  • Custom Sorting: Some catalogues allow custom item ordering
  • Featured Items: Highlight specific items (if available)

Updating Catalogue Items

To update items in an existing catalogue:

  1. Open the catalogue for editing
  2. Navigate to Items section
  3. Add new items or remove existing ones
  4. Update item selections based on catalogue type
  5. Save changes

Bulk Item Management

For catalogues with many items:

  • Bulk Add: Add multiple items at once
  • Bulk Remove: Remove multiple items
  • Category-based Updates: Update all items in a category
  • Brand-based Updates: Update all items in a brand

Item Availability

Control item visibility based on availability:

  • Show Out of Stock: Display items even when out of stock
  • Hide Out of Stock: Hide unavailable items
  • Stock Status Display: Show stock availability to customers

Item Pricing in Catalogue

Manage how prices are displayed:

  • Price Group: Select which price group to use
  • Show Prices: Enable/disable price display
  • Show MRP: Display Maximum Retail Price
  • Show Discount: Display discount information
  • Price Updates: Prices update automatically from item master

Item Images

Manage product images:

  • Show Images: Enable image display
  • Image Quality: Ensure images are optimized for web
  • Multiple Images: Some items may have multiple images
  • Image Updates: Images update from item master data

Filtering and Search

Customers can filter and search items:

  • Category Filters: Filter by category
  • Brand Filters: Filter by brand
  • Price Range: Filter by price range
  • Search: Search by name, code, or description

Item Details Management

For items with variants:

  • Item Variants: Include all variants or specific ones
  • Variant Selection: Choose which variants to display
  • Variant Pricing: Show variant-specific prices

Best Practices

  • Regularly review and update catalogue items
  • Remove discontinued or unavailable items
  • Add new products promptly
  • Ensure item information is accurate and up-to-date
  • Use high-quality product images
  • Organize items logically using categories
  • Keep pricing information current
  • Test item display on different devices
  • Monitor customer interactions with items

Troubleshooting

Common issues and solutions:

  • Items Not Showing: Check if items are active and in stock
  • Wrong Prices: Verify price group selection and item pricing
  • Missing Images: Ensure images are uploaded in item master
  • Items Not Updating: Refresh catalogue or check item status

Next Steps

After managing catalogue items:

  • Publish the catalogue with updated items
  • Share the catalogue with customers
  • Monitor item performance and customer interest
  • Update items regularly based on inventory changes
