Public Catalogue Setup

The public catalogue feature in EquiBillBook allows you to create and publish online product catalogues that customers can browse on your website. This guide explains how to set up your public catalogue.

What is a Public Catalogue?

A public catalogue is an online showcase of your products that customers can access through a unique URL. It displays your items with images, descriptions, prices, and allows customers to browse, search, and even add items to cart for quotation requests.

Accessing Catalogue Management

To access catalogue management:

  1. Navigate to Catalogue from the main menu
  2. Click on Create New Catalogue or Add Catalogue
  3. You'll see the catalogue setup form

Basic Catalogue Information

Start by entering basic information:

  • Catalogue Name: Enter a descriptive name for your catalogue (e.g., "Summer Collection 2024")
  • Tagline: Add a short tagline or description for your catalogue
  • Unique Slug: Create a unique URL-friendly identifier (e.g., "summer-collection-2024")
  • Branch: Select the branch for this catalogue (if using multi-branch setup)

    • Catalogue Type Selection

    Choose what items to include in your catalogue:

    • All Items: Include all active items from your inventory
    • Selected Items: Manually select specific items to include
    • By Category: Include items from specific categories
    • By Brand: Include items from specific brands

    Display Settings

    Configure what information to display:

    • Show Prices: Display selling prices to customers
    • Show MRP: Display Maximum Retail Price
    • Show Discount: Show discount information if applicable
    • Show Stock: Display stock availability
    • Show Out of Stock: Display items that are out of stock
    • Show Product Code: Display item codes/SKUs
    • Show Images: Display product images

    Business Information Display

    Configure business details to show:

    • Show Business Logo: Display your business logo
    • Show Business Name: Display business name
    • Show Address: Display business address
    • Show Contact: Display contact information
    • Show WhatsApp: Display WhatsApp contact
    • Show Email: Display email address
    • Show GSTIN: Display GST registration number

    Interactive Features

    Enable customer interaction features:

    • Enable WhatsApp Enquiry: Allow customers to enquire via WhatsApp
    • Enable WhatsApp Enquiry for Items: Enable item-level WhatsApp enquiry
    • Enable Add to Cart: Allow customers to add items to cart
    • Cart Disclaimer Text: Add disclaimer text for cart/quotation requests
    • Add to Cart Mode: Configure how cart items are processed (quotation, order, etc.)

    Pricing Configuration

    Set up pricing display:

    • Selling Price Group: Select which price group to use for displaying prices
    • Price Display: Configure how prices are shown (with/without tax)

    Validity Period

    Set catalogue availability:

    • Valid From: Start date for catalogue availability
    • Valid To: End date for catalogue availability
    • Never Expires: Keep catalogue active indefinitely

    Design and Theme

    Customize catalogue appearance:

    • Theme: Select a theme for your catalogue
    • Header Color: Set header background color
    • Header Text Color: Set header text color
    • Button Colors: Customize button colors
    • Background Colors: Set background and text colors
    • Fixed Header: Keep header fixed while scrolling

    SEO Settings

    Optimize for search engines:

    • Meta Title: SEO title for search engines
    • Meta Description: SEO description for search results
    • Google Analytics: Add Google Analytics tracking ID

    Security Settings

    Protect your catalogue if needed:

    • Password Protection: Require password to access catalogue
    • Password: Set access password

    Pagination and Sorting

    Configure display options:

    • Items Per Page: Number of items to show per page
    • Default Sort Order: Default sorting (Popularity, Price Low-High, Price High-Low, Newest)

    Saving and Activating

    After configuring all settings:

    1. Review all settings
    2. Check Is Active to activate the catalogue
    3. Click Save or Create

    Accessing Your Catalogue

    Once created, your catalogue will be accessible at:

    • Public URL: Your website URL + /c/ + your unique slug
    • Example: https://yoursite.com/c/summer-collection-2024

    Best Practices

    • Use descriptive and memorable catalogue names
    • Create unique, SEO-friendly slugs
    • Select appropriate items based on your target audience
    • Enable features that enhance customer experience
    • Set validity periods for time-sensitive catalogues
    • Optimize SEO settings for better search visibility
    • Test the catalogue URL before sharing with customers
    • Keep catalogue information updated regularly

    Next Steps

    After setting up your catalogue:

    • Manage catalogue items and update selections
    • Publish and share the catalogue URL
    • Monitor customer interactions and enquiries
    • Update catalogue content as needed
