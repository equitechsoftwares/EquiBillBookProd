Favorites and Bookmarks

Favorites and Bookmarks in EquiBillBook allow you to save frequently accessed pages, reports, customers, items, and other records for quick access. This guide explains how to use favorites and bookmarks to improve navigation efficiency.

Understanding Favorites and Bookmarks

Favorites and bookmarks enable you to:

  • Save frequently accessed pages
  • Quickly navigate to important records
  • Access reports instantly
  • Bookmark customers, suppliers, items
  • Create shortcuts to important features
  • Improve navigation efficiency

Adding Favorites

To add items to favorites:

Adding Process

  1. Navigate to the page, record, or report you want to favorite
  2. Look for Star Icon or "Add to Favorites" button
  3. Click the star icon or button
  4. Item will be added to favorites
  5. Star icon will be filled/highlighted

What Can Be Favorited

  • Pages: Dashboard, Reports, Settings pages
  • Records: Customers, Suppliers, Items
  • Reports: Frequently used reports
  • Transactions: Important invoices, bills
  • Categories: Item categories, customer categories

Viewing Favorites

To view your favorites:

Access Methods

  • Favorites Menu: Click "Favorites" in main menu
  • Star Icon: Click star icon in header
  • Quick Access: Favorites dropdown in header
  • Sidebar: Favorites sidebar panel

Favorites Display

  • List View: List of favorite items
  • Grouped: Grouped by type (Customers, Reports, etc.)
  • Recent First: Most recently used first
  • Search: Search within favorites

Removing Favorites

To remove items from favorites:

Removal Methods

  • Click Star: Click filled star icon to unfavorite
  • Context Menu: Right-click and select "Remove from Favorites"
  • Favorites Page: Remove from favorites list page
  • Bulk Remove: Remove multiple favorites at once

    Bookmarks

    Using bookmarks for pages:

    Creating Bookmarks

    1. Navigate to page you want to bookmark
    2. Click "Bookmark" or "Add Bookmark"
    3. Enter bookmark name
    4. Select folder (if using folders)
    5. Save bookmark

    Accessing Bookmarks

    • Bookmarks Menu: Access from bookmarks menu
    • Bookmark Bar: Quick access bar
    • Search: Search bookmarks

    Quick Access

    Quick access to favorites and bookmarks:

    Quick Access Features

    • Keyboard Shortcut: Use keyboard shortcut to open favorites
    • Quick Menu: Quick access menu
    • Recent Favorites: Show recently accessed favorites
    • Suggestions: Suggest favorites based on usage

    Common Scenarios

    Scenario 1: Favorite Regular Customers

    Favoriting customers you work with frequently:

    1. Go to customer list
    2. Open customer detail page
    3. Click star icon
    4. Customer added to favorites
    5. Quick access from favorites menu

    Scenario 2: Bookmark Sales Report

    Bookmarking frequently used sales report:

    1. Generate sales report with your filters
    2. Click "Add to Favorites" or bookmark
    3. Enter name (e.g., "Monthly Sales Report")
    4. Report saved
    5. Quick access next time

    Scenario 3: Favorite Quick Items

    Favoriting items you sell frequently:

    1. Go to items list
    2. Open item detail
    3. Click star icon
    4. Item added to favorites
    5. Quick add when creating invoices

    Troubleshooting

    Issue: Favorites Not Saving

    Possible Causes:

    • Browser storage issues
    • Permission problems
    • Session expired

    Solution: Check browser storage, verify permissions, refresh page

    Issue: Favorites Missing

    Possible Causes:

    • Cleared browser data
    • Different browser/device
    • Account switched

    Solution: Favorites are usually account-specific, check if logged into correct account
