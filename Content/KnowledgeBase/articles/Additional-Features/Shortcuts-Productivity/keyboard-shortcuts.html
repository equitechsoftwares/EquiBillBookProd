Keyboard Shortcuts

Keyboard Shortcuts in EquiBillBook allow you to navigate and perform actions quickly using keyboard combinations. This guide lists all available keyboard shortcuts to improve your productivity.

Understanding Keyboard Shortcuts

Keyboard shortcuts enable you to:

  • Navigate faster without using mouse
  • Perform actions quickly
  • Improve productivity
  • Work more efficiently
  • Access features instantly

General Navigation Shortcuts

Global Shortcuts

  • Ctrl + /: Open search or help
  • Ctrl + K: Quick search or command palette
  • Esc: Close dialog or cancel action
  • Enter: Confirm or submit
  • Tab: Move to next field
  • Shift + Tab: Move to previous field

Page Navigation

  • Alt + H: Go to Home/Dashboard
  • Alt + S: Go to Sales
  • Alt + P: Go to Purchase
  • Alt + I: Go to Items/Inventory
  • Alt + A: Go to Accounts
  • Alt + R: Go to Reports
  • Alt + T: Go to Settings

Form and Data Entry Shortcuts

Form Navigation

  • Ctrl + S: Save form
  • Ctrl + Enter: Save and continue
  • Ctrl + N: New record
  • Ctrl + E: Edit current record
  • Ctrl + D: Delete record
  • Ctrl + P: Print

Field Navigation

  • Arrow Keys: Navigate within fields
  • Ctrl + A: Select all text in field
  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + X: Cut
  • Ctrl + Z: Undo
  • Ctrl + Y: Redo

List and Table Shortcuts

List Navigation

  • Arrow Up/Down: Navigate rows
  • Page Up/Down: Scroll page up/down
  • Home: Go to first row
  • End: Go to last row
  • Enter: Open selected record
  • Space: Select/deselect row

Table Operations

  • Ctrl + F: Find in list
  • Ctrl + Shift + F: Advanced search
  • Ctrl + R: Refresh list
  • Ctrl + E: Export selected
  • Delete: Delete selected records

Sales Module Shortcuts

Sales Actions

  • Ctrl + Shift + I: New Sales Invoice
  • Ctrl + Shift + Q: New Quotation
  • Ctrl + Shift + O: New Sales Order
  • Ctrl + Shift + P: New POS Sale
  • Ctrl + Shift + R: New Sales Return

Invoice Operations

  • F2: Add item to invoice
  • F3: Add customer
  • F4: Calculate totals
  • F5: Apply discount
  • F8: Save invoice
  • F9: Print invoice

Purchase Module Shortcuts

Purchase Actions

  • Ctrl + Shift + B: New Purchase Bill
  • Ctrl + Shift + Q: New Purchase Quotation
  • Ctrl + Shift + O: New Purchase Order
  • Ctrl + Shift + R: New Purchase Return

Item and Inventory Shortcuts

Item Actions

  • Ctrl + Shift + I: New Item
  • Ctrl + Shift + A: Stock Adjustment
  • Ctrl + Shift + T: Stock Transfer
  • F6: Search items
  • F7: Quick add item

Customer and Supplier Shortcuts

Quick Actions

  • Ctrl + Shift + C: New Customer
  • Ctrl + Shift + S: New Supplier
  • Ctrl + Shift + P: New Payment

Report Shortcuts

Report Actions

  • Ctrl + Shift + R: Open Reports
  • F10: Generate report
  • F11: Export report
  • F12: Print report

Dialog and Modal Shortcuts

Dialog Operations

  • Esc: Close dialog
  • Enter: Confirm/OK
  • Alt + C: Cancel
  • Alt + S: Save
  • Alt + N: Next
  • Alt + P: Previous

Browser Shortcuts

Standard browser shortcuts also work:

Browser Navigation

  • Ctrl + T: New tab
  • Ctrl + W: Close tab
  • Ctrl + Tab: Next tab
  • Ctrl + Shift + Tab: Previous tab
  • Ctrl + R: Refresh page
  • F5: Refresh
  • Ctrl + F5: Hard refresh

Text Editing Shortcuts

Text Operations

  • Ctrl + B: Bold
  • Ctrl + I: Italic
  • Ctrl + U: Underline
  • Ctrl + F: Find
  • Ctrl + H: Replace

Using Shortcuts Effectively

Tips for Using Shortcuts

  • Learn Gradually: Learn shortcuts gradually, start with most used
  • Practice: Practice using shortcuts regularly
  • Reference: Keep shortcut reference handy
  • Customize: Some shortcuts may be customizable
  • Combine: Combine shortcuts for complex operations

Shortcut Conflicts

Some shortcuts may conflict with browser or system shortcuts:

Handling Conflicts

  • Browser Override: Browser shortcuts may override application shortcuts
  • System Override: System shortcuts may take precedence
  • Focus Required: Some shortcuts require focus on specific elements
  • Context Matters: Shortcuts may work differently in different contexts

Accessibility

Keyboard shortcuts improve accessibility:

Accessibility Benefits

  • Keyboard Navigation: Full functionality via keyboard
  • Faster Access: Quick access to features
  • Reduced Mouse Use: Less dependency on mouse
  • Efficiency: Improved efficiency for all users

Customizing Shortcuts

Some shortcuts may be customizable:

Customization Options

  • Settings: Check Settings for shortcut customization
  • User Preferences: Some shortcuts may be user-configurable
  • Browser Extensions: Browser extensions may allow customization

Best Practices

  • Start Simple: Start with basic navigation shortcuts
  • Practice Daily: Use shortcuts in daily work
  • Learn Module-Specific: Learn shortcuts for modules you use most
  • Keep Reference: Keep shortcut list for reference
  • Share Knowledge: Share shortcuts with team members

Common Shortcut Combinations

Workflow Shortcuts

  • Create Invoice: Ctrl+Shift+I → F3 (select customer) → F2 (add items) → F8 (save)
  • Quick Search: Ctrl+K → type search term → Enter
  • Export Data: Select records → Ctrl+E → Choose format

What's Next?

After learning keyboard shortcuts:

  3. Gradually learn more shortcuts
  4. Improve your productivity
  5. Share shortcuts with your team
