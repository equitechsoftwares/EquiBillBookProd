Dashboard Widgets

Dashboard Widgets in EquiBillBook provide customizable information panels on your dashboard, allowing you to monitor key metrics, view summaries, and access important information at a glance. This guide explains how to use and customize dashboard widgets.

Understanding Dashboard Widgets

Dashboard widgets enable you to:

  • View key metrics at a glance
  • Monitor business performance
  • Access important information quickly
  • Customize your dashboard
  • Track various aspects of your business
  • Improve decision-making

Available Widget Types

EquiBillBook provides various widget types:

1. Summary Widgets

  • Sales Summary: Total sales, today's sales, monthly sales
  • Purchase Summary: Total purchases, monthly purchases
  • Profit Summary: Profit margins, profitability
  • Cash Flow: Cash in, cash out, net cash flow

2. Chart Widgets

  • Sales Chart: Sales trends over time
  • Profit Chart: Profit trends
  • Category Chart: Sales by category
  • Comparison Chart: Period comparisons

3. List Widgets

  • Recent Invoices: Latest sales invoices
  • Pending Orders: Pending sales/purchase orders
  • Low Stock Items: Items with low stock
  • Recent Customers: Recently added customers

4. Status Widgets

  • Outstanding Receivables: Amounts due from customers
  • Outstanding Payables: Amounts due to suppliers
  • Stock Status: Stock levels and alerts
  • Task Status: Pending tasks and reminders

5. Quick Action Widgets

  • Quick Create: Quick access to create records
  • Shortcuts: Shortcut links to features
  • Recent Items: Recently accessed items

Accessing Dashboard Widgets

To access and manage widgets:

Dashboard Access

  1. Go to Dashboard or Home
  2. You'll see default widgets
  3. Click "Customize" or "Edit Dashboard" to manage widgets

Adding Widgets

To add widgets to your dashboard:

Adding Process

  1. Go to Dashboard
  2. Click "Add Widget" or "Customize"
  3. Browse available widgets
  4. Select widget you want to add
  5. Configure widget settings
  6. Click "Add" or "Save"
  7. Widget will appear on dashboard

Widget Selection

  • Browse Categories: Browse widgets by category
  • Search Widgets: Search for specific widgets
  • Preview: Preview widget before adding
  • Recommended: View recommended widgets

Configuring Widgets

Configure widget settings:

Configuration Options

  • Title: Customize widget title
  • Date Range: Set date range for data
  • Filters: Apply filters to widget data
  • Size: Set widget size (small, medium, large)
  • Refresh Rate: Set auto-refresh interval
  • Display Options: Configure display settings

Widget Settings

  • Period: Today, This Week, This Month, Custom
  • Branch: Select branch for branch-specific widgets
  • Category: Filter by category
  • Format: Number format, currency format

Arranging Widgets

Arrange widgets on dashboard:

Drag and Drop

  • Move Widgets: Drag widgets to reposition
  • Reorder: Change widget order
  • Grid Layout: Arrange in grid layout

Layout Options

  • Grid Size: Adjust grid size
  • Columns: Set number of columns
  • Spacing: Adjust spacing between widgets

Resizing Widgets

Resize widgets to fit your needs:

Resize Options

  • Small: Compact widget size
  • Medium: Standard widget size
  • Large: Expanded widget size
  • Custom: Custom size if available

Resize Process

  1. Hover over widget
  2. Click resize handle or icon
  3. Drag to resize
  4. Or select size from menu
  5. Widget will resize

Removing Widgets

Remove widgets from dashboard:

Removal Process

  1. Hover over widget
  2. Click "Remove" or "Delete" icon
  3. Or right-click and select "Remove"
  4. Confirm removal
  5. Widget will be removed

Refreshing Widgets

Refresh widget data:

Refresh Options

  • Manual Refresh: Click refresh icon on widget
  • Auto Refresh: Set automatic refresh interval
    • Refresh All: Refresh all widgets at once

Widget Interactions

Interact with widgets:

Click Actions

  • Drill Down: Click to view details
  • Open Report: Click to open full report
  • Navigate: Click to navigate to related page
  • Filter: Click to apply filters

Widget Customization

Customize widget appearance:

Appearance Options

  • Color Theme: Change widget colors
  • Chart Type: Change chart type (bar, line, pie)
  • Display Style: Change display style
  • Icons: Show/hide icons

Dashboard Layouts

Choose dashboard layout:

Layout Options

  • Default Layout: Standard layout
  • Compact Layout: Compact view
  • Wide Layout: Wide view
  • Custom Layout: Custom arrangement

Best Practices

  • Focus on Key Metrics: Add widgets for most important metrics
  • Organize Logically: Arrange widgets logically
  • Regular Review: Review and update widgets regularly
  • Balance Information: Balance different types of information
  • Keep It Clean: Don't overcrowd dashboard
  • Update Periods: Set appropriate date ranges

Troubleshooting

Issue: Widget Not Loading

Possible Causes:

  • Data not available
  • Permission issues
  • Configuration errors

Solution: Check data availability, verify permissions, reconfigure widget

Issue: Widget Shows Wrong Data

Possible Causes:

  • Incorrect filters
  • Wrong date range
  • Cache issues

Solution: Check filters, verify date range, refresh widget

What's Next?

After setting up dashboard widgets:

  1. Monitor key metrics regularly
  2. Adjust widgets based on needs
  3. Add/remove widgets as required
  4. Keep dashboard updated
  5. Use widgets for decision-making

Well-configured dashboard widgets provide valuable insights and improve your ability to monitor and manage your business in EquiBillBook.
