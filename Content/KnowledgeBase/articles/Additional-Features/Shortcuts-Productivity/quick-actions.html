Quick Actions

Quick Actions in EquiBillBook provide fast access to frequently used functions and common tasks. This guide explains how to use quick actions to improve your productivity.

Understanding Quick Actions

Quick actions enable you to:

  • Access common functions quickly
  • Perform frequent tasks faster
  • Navigate to important features
  • Create records quickly
  • Improve workflow efficiency
  • Save time on repetitive tasks

Quick Action Locations

Quick actions are available in various places:

Dashboard Quick Actions

  • Quick Create Buttons: Fast access to create new records
  • Action Cards: Quick access cards for common tasks
  • Shortcut Links: Direct links to frequently used features

List View Quick Actions

  • Action Buttons: Quick action buttons in list views
  • Context Menu: Right-click context menu actions
  • Bulk Actions: Actions for multiple selected items

Detail View Quick Actions

  • Action Bar: Quick action buttons on detail pages
  • Action Menu: Dropdown menu with actions
  • Floating Actions: Floating action buttons

Using Quick Actions

How to use quick actions:

From Dashboard

  1. Go to Dashboard
  2. Find quick action buttons or cards
  3. Click on the desired action
  4. Action will be performed or form will open

From List View

  1. Navigate to any list (Customers, Items, etc.)
  2. Look for quick action buttons
  3. Or right-click on a record
  4. Select action from context menu

From Detail View

  1. Open any record detail page
  2. Find action buttons or menu
  3. Click on desired action
  4. Action will be performed

Quick Action Menu

Access quick actions from menu:

Menu Access

  • Plus Icon: Click + icon for quick create menu
  • Action Menu: Three-dot menu for actions
  • Context Menu: Right-click for context menu

Quick Action Examples

Example 1: Quick Invoice Creation

Creating invoice quickly:

  1. Click "+" icon or "New Invoice" button
  2. Quick invoice form opens
  3. Select customer (or quick add)
  4. Add items quickly
  5. Save invoice

Example 2: Quick Customer Payment

Recording payment quickly:

  1. Go to customer detail page
  2. Click "Quick Payment" button
  3. Payment form opens with customer pre-filled
  4. Enter amount and details
  5. Save payment

Example 3: Quick Item Add

Adding item to invoice quickly:

  1. While creating invoice
  2. Click "Quick Add Item" or press shortcut
  3. Quick item form opens
  4. Enter basic item details
  5. Item is added to invoice
