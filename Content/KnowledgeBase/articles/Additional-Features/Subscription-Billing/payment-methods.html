Payment Methods

Managing payment methods allows you to add, update, and manage how you pay for your EquiBillBook subscription. This guide covers all aspects of payment method management.

What are Payment Methods?

Payment methods are the ways you pay for your EquiBillBook subscription, such as credit cards, debit cards, bank accounts, or other payment options accepted by the platform.

Accessing Payment Methods

To manage payment methods:

  1. Navigate to Subscription or Billing
  2. Click on Payment Methods or Payment Settings
  3. View your current payment methods
  4. Manage payment options

Supported Payment Methods

Common payment methods include:

  • Credit Cards: Visa, Mastercard, American Express, etc.
  • Debit Cards: Debit cards from various banks
  • Bank Transfer: Direct bank transfers
  • Online Payment Gateways: Razorpay, Stripe, PayPal, etc.
  • UPI: Unified Payments Interface (if available)
  • Net Banking: Internet banking
  • Wallet Payments: Digital wallets (if available)

Viewing Current Payment Methods

Your payment methods page shows:

  • Active Methods: Currently active payment methods
  • Default Method: Primary payment method
  • Method Details: Last 4 digits, expiry, etc.
  • Status: Active, expired, or failed

    • Adding a Payment Method

    To add a new payment method:

    1. Click Add Payment Method or Add Card
    2. Select payment method type
    3. Enter payment details
    4. For cards: Enter card number, expiry, CVV, name
    5. For bank: Enter bank account details
    6. Verify payment method (if required)
    7. Save the payment method

    Setting Default Payment Method

    To set a default payment method:

    1. Go to payment methods
    2. Find the payment method you want as default
    3. Click Set as Default or Make Default
    4. Default method is used for automatic payments

    Updating Payment Method

    To update a payment method:

    1. Find the payment method to update
    2. Click Edit or Update
    3. Update the details (card number, expiry, etc.)
    4. Save the changes
    5. Verify if required

    Removing Payment Method

    To remove a payment method:

    1. Go to payment methods
    2. Find the method to remove
    3. Click Remove or Delete
    4. Confirm removal
    5. Ensure you have another active method if this was default

    Payment Method Security

    Security features for payment methods:

    • Encryption: Payment details are encrypted
    • PCI Compliance: Secure payment processing
    • Tokenization: Card details are tokenized
    • Verification: Payment methods may require verification

    Payment Verification

    Some payment methods require verification:

    • Card Verification: Small charge or authorization hold
    • Bank Verification: Bank account verification process
    • Email Verification: Verify email for payment method
    • Verification Status: Check verification status

      • Automatic Payments

      Setting up automatic payments:

      • Enable Auto-Pay: Turn on automatic payments
      • Default Method: Use default payment method
      • Payment Schedule: Payments on renewal dates
      • Payment Notifications: Get notified before payments

      Failed Payments

      If a payment fails:

      • Notification: You'll be notified of failure
      • Retry Payment: Retry with same or different method
      • Update Method: Update payment method if expired
      • Contact Support: Contact support if issues persist

      Expired Payment Methods

      Managing expired payment methods:

      • Expiry Notifications: Get notified before expiry
      • Update Expired Cards: Update card details
      • Remove Expired Methods: Remove methods no longer valid
      • Set New Default: Set new default if old one expired

        • Payment Method Limits

        Understanding payment limits:

        • Transaction Limits: Limits per transaction
        • Daily Limits: Daily payment limits
        • Method-Specific Limits: Limits for specific methods
        • Bank Limits: Bank-imposed limits

          • Billing Address

          Managing billing address:

          • Billing Address: Address associated with payment
          • Update Address: Update billing address
          • Address Verification: Verify billing address
          • Tax Information: Billing address for tax purposes

            • Best Practices

            • Keep at least one active payment method
            • Set a default payment method
            • Update expired cards promptly
            • Enable auto-pay for uninterrupted service
            • Monitor payment method status
            • Keep payment methods secure
            • Verify payment methods when added
            • Remove unused payment methods

            Security Tips

            • Never share payment method details
            • Use secure networks when adding methods
            • Monitor payment activity regularly
            • Report suspicious activity immediately
            • Keep payment method information updated

            Next Steps

            After managing payment methods:

            • Ensure default method is set
            • Enable auto-pay if desired
            • Monitor payment status
            • Keep payment methods updated
