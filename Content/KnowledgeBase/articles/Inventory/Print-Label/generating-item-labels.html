Generating Item Labels

Item labels are essential for inventory management, product identification, and efficient stock tracking. EquiBillBook allows you to generate professional labels with barcodes, product information, and pricing details.

Accessing the Print Label Module

To generate item labels:

  1. Navigate to the Items module from the main menu
  2. Click on Print Labels or Label Printing
  3. Select the business location/branch (if using multi-branch setup)

Selecting Items for Label Generation

You can select items in several ways:

  • Search by SKU Code: Enter the SKU code in the search field
  • Search by Item Name: Type the item name to find products
  • Browse Items: Select items from the available list
  • Select Multiple Items: Add multiple items to generate labels in batch

    • Adding Items to Label List

    Once you find an item:

    1. Enter the SKU code or item name in the search field
    2. The system will display matching items
    3. Select the item from the dropdown or search results
    4. The item will be added to the label generation table
    5. Repeat for all items you want to label

    Configuring Label Details

    For each item in the label list, configure:

    • Unit: Select the unit of measurement (if multiple units are available)
    • Number of Labels: Specify how many labels to generate for each item
    • Packaging Date: Enter the packaging or manufacturing date (if applicable)
    • Price: The price will be automatically populated based on item settings

      • Item Information Displayed

      Each label can include:

      • Item Name: Product name
      • SKU Code: Stock keeping unit code
      • Variation: Product variation details (if applicable)
      • Price: Selling price (ex-tax or inc-tax based on settings)
      • Barcode: Automatically generated barcode
      • Branch Name: Business location name
      • Packaging Date: Date information (if enabled)

      Label Preview

      Before printing, you can:

      • Preview how labels will look
      • Check label layout and formatting
      • Verify all information is correct
      • See how labels will appear on paper
      • Make adjustments if needed

        • Label Quantity Management

        Set the number of labels:

        • Single Label: Generate one label per item
        • Multiple Labels: Generate multiple copies for the same item
        • Batch Quantity: Set quantity for bulk labeling
        • Custom Count: Enter specific number of labels needed

        Unit Selection

        If items have multiple units:

        • Select the appropriate unit from the dropdown
        • Price will adjust based on unit selection
        • Label will display the selected unit
        • Ensure unit matches your labeling needs

        Packaging Date

        For items with packaging dates:

        • Enter the packaging or manufacturing date
        • Date will be displayed on the label (if enabled in format)
        • Useful for perishable items or quality tracking
        • Can be left blank if not applicable

        Removing Items from List

        To remove items:

        1. Find the item in the label generation table
        2. Click the remove or delete button for that item
        3. Item will be removed from the label list

        Clearing All Items

        To start fresh:

        • Click the clear or reset button
        • All items will be removed from the list
        • You can start adding items again

        Label Generation Process

        After configuring all items:

        1. Review all items in the list
        2. Verify quantities and details
        3. Check label format settings
        4. Click Generate Labels or Preview
        5. Review the label preview
        6. Proceed to printing

        Stock Availability

        The system will:

        • Show only items with available stock (if stock tracking is enabled)
        • Display items from selected branch/location
        • Filter items based on stock quantity
        • Prevent labeling of out-of-stock items (if configured)

        Item Types Supported

        You can generate labels for:

        • Single Products: Standard items with single SKU
        • Variable Products: Items with variations (size, color, etc.)
        • Stock Tracked Items: Items with inventory management
        • Non-Stock Items: Services or non-inventory items

        Tips for Generating Labels

        • Use SKU codes for faster item selection
        • Generate labels in batches for efficiency
        • Verify item details before generating
        • Check label format settings before printing
        • Generate preview before printing large quantities
        • Keep label format consistent for professional appearance
        • Update packaging dates regularly for accuracy

        Common Scenarios

        New Product Launch:

        • Generate labels for all new items
        • Ensure barcodes are properly generated
        • Verify pricing information

        Price Updates:

        • Regenerate labels when prices change
        • Update existing labels with new pricing
        • Maintain accurate price display

        Replacement Labels:

        • Generate replacement labels for damaged ones
        • Maintain same format and information
        • Ensure consistency across all labels

        Troubleshooting

        If you encounter issues:

        • Item not found: Verify SKU code or item name spelling
        • No stock available: Check stock levels and branch selection
        • Price not showing: Verify item pricing configuration
        • Barcode missing: Check barcode generation settings
        • Wrong unit: Select correct unit from dropdown

        Best Practices

        • Generate labels in batches for efficiency
        • Verify all information before printing
        • Use consistent label formats across products
        • Keep packaging dates updated
        • Regularly review and update label information
        • Maintain backup of label configurations
        • Test label format before bulk printing

        Generating item labels in EquiBillBook is straightforward and helps maintain organized inventory management. Use the preview feature to ensure labels meet your requirements before printing.
