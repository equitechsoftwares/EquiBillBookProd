Label Format Configuration

Label format configuration allows you to customize how labels appear, what information is displayed, and how elements are arranged. Proper configuration ensures labels meet your business needs and are professional and functional.

Accessing Format Configuration

To configure label formats:

  1. Navigate to Items â†’ Print Labels
  2. Go to Label Settings or Format Configuration
  3. Access format settings and options

Label Dimensions

Configure label size:

  • Width: Set label width (in inches, millimeters, or centimeters)
  • Height: Set label height
  • Standard Sizes: Choose from common label sizes (1x1, 2x1, 4x2 inches, etc.)
  • Custom Sizes: Define custom dimensions

Match dimensions to your label stock size for proper printing.

Font Size Configuration

Set font sizes for different elements:

  • Branch Name Size: Font size for business location name (in pixels)
  • Product Name Size: Font size for item name
  • Product Variation Size: Font size for variation details
  • Product Price Size: Font size for price information
  • Packaging Date Size: Font size for date display

Adjust sizes based on label dimensions and readability requirements.

Field Visibility Settings

Control which fields appear on labels:

  • Show Branch Name: Display or hide branch/location name
  • Show Product Variation: Show variation when available (size, color, etc.)
  • Show Packaging Date: Display packaging or manufacturing date
  • Show Price: Include or exclude price on label
  • Show Barcode: Display barcode on label

Enable only fields relevant to your labeling needs.

Layout Configuration

Configure label layout:

  • Text Alignment: Left, center, or right align text
  • Vertical Spacing: Space between elements
  • Element Order: Arrange fields in desired sequence
  • Barcode Position: Where barcode appears (top, bottom, center)

Barcode Configuration

Configure barcode settings:

  • Barcode Type: Select format (Code 128, EAN-13, UPC, etc.)
  • Barcode Height: Set vertical size
  • Barcode Width: Set horizontal size (auto or fixed)
  • Show Barcode Value: Display numbers below barcode
  • Barcode Position: Where barcode appears on label

Price Display Configuration

Configure how price is displayed:

  • Price Format: Ex-tax or inc-tax
  • Currency Symbol: Display currency symbol
  • Price Label: Text before price (e.g., "Price:", "MRP:")
  • Decimal Places: Number of decimal places

Text Formatting

Configure text appearance:

  • Font Family: Select font type (if customizable)
  • Font Weight: Bold, normal, or light
  • Text Color: Set text color (if supported)
  • Line Height: Spacing between lines

Spacing and Margins

Configure spacing:

  • Top Margin: Space from top edge
  • Bottom Margin: Space from bottom edge
  • Left Margin: Space from left edge
  • Right Margin: Space from right edge
  • Element Spacing: Space between label elements

Multi-Line Configuration

For long text fields:

  • Line Breaks: Enable automatic line breaks
  • Text Wrapping: Wrap long text to multiple lines
  • Max Lines: Maximum number of lines per field
  • Text Truncation: Truncate text if too long

Label Template Selection

Choose or create templates:

  • Default Template: Use system default format
  • Custom Templates: Create and use custom templates
  • Template Switching: Switch between templates
  • Template Saving: Save format as new template

Preview Configuration

Use preview to verify settings:

  • Preview shows how labels will appear
  • Test with sample item data
  • Verify all elements are visible
  • Check label dimensions
  • Ensure readability

Format Presets

Common format presets:

  • Retail Label: Product name, price, barcode
  • Warehouse Label: SKU, barcode, minimal text
  • Inventory Label: All details, comprehensive format
  • Simple Label: Basic information only

Select preset or customize as needed.

Configuration Best Practices

  • Maintain consistency across labels

  4. Verify all elements appear correctly
  5. Check barcode scanning
  6. Ensure readability

Common Format Scenarios

Small Labels (1x1 inch):

  • Minimal information
  • Small font sizes
  • Barcode and SKU only

Medium Labels (2x1 inch):

  • Product name and price
  • Barcode included
  • Moderate font sizes

Large Labels (4x2 inch):

  • All information included
  • Larger font sizes
  • Comprehensive details

Format Troubleshooting

Common issues:

  • Text cut off: Adjust label dimensions or font sizes
  • Elements overlapping: Increase spacing or reduce font sizes
  • Barcode too small: Increase barcode size
  • Poor readability: Increase font sizes or improve contrast
  • Wrong dimensions: Verify label stock size matches settings
