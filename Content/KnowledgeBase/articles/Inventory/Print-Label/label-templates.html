Label Templates

Label templates in EquiBillBook allow you to create and customize professional label layouts for your products. Templates define what information appears on labels, how it's arranged, and the overall design.

Understanding Label Templates

Label templates control:

  • Layout and design of labels
  • Which fields are displayed
  • Font sizes and styles
  • Label dimensions
  • Barcode placement and size
  • Overall appearance and branding

Accessing Label Templates

To manage label templates:

  1. Navigate to Items â†’ Print Labels
  2. Go to Label Settings or Template Configuration
  3. View existing templates or create new ones

Default Template

EquiBillBook comes with a default label template that includes:

  • Branch name
  • Product name
  • Product variation (if applicable)
  • Price information
  • Barcode
  • Packaging date (if enabled)

Template Elements

Available elements for label templates:

  • Branch Name: Business location name
  • Product Name: Item name
  • Product Variation: Variation details (size, color, etc.)
  • SKU Code: Stock keeping unit
  • Price: Selling price (ex-tax or inc-tax)
  • Barcode: Product barcode
  • Packaging Date: Manufacturing or packaging date
  • QR Code: Quick response code (if enabled)

Creating Custom Templates

To create a new template:

  1. Access label template settings
  2. Click Create New Template or Add Template
  3. Enter template name
  4. Configure template settings
  5. Select which fields to include
  6. Set font sizes and styles
  7. Configure label dimensions
  8. Save the template

Template Configuration Options

Configure template settings:

  • Label Width: Set label width in inches or millimeters
  • Label Height: Set label height
  • Font Sizes: Configure font size for each element
  • Field Visibility: Show or hide specific fields
  • Layout: Arrange elements vertically or horizontally
  • Alignment: Center, left, or right align text

Font Size Configuration

Set font sizes for different elements:

  • Branch Name Size: Font size for branch name
  • Product Name Size: Font size for product name
  • Product Variation Size: Font size for variation
  • Price Size: Font size for price
  • Packaging Date Size: Font size for date

Adjust sizes based on label dimensions and readability needs.

Field Visibility Settings

Control which fields appear on labels:

  • Show Branch Name: Display or hide branch name
  • Show Product Variation: Show variation when available
  • Show Packaging Date: Display packaging date
  • Show Price: Include or exclude price
  • Show Barcode: Display barcode on label

Label Dimensions

Configure label size:

  • Standard Sizes: Common label sizes (1x1, 2x1, 4x2 inches, etc.)
  • Custom Sizes: Define custom dimensions
  • Metric Units: Use millimeters or centimeters
  • Imperial Units: Use inches

Match label dimensions to your label stock size.

Barcode Configuration

Configure barcode settings:

  • Barcode Type: Code 128, EAN-13, UPC, etc.
  • Barcode Size: Height and width of barcode
  • Barcode Position: Where barcode appears on label
  • Show Barcode Value: Display numbers below barcode

Template Preview

Before saving:

  • Preview how the template will look
  • Test with sample item data
  • Verify all elements are visible
  • Check label dimensions
  • Ensure readability

Multiple Templates

You can create multiple templates for:

  • Different Product Types: Different templates for different categories
  • Different Label Sizes: Templates for various label dimensions
  • Different Use Cases: Retail labels, warehouse labels, etc.
  • Different Branches: Branch-specific templates

Selecting Templates

When generating labels:

  • Choose the appropriate template
  • Template determines label layout
  • Can switch templates before printing
  • Preview shows selected template

Editing Templates

To modify existing templates:

  1. Access template management
  2. Select the template to edit
  3. Make changes to settings
  4. Preview changes
  5. Save updated template

Deleting Templates

To remove templates:

  • Select template to delete
  • Confirm deletion
  • Note: Default template may not be deletable
  • Ensure no labels are using the template before deleting

Template Best Practices

  • Keep templates simple and readable
  • Use appropriate font sizes for label size
  • Ensure barcode is scannable
  • Test templates before bulk printing
  • Maintain consistency across templates
  • Include essential information only
  • Consider label printer capabilities

Common Template Scenarios

Retail Labels:

  • Include product name, price, and barcode
  • Larger font for price visibility
  • Professional appearance

Warehouse Labels:

  • Focus on SKU and barcode
  • Smaller font, more information
  • Durable for storage conditions

Inventory Labels:

  • Include all product details
  • Packaging date if applicable
  • Clear identification information

Template Troubleshooting

Common issues:

  • Text cut off: Adjust label dimensions or font sizes
  • Barcode not scanning: Check barcode size and quality
  • Layout issues: Verify label dimensions match printer settings
  • Missing fields: Check field visibility settings
  • Poor readability: Increase font sizes or adjust layout

Exporting and Importing Templates

Some systems allow:

  • Exporting templates for backup
  • Importing templates from other systems
  • Sharing templates between branches
  • Backing up template configurations

Template Maintenance

  • Regularly review template effectiveness
  • Update templates when requirements change
  • Test templates after system updates
  • Keep templates organized and named clearly
  • Document template purposes and use cases

Label templates in EquiBillBook provide flexibility to create labels that meet your specific business needs. Experiment with different configurations to find the best layout for your products.
