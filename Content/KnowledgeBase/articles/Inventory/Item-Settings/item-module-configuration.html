Item Module Configuration

The Item Module Configuration in EquiBillBook allows you to customize and control various aspects of your item and inventory management. This comprehensive guide will help you understand and configure all the essential item settings to optimize your inventory operations.

Accessing Item Settings

To configure your item module settings:

  1. Navigate to Settings â†’ Item Settings
  2. You'll see various configuration sections for different aspects of item management
  3. Make your changes and click "Update" to apply the settings

Core Item Module Features

EquiBillBook's Item Module includes several configurable features:

  • Stock Tracking: Enable or disable stock/inventory tracking for items
  • Brands: Enable or disable brand management for items
  • Categories: Enable category, subcategory, and sub-subcategory management
  • Product Variations: Enable or disable product variations and attributes
  • Units of Measurement: Configure primary, secondary, tertiary, and quaternary units
  • Item Expiry: Enable or disable expiry date tracking for items
  • Barcode: Enable or disable barcode functionality
  • Item Images: Enable or disable item image uploads
  • Combo Products: Enable or disable combo/bundle product creation
  • Print Labels: Enable or disable label printing functionality

Category Configuration

Configure category management for organizing your items:

  • Enable Category: Allow categorization of items into main categories
  • Enable Sub Category: Allow subcategorization within main categories
  • Enable Sub Sub Category: Allow further categorization with sub-subcategories
  • Categories help organize items for better inventory management and reporting
  • Useful for businesses with diverse product ranges

Brand Management

Configure brand management for your items:

  • Enable Brands: Allow brand assignment to items
  • Track items by manufacturer or brand name
  • Useful for businesses selling branded products
  • Helps in brand-wise reporting and analysis

Unit of Measurement Configuration

Configure units of measurement for items:

  • Default Unit: Set the default primary unit of measurement
  • Enable Secondary Unit: Allow secondary unit conversion (e.g., Box, Carton)
  • Enable Tertiary Unit: Allow tertiary unit conversion
  • Enable Quaternary Unit: Allow quaternary unit conversion
  • Multiple units help manage items sold in different quantities
  • Automatic conversion between units simplifies inventory management

Product Variation Settings

Configure product variations and attributes:

  • Enable Product Variation: Allow creating item variations (size, color, etc.)
  • Create variations for items with different attributes
  • Track stock separately for each variation
  • Useful for businesses selling items with multiple options

Item Expiry Management

Configure expiry date tracking for items:

  • Enable Item Expiry: Track expiry dates for items
  • Expiry Type: Configure how expiry is tracked (Manufacturing Date, Expiry Date)
  • On Item Expiry: Configure action when item expires (Stop Selling, Alert Only)
  • Stop Selling Before Days: Set days before expiry to stop selling
  • Expiry Date Format: Configure date format for expiry tracking
  • Essential for businesses dealing with perishable items

Stock Management Features

Configure stock-related features:

  • Stock Accounting Method: Choose FIFO, LIFO, or Average cost method
  • Enable Stock Transfer: Allow transferring stock between locations
  • Enable Stock Adjustment: Allow manual stock adjustments
  • Enable Lot No: Track items by lot numbers
  • Enable Racks/Row/Position: Track physical location of items in warehouse

Pricing Configuration

Configure pricing-related settings:

  • Default Profit Percent: Set default profit percentage for items
  • Enable MRP: Enable Maximum Retail Price tracking
  • Enable Selling Price Group: Allow different prices for different customer groups
  • Enable Tax/Price Info: Show tax and price information on item forms

Additional Features

Configure additional item features:

  • Enable Barcode: Allow barcode generation and scanning
  • Enable Item Image: Allow uploading images for items
  • Enable Product Description: Allow detailed product descriptions
  • Enable Combo Product: Allow creating combo/bundle products
  • Enable Print Label: Allow printing labels for items
  • Enable Warranty: Track warranty information for items
  • Enable SALT: Enable SALT (if applicable to your business)

Best Practices for Item Module Configuration

  • Start with essentials: Enable only the features you currently need
  • Plan your categories: Set up a clear category structure before adding items
  • Configure units first: Set up all required units before creating items
  • Enable stock tracking: Enable stock tracking if you manage inventory
  • Review regularly: Periodically review settings as your business grows
  • Test changes: Test configuration changes before applying to production
  • Document settings: Keep a record of your configuration choices
  • Train your team: Ensure all users understand the configured settings

Saving Your Configuration

After making all necessary changes:

  1. Review all settings carefully
  2. Click the "Update" button
  3. Wait for the confirmation message
  4. Test the changes by creating a test item to verify all settings

Troubleshooting

Settings not saving?

  • Check that you have proper permissions to modify item settings
  • Ensure all required fields are filled
  • Check for validation errors
  • Try refreshing the page and saving again

Changes not reflecting in items?

  • Settings typically apply to new items only
  • Existing items may need to be updated manually
  • Clear browser cache if needed
  • Log out and log back in

Features not visible?

  • Ensure the features are enabled in item settings
  • Check if you have proper permissions
  • Verify that the module is activated in your subscription
