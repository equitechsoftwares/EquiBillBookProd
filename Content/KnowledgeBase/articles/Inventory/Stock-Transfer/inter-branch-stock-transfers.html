Inter-branch Stock Transfers

Inter-branch Stock Transfers in EquiBillBook enable you to move inventory between different branches or locations of your business. This feature is essential for multi-location businesses to maintain optimal inventory levels across all branches and meet customer demand efficiently.

What are Inter-branch Transfers?

Inter-branch transfers are stock movements between different branches or locations of the same organization. These transfers help:

  • Balance inventory across multiple branches
  • Replenish stock at branches with low inventory
  • Move excess stock from one branch to another
  • Fulfill customer orders from nearest branch
  • Optimize inventory distribution
  • Reduce overall inventory carrying costs

Prerequisites for Inter-branch Transfers

Before transferring stock between branches:

  • Multiple Branches: At least two branches must be configured
  • Branch Activation: Both source and destination branches must be active
  • Stock Tracking: Stock tracking must be enabled for items
  • Item Mapping: Items must be mapped to branches
  • Permissions: Users must have transfer permissions for both branches

Creating Inter-branch Transfers

To create an inter-branch transfer:

Step 1: Access Transfer Creation

  • Navigate to Inventory → Stock Transfer → Create
  • Select transfer type as "Inter-branch Transfer"

Step 2: Select Branches

  • Source Branch: Select branch from where stock will be transferred
  • Destination Branch: Select branch where stock will be received
  • Both branches must be different and active

Step 3: Add Transfer Items

  • Select items available at source branch
  • Enter transfer quantities
  • System validates stock availability at source
  • View projected stock levels at both branches

Step 4: Configure Transfer Details

  • Enter transfer date and expected delivery
  • Add transfer reason (replenishment, customer order, etc.)
  • Add notes and special instructions
  • Set transfer priority if applicable

Step 5: Submit Transfer

  • Review all transfer details
  • Submit for approval (if required)
  • Or save as draft for later processing

Inter-branch Transfer Workflow

The complete workflow:

1. Transfer Initiation

  • Transfer created at source branch
  • Status: "Pending" or "Draft"
  • Stock not yet affected

2. Approval (If Required)

  • Transfer submitted for approval
  • Branch managers or designated approvers review
  • Approval granted or rejected

3. Stock Deduction

  • Upon approval, stock deducted from source branch
  • Status changes to "In Transit"
  • Source branch stock levels updated

4. Physical Movement

  • Stock physically moved between branches
  • Transfer document accompanies stock
  • Status remains "In Transit"

5. Receipt at Destination

  • Destination branch confirms receipt
  • Quantities verified
  • Status changes to "Received"

6. Stock Addition

  • Stock added to destination branch
  • Destination branch stock levels updated
  • Status changes to "Completed"

Common Scenarios

Scenario 1: Replenishing Low Stock

  • Branch A has low stock of an item
  • Branch B has excess stock of the same item
  • Transfer stock from Branch B to Branch A
  • Balance inventory across branches

Scenario 2: Customer Order Fulfillment

  • Customer order at Branch A
  • Item not available at Branch A
  • Transfer from Branch B to Branch A
  • Fulfill order from nearest location

Scenario 3: Seasonal Stock Redistribution

  • Seasonal items at one branch
  • Transfer to branches with higher demand
  • Optimize inventory for seasonal sales
  • Reduce excess inventory

Scenario 4: New Branch Setup

  • Setting up new branch
  • Transfer initial stock from existing branches
  • Establish inventory at new location
  • Begin operations with adequate stock

Troubleshooting

Cannot select destination branch?

  • Verify destination branch is active
  • Check user permissions for destination branch
  • Ensure branch is properly configured
  • Verify branch access settings

Stock not available for transfer?

  • Check stock levels at source branch
  • Verify item is mapped to source branch
  • Check if stock is reserved for other transactions
  • Review item and branch configurations

Transfer not updating destination stock?

  • Verify transfer status is "Completed"
  • Check if receipt was confirmed
  • Verify destination branch stock tracking
  • Review transfer workflow completion
