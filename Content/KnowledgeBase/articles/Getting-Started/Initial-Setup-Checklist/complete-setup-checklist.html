Complete Setup Checklist

This comprehensive checklist will guide you through the essential setup steps to get your EquiBillBook account fully configured and ready for use.

Account Setup (Section 1.1)

  • ✓ Create your EquiBillBook account
  • ✓ Verify your email address
  • ✓ Set up a strong password
  • ✓ Complete your user profile
  • ✓ Familiarize yourself with the dashboard

Organization Configuration (Section 1.2)

  • ✓ Set up organization details (business name, type, etc.)
  • ✓ Upload your business logo
  • ✓ Configure business address and contact information
  • ✓ Set up GST number and tax details
  • ✓ Configure financial year settings
  • ✓ Set currency and regional settings
  • ✓ Configure multi-branch setup (if applicable)

Initial Setup Tasks (Section 1.3)

  • ✓ Complete the setup checklist (you're here!)
  • ✓ Import data from other software (if needed)
  • ✓ Set up opening balances for accounts
  • ✓ Configure user permissions and roles
  • ✓ Understand and activate required modules

Module-Specific Setup

Depending on your business needs, complete setup for:

  • ✓ Sales Module: Configure invoice numbering, payment terms, tax settings
  • ✓ Purchase Module: Set up purchase numbering, vendor payment terms
  • ✓ Inventory Module: Enable stock tracking, set up item categories, configure units of measurement
  • ✓ Accounts Module: Set up chart of accounts, configure accounting periods
  • ✓ Banking Module: Add bank accounts, set opening balances
  • ✓ Restaurant Module: Configure tables, floors, booking settings (if applicable)

Data Entry

  • ✓ Add customers and their details
  • ✓ Add suppliers and their details
  • ✓ Create items/products with pricing and tax information
  • ✓ Set up opening stock (if using inventory)
  • ✓ Configure payment methods

Settings & Preferences

  • ✓ Configure invoice templates
  • ✓ Set up email/SMS/WhatsApp notifications
  • ✓ Configure online payment gateways (if applicable)
  • ✓ Set up backup and security settings
  • ✓ Customize display and print settings

Testing & Verification

  • ✓ Create a test invoice to verify settings
  • ✓ Test payment recording
  • ✓ Verify tax calculations
  • ✓ Test report generation
  • ✓ Verify email/SMS notifications are working

Quick Setup Tips

  • Start Small: Don't try to configure everything at once. Start with the essentials and add more features as needed.
  • Use Templates: EquiBillBook provides default templates that you can customize later.
  • Import When Possible: If you have existing data, use the import feature to save time.
  • Test Before Going Live: Create test transactions to ensure everything works as expected.
  • Get Help: Don't hesitate to contact support if you need assistance during setup.

Priority Setup Order

  1. Day 1: Account setup, organization details, basic settings
  2. Day 2: Add customers, suppliers, and items
  3. Day 3: Configure modules you'll use, set up opening balances
  4. Day 4: Test transactions, verify calculations, customize templates
  5. Day 5: Train users, set up permissions, go live!
