User Profile Setup

Setting up your user profile helps personalize your EquiBillBook experience and ensures your account information is accurate. This guide covers everything you need to know about managing your user profile.

Accessing Your Profile

To access your user profile settings:

  1. Log in to your EquiBillBook account
  2. Click on your profile icon or name (usually in the top-right corner)
  3. Select "Profile" or "User Settings" from the dropdown menu
  4. Alternatively, navigate to Settings → User Settings → Profile

Profile Information You Can Update

1. Personal Information

  • Full Name: Your first and last name
  • Display Name: Name shown to other users (if applicable)
  • Email Address: Your primary email (requires verification if changed)
  • Phone Number: Contact number
  • Date of Birth: Optional date of birth

2. Profile Photo

Upload a profile picture:

  • Click on the profile photo area
  • Select an image file from your device
  • Recommended size: 200x200 pixels or larger
  • Supported formats: JPG, PNG, GIF
  • Crop and adjust if needed
  • Click "Save" to update

3. Contact Information

  • Primary Phone: Main contact number
  • Alternate Phone: Secondary contact number (optional)
  • Address: Physical address (optional)
  • City, State, Country: Location information

4. Preferences

  • Language: Preferred language (if multiple languages supported)
  • Time Zone: Your local time zone
  • Date Format: Preferred date display format
  • Currency: Default currency (if applicable)

Step-by-Step Profile Setup

Step 1: Open Profile Settings

Navigate to your profile settings page using the steps mentioned above.

Step 2: Update Basic Information

Fill in or update your personal details:

  • Enter your full name
  • Update email address if needed (new email will require verification)
  • Add or update phone number
  • Fill in any other required fields

Step 3: Upload Profile Photo (Optional)

Click on the profile picture area and upload an image. This helps with user identification, especially in multi-user accounts.

Step 4: Set Preferences

Configure your preferences:

  • Select your time zone
  • Choose date and time format
  • Set language preference

Step 5: Save Changes

Click the "Save" or "Update Profile" button to save your changes. You'll see a confirmation message once saved.

Changing Your Email Address

If you need to change your email address:

  1. Go to Profile Settings
  2. Update the email field with your new email address
  3. Save the changes
  4. Check your new email for a verification link
  5. Click the verification link to confirm the new email
  6. Your email will be updated after verification

Important: Until you verify the new email, you'll continue to receive notifications at your old email address.

Security Settings

In your profile, you may also access security settings:

  • Change Password: Update your account password
  • Two-Factor Authentication: Enable 2FA for additional security (if available)
  • Login History: View recent login activities
  • Active Sessions: Manage logged-in sessions on different devices

Notification Preferences

You may be able to configure notification preferences:

  • Email Notifications: Choose which emails you want to receive
  • SMS Notifications: Enable/disable SMS alerts
  • In-App Notifications: Configure dashboard notifications
  • Notification Frequency: Set how often you receive updates

Profile Privacy Settings

If working in a multi-user environment, you may have privacy options:

  • Control what information is visible to other users
  • Set profile visibility preferences
  • Manage contact information sharing

User Roles and Permissions

Your profile may display:

  • User Role: Your assigned role (Admin, Manager, User, etc.)
  • Permissions: Features and modules you can access
  • Branch Access: Which branches/locations you can access (if multi-branch)

Note: Roles and permissions are typically managed by administrators and may not be editable from your profile.

Best Practices for Profile Setup

  • Use Real Information: Provide accurate details for better account management
  • Keep Email Updated: Ensure your email is current to receive important notifications
  • Upload Professional Photo: Use a clear, professional profile picture
  • Set Correct Time Zone: This ensures dates and times display correctly
  • Review Regularly: Update your profile when information changes

Profile Completion

Some systems show a profile completion percentage. To complete your profile:

  • Fill in all required fields
  • Add optional information for a complete profile
  • Upload a profile photo
  • Verify your email address

Troubleshooting Profile Issues

Cannot Save Changes

  • Check that all required fields are filled
  • Verify email format is correct
  • Ensure phone number is in correct format
  • Try refreshing the page and updating again

Profile Photo Not Uploading

  • Check file size (may have size limits)
  • Ensure file format is supported (JPG, PNG, GIF)
  • Try a smaller image file
  • Check your internet connection

Email Verification Not Received

  • Check spam/junk folder
  • Wait a few minutes and check again
  • Request a new verification email
  • Verify the email address is correct

Deleting Your Profile Photo

To remove your profile photo:

  • Go to profile settings
  • Find the profile photo section
  • Click "Remove" or "Delete" button
  • Confirm the removal
  • A default avatar will be shown

Exporting Profile Data

You may be able to:

  • Download your profile information
  • Export account data (if available)
  • Request a data copy for your records

Need Help?

If you need assistance with profile setup:

  • Check field tooltips for guidance
  • Review this documentation
  • Contact support if you encounter issues

A complete and up-to-date profile helps you get the most out of EquiBillBook and ensures smooth communication and account management.
