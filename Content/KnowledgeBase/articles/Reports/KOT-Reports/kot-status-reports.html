KOT Status Reports

KOT Status Reports provide detailed analysis of Kitchen Order Tickets (KOTs) based on their status in EquiBillBook. These reports help you understand order workflow, track status transitions, identify bottlenecks, monitor completion rates, and optimize the order fulfillment process.

What are KOT Status Reports?

KOT Status Reports analyze KOTs by their current status and status transitions, providing:

  • Status distribution
  • Status transition patterns
  • Time in each status
  • Pending orders analysis
  • Completion rates
  • Workflow efficiency

Accessing KOT Status Reports

To generate KOT Status Reports:

  1. Navigate to Reports → KOT Reports → KOT Status Reports
  2. Select the date range for analysis
  3. Configure filters (status, station, table, etc.)
  4. Click "Generate Report"

Report Parameters

Configure the following parameters:

  • Date Range: Select period (Today, This Week, This Month, This Year, Custom Range)
  • Business Location: Filter by branch/location (if multi-branch)
  • Status: Filter by specific KOT status (Pending, In Progress, Completed, Cancelled, etc.)
  • Station: Filter by kitchen station (optional)
  • Table: Filter by table (optional)
  • Floor: Filter by restaurant floor (optional)

KOT Status Types

Common KOT statuses in EquiBillBook:

  • Pending: Order received but not yet started
  • In Progress: Order is being prepared
  • Ready: Order is ready for service
  • Served: Order has been served to customer
  • Completed: Order is fully completed
  • Cancelled: Order was cancelled
  • On Hold: Order is temporarily on hold

Report Contents

KOT Status Reports typically include:

  • Status Distribution: Count and percentage of KOTs in each status
  • Status Breakdown: Detailed breakdown by status
  • Pending Orders: List of orders in pending status
  • In Progress Orders: Orders currently being prepared
  • Completed Orders: Successfully completed orders
  • Cancelled Orders: Cancelled orders with reasons
  • Status Transition Times: Time spent in each status
  • Completion Rate: Percentage of completed orders
  • Status Trends: Status patterns over time
  • Workflow Efficiency: Efficiency metrics based on status flow

Understanding the Report

Key insights to analyze:

  • Workflow Health: Overall status distribution indicates workflow health
  • Bottleneck Identification: High counts in specific statuses indicate bottlenecks
  • Completion Efficiency: Completion rate shows overall efficiency
  • Status Flow: Status transitions show workflow patterns
  • Pending Orders: High pending counts may indicate capacity issues
  • Cancellation Patterns: Cancellation rates and reasons
  • Time Analysis: Time in each status shows efficiency
