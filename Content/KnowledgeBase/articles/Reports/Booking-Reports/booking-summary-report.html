Booking Summary Report

Booking Summary Report provides a comprehensive overview of all table bookings for your restaurant, helping you analyze booking patterns, track performance, and make informed operational decisions.

What is Booking Summary Report?

Booking Summary Report provides:

  • Total number of bookings for the selected period
  • Bookings breakdown by status
  • Revenue from bookings
  • Average party size
  • Booking trends over time
  • Performance metrics and statistics

Why Booking Summary Report is Important

This report helps you:

  • Track overall booking performance
  • Monitor booking trends and patterns
  • Analyze revenue from bookings
  • Identify peak booking periods
  • Make capacity planning decisions
  • Evaluate booking conversion rates
  • Plan staffing and resources
  • Compare performance across periods

Accessing Booking Summary Report

To view Booking Summary Report:

  1. Navigate to Reports â†’ Booking Reports
  2. Select Booking Summary or Booking Overview
  3. Set the date range for analysis
  4. Apply filters if needed (table, customer, status)
  5. Generate the report

Booking Summary Report Information

The report typically displays:

  • Total Bookings: Total number of bookings
  • Confirmed Bookings: Number of confirmed reservations
  • Pending Bookings: Number of pending reservations
  • Cancelled Bookings: Number of cancelled reservations
  • Completed Bookings: Number of completed bookings
  • No-Show Bookings: Number of no-show reservations
  • Total Revenue: Revenue generated from bookings
  • Average Party Size: Average number of guests per booking
  • Total Guests: Total number of guests across all bookings
  • Booking Conversion Rate: Percentage of confirmed bookings
  • Cancellation Rate: Percentage of cancelled bookings
  • No-Show Rate: Percentage of no-show bookings

Filtering Booking Summary Report

You can filter reports by:

  • Date Range: Daily, weekly, monthly, or custom periods
  • Table: Filter by specific tables or table types
  • Customer: Filter by specific customers
  • Booking Status: Confirmed, pending, cancelled, completed, no-show
  • Location/Floor: Filter by restaurant floor or location
  • Waiter: Filter by assigned waiter
  • Booking Source: Online, phone, walk-in, etc.

Bookings by Status

The report breaks down bookings by status:

  • Confirmed: Bookings that are confirmed
  • Pending: Bookings awaiting confirmation
  • Cancelled: Bookings that were cancelled
  • Completed: Bookings that were fulfilled
  • No-Show: Bookings where customers didn't arrive
  • Analyze status distribution and trends
