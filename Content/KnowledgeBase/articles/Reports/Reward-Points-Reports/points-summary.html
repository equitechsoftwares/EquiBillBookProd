Points Summary

Points Summary reports in EquiBillBook provide a comprehensive overview of reward points earned, redeemed, and available across your customer base. These reports help track points program performance, monitor customer engagement, and manage reward points effectively.

Understanding Points Summary Reports

Points Summary Reports help you:

  • Track overall points program performance
  • Monitor points earned and redeemed
  • View available points balance
  • Analyze points program effectiveness
  • Manage points liability
  • Make informed program decisions

Accessing Points Summary Reports

To generate points summary reports:

  1. Navigate to the Reports module from the main menu
  2. Click on Reward Points Reports â†’ Points Summary or Summary Report
  3. Select date range and filters
  4. Click "Generate Report"

Summary Metrics

Points Overview

Overall points statistics include:

  • Total Points Earned: Total points earned by all customers
  • Total Points Redeemed: Total points redeemed
  • Total Points Available: Current points balance
  • Total Points Expired: Points that expired
  • Points Liability: Outstanding points value
  • Active Points Accounts: Customers with points

Earning Summary

Points earning statistics:

  • Points Earned (Period): Points earned in selected period
  • Earning Transactions: Number of earning transactions
  • Average Points per Transaction: Average earning rate
  • Top Earning Customers: Highest point earners
  • Earning by Source: Breakdown by earning source
  • Earning Trends: Earning patterns over time

Redemption Summary

Points redemption statistics:

  • Points Redeemed (Period): Points redeemed in period
  • Redemption Transactions: Number of redemptions
  • Average Redemption Value: Average redemption amount
  • Top Redeeming Customers: Highest redeemers
  • Redemption by Type: Breakdown by redemption type
  • Redemption Trends: Redemption patterns over time

Balance Summary

Current points balance statistics:

  • Total Outstanding Points: Total available points
  • Average Points per Customer: Average balance
  • Points Expiring Soon: Points near expiration
  • High Balance Accounts: Customers with high balances
  • Inactive Points: Points in inactive accounts

Report Parameters

Date Range Selection

Select the period for summary:

  • Today: Today's points activity
  • This Week: Current week
  • This Month: Current month
  • This Quarter: Current quarter
  • This Year: Current year
  • Custom Range: Select specific dates
  • All Time: Complete history

Filter Options

Apply filters to summary:

  • Customer: Filter by specific customer(s)
  • Customer Group: Filter by customer category
  • Location/Branch: Filter by location
  • Points Status: Active, expired, redeemed
  • Earning Source: Filter by earning source
  • Redemption Type: Filter by redemption type

Summary Type Selection

Select summary type:

  • Overall Summary: Complete overview
  • Earning Summary: Earning-focused
  • Redemption Summary: Redemption-focused
  • Balance Summary: Balance-focused
  • Comprehensive Summary: All aspects

    Troubleshooting

    Report shows no data:

    • Check date range is correct
    • Verify reward points are enabled
    • Ensure points transactions exist
    • Check filters are not too restrictive

    Totals don't match expected:

    • Verify all points transactions are recorded
    • Check date range selected
    • Review filters applied
    • Ensure calculations are correct

    Charts not displaying:

    • Ensure sufficient data exists
    • Try refreshing the report
    • Check browser compatibility
    • Verify JavaScript is enabled
