Sharing Payment Links

Once you've generated a payment link in EquiBillBook, you need to share it with your customers so they can make payments. This guide covers various methods to share payment links effectively.

Accessing the Payment Link

To share a payment link:

  1. Navigate to Sales → Payment Link
  2. Find the payment link you want to share from the list
  3. Click on the payment link to view details, or
  4. Use the action buttons to copy or share the link

Copying the Payment Link

The most common way to share payment links:

  1. Open the payment link details page
  2. Locate the payment link URL (usually displayed prominently)
  3. Click the "Copy Link" button or manually copy the URL
  4. The link is now copied to your clipboard
  5. Paste it into your preferred communication channel

Sharing via Email

To share payment links via email:

  1. Copy the payment link URL
  2. Open your email client or use EquiBillBook's email feature
  3. Compose a new email to your customer
  4. Include the payment link in the email body
  5. Add a clear subject line (e.g., "Payment Request - Invoice #12345")
  6. Include payment details: amount, due date, and reference number
  7. Send the email

Email Template Tips

When sharing payment links via email:

  • Use a professional and friendly tone
  • Clearly state the payment amount and purpose
  • Include the payment due date or expiration date
  • Mention the reference number for easy tracking
  • Provide instructions on how to use the payment link
  • Include your contact information for support

Sharing via SMS

For quick payment collection, share links via SMS:

  1. Copy the payment link URL
  2. Open your SMS application or use EquiBillBook's SMS feature (if available)
  3. Compose a concise message with the payment link
  4. Include essential details: amount and reference number
  5. Send the SMS to the customer's mobile number

SMS Message Template

A good SMS template includes:

  • Greeting with customer name
  • Payment amount
  • Payment link URL
  • Reference number
  • Brief instructions

Example: "Hi [Customer Name], Please pay ₹[Amount] using this link: [Payment Link]. Ref: [Reference Number]. Thank you!"

Sharing via WhatsApp

If you use WhatsApp for business communication:

  1. Copy the payment link URL
  2. Open WhatsApp and select the customer's contact
  3. Send a message with the payment link
  4. Include payment details and instructions
  5. You can also use EquiBillBook's WhatsApp integration (if configured)

Sharing via Other Channels

You can share payment links through:

  • Messaging Apps: Telegram, Signal, or other messaging platforms
  • Social Media: Direct messages on social media platforms
  • Printed Documents: Include QR codes or links in printed invoices or statements
  • Customer Portal: If customers have access to a portal, display links there
  • In-Person: Share the link verbally or via QR code during meetings

Using QR Codes

For in-person or printed sharing:

  1. Generate a QR code for the payment link (if available in EquiBillBook)
  2. Print the QR code on invoices, receipts, or statements
  3. Customers can scan the QR code with their mobile phones
  4. The payment link will open automatically in their browser

Best Practices for Sharing Payment Links

  • Share promptly: Send payment links as soon as they're generated
  • Include context: Always mention what the payment is for
  • Set expectations: Inform customers about the expiration date
  • Follow up: Send reminders if payment is not received
  • Verify contact details: Ensure you have the correct email or phone number
  • Use secure channels: Prefer secure communication methods
  • Personalize messages: Address customers by name when possible

Payment Link Security

When sharing payment links:

  • Links are secure and use unique reference IDs
  • Only share links with the intended customer
  • Don't post payment links publicly on social media
  • Links expire after the set expiration date
  • Each link can typically be used only once (depending on settings)

Tracking Shared Links

You can monitor shared payment links:

  • View link status in the Payment Link list
  • See when the link was accessed (if tracking is enabled)
  • Monitor payment status in real-time
  • Receive notifications when payments are made

Resending Payment Links

If a customer hasn't made payment:

  1. Go to the Payment Link list
  2. Find the payment link
  3. Check if the link is still valid (not expired)
  4. Copy the link again and resend via your preferred channel
  5. Consider sending a reminder message

What Customers See

When customers click the payment link, they will:

  1. See a secure payment page
  2. View payment details: amount, customer name, reference number
  3. See available payment methods
  4. Be able to complete the payment online
  5. Receive a confirmation after successful payment

Troubleshooting

Common issues when sharing payment links:

  • Link not working: Check if the link has expired or if there's a typo in the URL
  • Customer can't access: Verify the link is correct and hasn't expired
  • Link already used: Some links are single-use; generate a new one if needed
  • Email not delivered: Check spam folder or verify email address

Tips for Better Payment Collection

  • Send payment links immediately after generating invoices
  • Use multiple communication channels for important payments
  • Set reasonable expiration dates to create urgency
  • Send friendly reminders before expiration
  • Thank customers after payment is received
  • Keep records of when and how links were shared

Sharing payment links effectively is key to ensuring timely payment collection. Choose the communication method that works best for your customers and business, and always include clear instructions and payment details.
