Setting Up POS

Point of Sale (POS) is a crucial feature for retail businesses that enables quick and efficient sales transactions. This guide will walk you through the complete setup process for POS in EquiBillBook.

What is POS?

POS (Point of Sale) is a system that allows you to process sales transactions at your physical store or retail location. It provides a streamlined interface for adding items, processing payments, and generating receipts quickly, making it ideal for walk-in customers and retail environments.

Prerequisites

Before setting up POS, ensure you have:

  • Active EquiBillBook account with Sales module enabled
  • Items added to your inventory (products you want to sell)
  • Payment accounts configured (Cash, Card, UPI, etc.)
  • Access to POS settings configuration

Step-by-Step POS Setup

Step 1: Access POS Settings

To begin configuring your POS:

  1. Navigate to Settings from the main menu
  2. Go to Sales Settings or POS Settings
  3. Click on "POS Configuration" or "POS Setup"

Step 2: Configure Basic POS Settings

Set up the fundamental POS configurations:

  • Enable POS Module: Activate the POS feature for your organization
  • Default Payment Method: Set your primary payment method (Cash, Card, etc.)
  • Default Customer: Choose a default walk-in customer for POS sales
  • Invoice Prefix: Set a specific prefix for POS invoices (e.g., "POS-")
  • Auto-print Receipt: Enable automatic receipt printing after each sale

Step 3: Configure POS Screen Settings

Customize the POS interface according to your needs:

  • Disable Draft Mode: Turn off draft functionality for faster checkout
  • Disable Quotation/Proforma: Hide options you don't need in POS
  • Disable Credit Sale: Restrict credit sales if needed
  • Disable Hold: Remove hold transaction option
  • Disable Multiple Payment: Restrict to single payment method
  • Enable Express Checkout: Speed up transactions with quick checkout

Step 4: Configure Product Display

Customize how products appear on POS:

  • Product Suggestion: Show/hide product suggestions while typing
  • Recent Transactions: Display recent sales for quick reference
  • Show Pricing on Tooltip: Display prices when hovering over products
  • Product SKU Length: Configure barcode/SKU scanning length

Step 5: Configure Quantity Settings

Set up quantity handling:

  • Quantity Integer Part: Maximum digits before decimal point
  • Quantity Fractional Part: Maximum decimal places
  • Enable Weighing Scale: Integrate with weighing scale if available
  • Weighing Scale Barcode Prefix: Set prefix for scale-generated barcodes

Step 6: Configure Discount and Tax Settings

Set up pricing adjustments:

  • Disable Discount: Turn off discount functionality if needed
  • Disable Order Tax: Hide tax options in POS
  • Subtotal Editable: Allow manual subtotal editing

Step 7: Configure Staff and Date Settings

Set up staff and transaction management:

  • Enable Service Staff: Assign staff to transactions
  • Service Staff Required: Make staff assignment mandatory
  • Enable Transaction Date: Allow custom transaction dates

Step 8: Configure Invoice Settings

Customize invoice/receipt options:

  • Show Invoice Scheme: Display invoice schemes in POS
  • Show Invoice Layout Dropdown: Allow layout selection
  • Print Invoice on Hold: Print receipt even for held transactions

Step 9: Save Settings

After configuring all settings:

  1. Review all your configurations
  2. Click "Save" or "Save & Apply"
  3. Settings will be immediately applied to POS screen

POS Setup Features

  • Quick Configuration: Easy-to-use settings interface
  • Flexible Options: Enable/disable features as per your business needs
  • Multi-branch Support: Different POS settings for different locations
  • Real-time Application: Changes apply immediately without restart
  • Default Values: Sensible defaults for quick setup

Best Practices

  • Test POS settings in a test environment before production use
  • Configure payment methods according to what you accept
  • Enable express checkout for faster transactions during peak hours
  • Disable features you don't use to reduce clutter and improve speed
  • Set up proper invoice numbering to distinguish POS sales from regular invoices
  • Configure barcode scanning if you use barcode scanners
  • Train staff on POS operations before going live
