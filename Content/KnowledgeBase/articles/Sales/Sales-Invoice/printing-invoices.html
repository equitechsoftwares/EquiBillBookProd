Printing Invoices

Printing invoices is essential for providing physical copies to customers and maintaining records. EquiBillBook offers flexible printing options with customizable templates and print settings.

How to Print an Invoice

To print an invoice:

  1. Navigate to the invoice you want to print
  2. Open the invoice details/view page
  3. Click the Print button (usually at the top or bottom of the invoice)
  4. Review the print preview (if available)
  5. Configure print settings if needed
  6. Click Print in the print dialog
  7. Select your printer and click Print

Print Options

You may have different print options:

  • Print Invoice: Standard invoice print
  • Print Duplicate: Print duplicate copy
  • Print Receipt: Print payment receipt format
  • Print Packing Slip: Print delivery/packing slip
  • Print Thermal: Print on thermal paper (for POS)

Print Preview

Before printing, you can:

  • Preview how the invoice will look
  • Check layout and formatting
  • Verify all details are included
  • Review page breaks
  • Make adjustments if needed

Invoice Templates

Choose from available templates:

  • Standard Template: Default professional template
  • Custom Templates: Your customized templates
  • Simplified Template: Minimal design
  • Detailed Template: Comprehensive format

Select template before printing if multiple options are available.

Print Settings

Configure print settings:

  • Paper Size: A4, Letter, Thermal, etc.
  • Orientation: Portrait or Landscape
  • Margins: Adjust page margins
  • Scale: Fit to page or actual size
  • Headers/Footers: Include or exclude

What Gets Printed

The printed invoice typically includes:

  • Company logo and details
  • Invoice number and date
  • Customer information
  • Billing and shipping addresses
  • Item details (name, quantity, price, amount)
  • Subtotal, discounts, taxes
  • Grand total
  • Payment terms and due date
  • Notes and terms & conditions
  • Payment instructions

Printing from List View

You can also print:

  • Multiple invoices at once (batch printing)
  • Select invoices from list and print
  • Print filtered invoices
  • Export to PDF for printing later

Print Quality Settings

Adjust print quality:

  • Draft: Fast printing, lower quality
  • Normal: Standard quality
  • High Quality: Best quality, slower printing

Printer Selection

Choose the right printer:

  • Default Printer: Uses system default
  • Select Printer: Choose from available printers
  • PDF Printer: Save as PDF instead of printing
  • Network Printer: Print to shared printers

Thermal Printing (POS)

For POS or thermal printers:

  • Use thermal print template
  • Configure for receipt paper size
  • Optimized for 58mm or 80mm paper
  • May include barcode or QR code

Printing Logo and Branding

Ensure professional appearance:

  • Company logo is printed (if configured)
  • Branding colors are used (if supported)
  • Company details are accurate
  • Professional layout is maintained

Print Troubleshooting

Common printing issues:

  • Print button not working: Check browser settings and popup blockers
  • Layout issues: Adjust print settings or template
  • Missing elements: Verify template configuration
  • Cut-off content: Adjust margins or scale settings
  • Printer not found: Check printer connection and selection
  • Poor quality: Adjust print quality settings

Print as PDF

Instead of physical printing:

  • Save invoice as PDF
  • Email PDF to customers
  • Store PDF for records
  • Print PDF later if needed
  • Use "Save as PDF" or PDF printer option

Batch Printing

Print multiple invoices:

  1. Select invoices from list (checkboxes)
  2. Click batch print or print selected
  3. Configure print settings
  4. Print all selected invoices

Note: Each invoice may print on separate pages.

Print History

Some systems track:

  • When invoices were printed
  • Who printed the invoice
  • Print count (original, duplicate, etc.)

Best Practices

  • Always preview before printing
  • Use appropriate paper size and quality
  • Keep printer settings consistent
  • Print on company letterhead when possible
  • Verify all details before printing
  • Store printed copies securely
  • Consider PDF for electronic records

Legal and Compliance

Printed invoices should:

  • Meet legal requirements for your region
  • Include all required information
  • Be clearly readable
  • Be stored for required retention period
  • Comply with tax regulations
