Multiple Challan Management

Managing multiple delivery challans in EquiBillBook allows you to handle complex delivery scenarios, track multiple shipments, and efficiently organize deliveries for different customers, orders, or time periods.

Understanding Multiple Challan Scenarios

You may need to manage multiple challans in various situations:

  • Multiple deliveries for the same customer
  • Partial deliveries from a single order
  • Multiple challans for different customers on the same day
  • Bulk deliveries across different locations
  • Recurring deliveries for subscription customers

Creating Multiple Challans

From a Single Sales Order

When an order needs to be delivered in multiple shipments:

  1. Navigate to the Sales Order details
  2. Click "Create Delivery Challan"
  3. Modify quantities to reflect the first shipment
  4. Save the first challan
  5. Create another challan from the same order for remaining items
  6. Repeat until all items are covered

Bulk Challan Creation

For creating multiple challans efficiently:

  • Use the challan list view to see all pending orders
  • Select multiple orders that need challans
  • Create challans in batch if the feature is available
  • Or create them individually but systematically

Managing Multiple Challans

Challan List View

The challan list provides comprehensive management:

  • Filter Options: Filter by customer, date range, status, branch
  • Search: Quickly find specific challans by number, customer, or reference
  • Sort: Sort by date, customer, status, or amount
  • Bulk Actions: Select multiple challans for batch operations

Status Management

Track multiple challans by status:

  • Pending: Challans ready for dispatch
  • In Transit: Challans currently being delivered
  • Delivered: Successfully completed deliveries
  • Partially Delivered: Partial shipments completed
  • Cancelled: Cancelled challans

Best Practices for Multiple Challan Management

  • Use consistent naming and numbering for easy identification
  • Update challan statuses promptly as deliveries progress
  • Group related challans using tags or categories
  • Maintain clear notes on each challan for reference
  • Regularly review pending challans to ensure timely delivery
  • Use filters and search to quickly find specific challans
  • Keep delivery schedules organized and up-to-date
  • Monitor delivery performance and identify bottlenecks
  • Convert delivered challans to invoices promptly
  • Archive completed challans for record-keeping

Workflow Automation

For managing multiple challans efficiently:

  • Set up automated status updates based on delivery milestones
  • Configure notifications for pending deliveries
  • Use templates for similar delivery scenarios
  • Implement approval workflows for bulk operations

Troubleshooting

Can't find specific challans in the list?

  • Use search and filter options
  • Check date range filters
  • Verify status filters are set correctly
  • Clear filters to see all challans

Bulk operations are not working?

  • Ensure you have permissions for bulk actions
  • Check if challans are in the correct status
  • Verify all selected challans meet operation criteria
  • Try operations on smaller batches

Partial deliveries are confusing?

  • Use clear reference numbers linking challans to orders
  • Add notes explaining partial delivery reasons
  • Track fulfillment percentage in order details
  • Generate reports to see complete delivery picture
