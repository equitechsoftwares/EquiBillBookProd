Station Type Setup

Station Types in EquiBillBook help categorize and organize your kitchen stations based on their function or location. This guide will walk you through setting up station types to better manage your kitchen operations.

What are Station Types?

Station Types are classifications that group kitchen stations by their purpose, location, or function. Examples include "Main Kitchen", "Cold Station", "Hot Station", "Beverage Station", "Grill Station", "Salad Station", etc. Using station types helps organize your kitchen layout and makes it easier to manage multiple stations.

Accessing Station Types

To set up station types:

  1. Navigate to Settings from the main menu
  2. Go to Restaurant Settings â†’ Station Types
  3. Click the "Add Station Type" or "Create New" button

Creating a Station Type

When creating a new station type:

  • Station Type Name: Enter a descriptive name (e.g., "Hot Station", "Cold Station", "Beverage Station", "Grill Station")
  • Description: (Optional) Add a brief description explaining the purpose of this station type
  • Active Status: Enable or disable the station type. Inactive types won't be available when creating new stations

    • Quick Add from Station Form

    You can also create station types directly from the Kitchen Station form:

    1. When creating or editing a kitchen station, click the "+" button next to the Station Type dropdown
    2. A modal window will open for quick station type creation
    3. Enter the station type name
    4. Click "Save" to create and automatically select it for the current station

    Common Station Types

    Here are some common station types you might want to create:

    • Main Kitchen: Primary cooking station
    • Hot Station: For hot food preparation
    • Cold Station: For salads, cold appetizers, and cold dishes
    • Beverage Station: For drinks, juices, and beverages
    • Grill Station: For grilled items
    • Dessert Station: For desserts and sweets
    • Pizza Station: For pizza preparation
    • Bar Station: For bar drinks and cocktails

    Managing Station Types

    After creating station types, you can:

    • Edit: Modify station type names and descriptions
    • Activate/Deactivate: Enable or disable station types without deleting them
    • View Usage: See which stations are using each station type
    • Delete: Remove station types that are no longer needed (only if not in use)

    Best Practices

    • Create station types that match your kitchen's physical layout
    • Use consistent naming conventions across all station types
    • Keep station type names short and descriptive
    • Don't create too many station types - keep it simple and practical
    • Ensure station types reflect your actual kitchen operations
    • Review and update station types periodically to match your evolving kitchen needs

    Assigning Station Types to Stations

    Once station types are created:

    1. When creating or editing a kitchen station, select the appropriate station type from the dropdown
    2. Each station can have only one station type
    3. You can filter and group stations by station type for better organization
    4. Station types help organize your station list and reports
