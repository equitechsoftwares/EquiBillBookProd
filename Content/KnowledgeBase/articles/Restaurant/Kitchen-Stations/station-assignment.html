Station Assignment

Station Assignment in EquiBillBook determines which kitchen station handles which food items when KOTs (Kitchen Order Tickets) are created. This guide explains how to assign item categories to stations for efficient order routing and kitchen workflow.

What is Station Assignment?

Station Assignment is the process of linking item categories to specific kitchen stations. When a KOT is created with items from different categories, each item is automatically routed to the appropriate station based on its category assignment. This ensures orders are prepared at the correct location in your kitchen.

How Station Assignment Works

The assignment process works as follows:

  1. Items in EquiBillBook are organized into categories (e.g., Appetizers, Main Course, Desserts, Beverages)
  2. Each kitchen station can be assigned one or more categories
  3. When a KOT is created, items are automatically routed to stations based on their category
  4. If an item's category is assigned to multiple stations, the item may appear at all assigned stations

Assigning Categories to Stations

To assign categories to a kitchen station:

  1. Navigate to Settings â†’ Kitchen Stations
  2. Create a new station or edit an existing one
  3. In the Categories field, select one or more categories that this station will handle
  4. You can select multiple categories by holding Ctrl (Windows) or Cmd (Mac) while clicking
  5. Save the station to apply the category assignments

Category Assignment Examples

Here are some common assignment scenarios:

  • Main Kitchen Station: Assign "Main Course", "Rice", "Curries", "Breads"
  • Salad Station: Assign "Salads", "Cold Appetizers", "Soups"
  • Dessert Station: Assign "Desserts", "Ice Cream", "Sweets"
  • Beverage Station: Assign "Beverages", "Juices", "Soft Drinks", "Coffee"
  • Grill Station: Assign "Grilled Items", "BBQ", "Tandoor"

Multiple Station Assignment

You can assign the same category to multiple stations if needed:

  • This is useful when multiple stations can prepare the same type of items
  • Items from that category will appear at all assigned stations
  • Use this when you have backup stations or when items can be prepared at different locations
  • Example: Assign "Beverages" to both "Bar Station" and "Beverage Station"

Ensuring Complete Coverage

Important considerations:

  • Ensure every item category used in your menu is assigned to at least one active station
  • If a category is not assigned to any station, items from that category may not appear in KOTs
  • Review category assignments periodically to ensure they match your current menu
  • Update assignments when you add new categories or change your kitchen layout

    • Viewing Station Assignments

    To view which categories are assigned to each station:

    1. Go to Kitchen Stations list
    2. Click on a station to view or edit it
    3. The assigned categories will be displayed in the Categories field
    4. You can also view station assignments in station reports

    Best Practices

    • Assign categories logically based on your kitchen's physical layout
    • Ensure all menu item categories are assigned to at least one station
    • Review assignments when you add new menu items or categories
    • Keep assignments simple - avoid over-complicating the routing
    • Test assignments by creating sample KOTs to ensure items route correctly
    • Document your assignment strategy for team reference
    • Update assignments when kitchen layout or menu changes

    Troubleshooting Assignment Issues

    If items are not appearing at the expected stations:

    • Verify that the item's category is assigned to the station
    • Check that the station is active
    • Ensure the category is active and assigned to items
    • Review KOT creation to confirm items are being added correctly
    • Check if multiple stations are assigned and items are routing to all of them
