Table QR Code Generation

QR codes for tables enable contactless ordering, easy table identification, and seamless customer experience. This guide explains how to generate and use QR codes for restaurant tables in EquiBillBook.

Understanding Table QR Codes

Table QR codes provide:

  • Quick access to table-specific menu and ordering
  • Contactless ordering experience
  • Easy table identification for staff
  • Direct link to table booking or information
  • Integration with digital menu and ordering systems

Accessing QR Code Generation

To generate QR codes for tables:

  1. Navigate to Restaurant from the main menu
  2. Go to Tables or Restaurant Tables
  3. Select the table(s) you want to generate QR codes for
  4. Click "Generate QR Code" or "QR Codes"

Generating QR Codes

To generate a QR code for a table:

  1. Select the table from the table list
  2. Click "Generate QR Code"
  3. Configure QR code settings (if applicable)
  4. The QR code is generated and displayed
  5. Download or print the QR code

QR Code Configuration

You can configure the following for QR codes:

  • QR Code Size: Set the size for printing
  • QR Code Format: Choose PNG, SVG, or PDF format
  • Include Table Information: Add table number/name on QR code
  • Custom Logo: Add your restaurant logo to QR code
  • QR Code Link: Configure what the QR code links to

QR Code Links

QR codes can link to:

  • Digital Menu: Table-specific menu for ordering
  • Ordering System: Direct ordering interface for the table
  • Table Information: Table details and booking information
  • Feedback Form: Customer feedback and rating
  • Payment Portal: Direct payment link

Bulk QR Code Generation

To generate QR codes for multiple tables:

  1. Select multiple tables from the table list
  2. Click "Bulk Generate QR Codes"
  3. Configure settings for all QR codes
  4. Generate all QR codes at once
  5. Download as a batch or individual files

Printing QR Codes

To print QR codes:

  1. Generate the QR code for a table
  2. Click "Print" or "Download"
  3. Choose print settings (size, format, quantity)
  4. Print on labels, cards, or table stands
  5. Place QR codes on tables

QR Code Placement

Best practices for placing QR codes:

  • Place on table stands for easy visibility
  • Use waterproof labels for outdoor tables
  • Ensure QR codes are easily scannable
  • Include instructions for customers
  • Replace damaged or faded QR codes

Using QR Codes

When customers scan the QR code:

  1. QR code opens the configured link
  2. Customer sees table-specific menu or ordering interface
  3. Customer can place orders directly
  4. Orders are linked to the table
  5. Staff receives orders through KOT system

Best Practices

  • Generate QR codes for all customer-facing tables
  • Test QR codes before placing on tables
  • Keep QR codes clean and scannable
  • Update QR codes if table information changes
  • Provide instructions for customers on how to use QR codes
  • Have backup ordering methods for customers without smartphones

QR Code Management

You can manage QR codes by:

  • Regenerating QR codes if links change
  • Updating QR codes when table information changes
  • Tracking which tables have QR codes
  • Monitoring QR code usage and scans
  • Replacing damaged or lost QR codes

  • Print and place QR codes on tables
  • Test QR code functionality
  • Train staff on QR code usage
