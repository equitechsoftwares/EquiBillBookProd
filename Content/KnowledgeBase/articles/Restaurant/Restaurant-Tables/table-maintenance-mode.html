Table Maintenance Mode

Table maintenance mode allows you to mark tables as unavailable when they need repairs, cleaning, or are temporarily out of service. This guide explains how to enable and manage table maintenance mode in EquiBillBook.

Understanding Maintenance Mode

Maintenance mode is used when:

  • Tables need repairs or refurbishment
  • Tables are being deep cleaned
  • Tables are temporarily out of service
  • Tables are being replaced or relocated
  • Special maintenance is scheduled

Accessing Maintenance Mode

To enable maintenance mode for a table:

  1. Navigate to Restaurant from the main menu
  2. Go to Tables or Restaurant Tables
  3. Select the table you want to put in maintenance
  4. Click "Set Status" or "Maintenance Mode"

Enabling Maintenance Mode

To enable maintenance mode:

  1. Select the table(s) that need maintenance
  2. Click "Enable Maintenance" or change status to "Maintenance"
  3. Enter maintenance reason (optional but recommended)
  4. Set maintenance start date/time
  5. Set expected completion date/time (if known)
  6. Add maintenance notes
  7. Save the maintenance mode setting

Maintenance Information

When enabling maintenance mode, you can specify:

  • Maintenance Reason: Why the table is in maintenance
  • Start Date/Time: When maintenance begins
  • Expected End Date/Time: When maintenance is expected to complete
  • Maintenance Notes: Additional details about the maintenance
  • Assigned Staff: Who is handling the maintenance (if applicable)

Maintenance Status

When a table is in maintenance mode:

  • Table status changes to "Maintenance"
  • Table is removed from available booking options
  • Table appears in gray or maintenance color on floor plan
  • Existing bookings may need to be rescheduled
  • Table cannot be booked for new reservations

Scheduled Maintenance

You can schedule maintenance in advance:

  1. Select the table
  2. Enable scheduled maintenance
  3. Set maintenance start and end dates/times
  4. The table will automatically enter maintenance mode at the scheduled time
  5. Table will return to available status after maintenance period

Handling Existing Bookings

When putting a table in maintenance:

  • System may notify customers with existing bookings
  • You may need to reschedule affected bookings
  • Consider alternative tables for customers
  • Communicate maintenance schedule to customers
  • Update booking statuses as needed

Disabling Maintenance Mode

To disable maintenance mode and make table available:

  1. Go to the table in maintenance
  2. Click "End Maintenance" or change status to "Available"
  3. Confirm maintenance is complete
  4. Add completion notes (optional)
  5. Table returns to available status

Maintenance History

The system tracks maintenance history:

  • Maintenance start and end dates
  • Maintenance reasons and notes
  • Duration of maintenance periods
  • Frequency of maintenance for each table

Maintenance Reports

You can generate reports to analyze:

  • Tables currently in maintenance
  • Maintenance history and frequency
  • Maintenance duration and costs
  • Scheduled maintenance calendar

Best Practices

  • Enable maintenance mode immediately when issues are identified
  • Provide clear maintenance reasons for tracking
  • Set realistic maintenance completion dates
  • Notify customers with affected bookings
  • Schedule maintenance during low-traffic periods when possible
  • Keep maintenance notes for future reference
  • Disable maintenance mode promptly when complete

Next Steps

After setting up maintenance mode, you can:

  • Schedule regular maintenance for tables
  • Track maintenance history and costs
  • Generate maintenance reports
  • Optimize maintenance scheduling
  • Monitor table availability and maintenance patterns
