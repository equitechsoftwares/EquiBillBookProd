Table Slugs for Public Booking

Table slugs create unique, shareable URLs for each table, enabling public booking and easy table access. This guide explains how to configure table slugs for public booking in EquiBillBook.

Understanding Table Slugs

Table slugs are:

  • Unique URL identifiers for each table
  • Shareable links for direct table booking
  • Public-facing booking URLs
  • Easy-to-remember table identifiers
  • SEO-friendly table links

What are Table Slugs Used For?

Table slugs enable:

  • Direct table booking through unique URLs
  • Sharing table booking links with customers
  • Embedding table booking on websites
  • Social media sharing of table links
  • Email marketing with specific table links

Accessing Table Slug Configuration

To configure table slugs:

  1. Navigate to Restaurant from the main menu
  2. Go to Tables or Restaurant Tables
  3. Select the table you want to configure
  4. Click "Edit" and go to "Public Booking" or "Slug Settings"

Enabling Public Booking

To enable public booking for a table:

  1. Select the table
  2. Enable "Public Booking" option
  3. Configure booking settings
  4. Generate or set the table slug
  5. Save the configuration

Creating Table Slugs

To create a table slug:

  1. Enable public booking for the table
  2. Enter a unique slug (e.g., "table-1", "window-table-5", "vip-table-3")
  3. Or use the auto-generate option to create a slug automatically
  4. Ensure the slug is unique and descriptive
  5. Save the slug

Slug Format

Table slugs should be:

  • Unique: Each table must have a unique slug
  • URL-friendly: Use lowercase letters, numbers, and hyphens
  • Descriptive: Easy to understand and remember
  • Short: Keep slugs concise for easy sharing

Examples of good slugs:

  • table-1
  • window-table-5
  • vip-table-3
  • outdoor-table-2

Table Booking URL

Once a slug is created, the table booking URL will be:

https://yourdomain.com/booking/table-slug

Or:

https://yourdomain.com/restaurant/booking/table-slug

Sharing Table Links

You can share table booking links through:

  • Email: Send booking links to customers
  • SMS: Text booking links to customers
  • Website: Embed links on your restaurant website
  • Social Media: Share links on social platforms
  • QR Codes: Generate QR codes with table booking links

    • Booking Requirements: Set minimum party size, deposit, etc.

    Managing Table Slugs

    You can manage table slugs by:

    • Editing slugs if table information changes
    • Regenerating slugs if needed
    • Disabling public booking for specific tables
    • Viewing all table slugs in a list
    • Tracking booking activity per table slug

    Best Practices

    • Use descriptive, memorable slugs
    • Keep slugs consistent with table naming
    • Test booking links before sharing
    • Update slugs if table information changes
    • Monitor booking activity through slugs
    • Keep slugs short and URL-friendly
