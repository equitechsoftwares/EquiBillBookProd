Table Capacity Setup

Table capacity determines how many guests can be seated at each table. Proper capacity setup ensures efficient booking management and optimal table utilization. This guide explains how to configure table capacity in EquiBillBook.

Understanding Table Capacity

Table capacity includes:

  • Maximum Capacity: The maximum number of guests a table can accommodate
  • Minimum Party Size: The minimum number of guests required to book the table
  • Optimal Capacity: The recommended number of guests for comfortable seating

Accessing Capacity Settings

To configure table capacity:

  1. Navigate to Restaurant from the main menu
  2. Go to Tables or Restaurant Tables
  3. Select the table you want to configure
  4. Click "Edit" and go to "Capacity Settings"

Setting Maximum Capacity

To set the maximum seating capacity:

  1. Enter the maximum number of guests the table can seat
  2. Consider the physical table size and available space
  3. Account for chairs and comfort requirements
  4. Save the capacity setting

Configuring Minimum Party Size

Setting minimum party size helps:

  • Prevent small parties from booking large tables
  • Optimize table utilization
  • Ensure appropriate table allocation

To configure minimum party size:

  1. Enter the minimum number of guests required
  2. Consider your restaurant's booking policies
  3. Set different minimums for different table types if needed

Capacity by Table Type

Different table types may have different capacity requirements:

  • VIP Tables: Often smaller, intimate settings (2-4 guests)
  • Normal Tables: Standard dining tables (4-8 guests)
  • Outdoor Tables: May vary based on space (4-10 guests)
  • Private Dining: Larger tables for groups (8-20 guests)

Capacity Validation

The system validates capacity during booking:

  • Prevents booking more guests than maximum capacity
  • Warns if party size is below minimum requirement
  • Suggests alternative tables if capacity doesn't match
  • Allows flexibility for special arrangements (with approval)

Best Practices

  • Set accurate capacity based on actual table size
  • Consider comfort and spacing requirements
  • Set appropriate minimum party sizes to optimize utilization
  • Review and update capacity settings regularly
  • Account for special arrangements or high chairs
  • Document any capacity exceptions or special cases

Capacity Reports

You can generate reports to analyze:

  • Table utilization by capacity
  • Average party size per table
  • Capacity efficiency metrics
  • Tables with low utilization

Next Steps

After setting up table capacity, you can:

  • Configure booking rules based on capacity
  • Set up automatic table suggestions based on party size
  • Generate capacity utilization reports
  • Optimize table allocation based on capacity data
