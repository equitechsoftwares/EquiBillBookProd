Table Type Configuration

Table type configuration allows you to customize properties and settings for each table type in EquiBillBook. This guide will help you configure table types to match your restaurant's needs.

Accessing Table Type Configuration

To configure a table type:

  1. Navigate to Restaurant from the main menu
  2. Go to Table Types
  3. Select the table type you want to configure
  4. Click "Edit" or "Configure"

Configuration Options

You can configure the following properties for each table type:

Basic Settings

  • Table Type Name: Update the name of the table type
  • Description: Modify or add a description for the table type
  • Status: Activate or deactivate the table type

Visual Settings

  • Icon Selection: Choose an icon to represent this table type on the floor plan
  • Color Coding: Assign a color to visually distinguish this table type
  • Display Order: Set the order in which table types appear in lists

Capacity Settings

  • Default Capacity: Set the default seating capacity for tables of this type
  • Minimum Capacity: Configure minimum party size (if applicable)
  • Maximum Capacity: Set maximum seating limit

Assignment Settings

  • Floor Assignment: Specify which floors this table type can be used on
  • Auto-assignment Rules: Configure rules for automatically assigning tables to this type

Assigning Tables to Types

To assign existing tables to a table type:

  1. Go to Restaurant Tables
  2. Select the table(s) you want to assign
  3. Choose the appropriate table type from the dropdown
  4. Save the changes

Bulk Configuration

You can configure multiple tables at once:

  • Select multiple tables from the table list
  • Use bulk actions to assign them to a table type
  • Update capacity or other settings for all selected tables

    • Table Type Properties

    Each table type can have specific properties:

    • Booking Restrictions: Set if this table type has special booking requirements
    • Pricing Rules: Configure if this table type has different pricing (if applicable)
    • Availability Rules: Set availability rules specific to this table type

    Best Practices

    • Use consistent naming conventions for table types
    • Assign meaningful icons and colors for easy identification
    • Set appropriate default capacities based on actual table sizes
    • Review and update table type configurations regularly
    • Ensure table types reflect your actual restaurant layout

    Next Steps

    After configuring table types, you can:

    • Assign tables to their appropriate types
    • Configure table type pricing (if applicable)
    • Use table types for filtering bookings and reports
    • Organize your floor plan visualization by type
