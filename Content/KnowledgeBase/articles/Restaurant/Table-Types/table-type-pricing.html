Table Type Pricing (if applicable)

Some restaurants charge different rates for premium table types such as VIP tables, window tables, or private dining areas. This guide explains how to configure pricing for different table types in EquiBillBook.

Understanding Table Type Pricing

Table type pricing allows you to:

  • Charge premium rates for VIP or special tables
  • Apply additional charges for tables with better views or locations
  • Set different pricing for outdoor tables or private dining areas
  • Configure minimum spend requirements for certain table types

Accessing Table Type Pricing

To configure pricing for a table type:

  1. Navigate to Restaurant from the main menu
  2. Go to Table Types
  3. Select the table type you want to configure pricing for
  4. Click on "Pricing" or "Configure Pricing"

Pricing Configuration Options

You can configure the following pricing options:

Fixed Charges

  • Table Charge: Set a fixed charge for booking this table type
  • Per Person Charge: Configure a charge per person for this table type
  • Minimum Spend: Set a minimum spending requirement

Time-Based Pricing

  • Peak Hours Pricing: Set higher charges during peak dining hours
  • Weekend Pricing: Configure different pricing for weekends
  • Special Event Pricing: Set pricing for special events or holidays

Deposit Requirements

  • Deposit Amount: Set a deposit requirement for premium table types
  • Deposit Percentage: Configure deposit as a percentage of estimated bill

Setting Up Table Type Pricing

To set up pricing for a table type:

  1. Select the table type (e.g., VIP, Window, Private Dining)
  2. Enable pricing for this table type
  3. Choose the pricing model (fixed charge, per person, minimum spend, etc.)
  4. Enter the pricing amount or percentage
  5. Configure time-based pricing rules (if applicable)
  6. Set deposit requirements (if applicable)
  7. Save the configuration

Pricing Examples

Here are some common pricing scenarios:

  • VIP Tables: Additional ₹500-1000 charge per booking
  • Window Tables: 10% premium on total bill
  • Private Dining: Minimum spend of ₹5000
  • Outdoor Tables: ₹200 per person during peak hours

Applying Pricing to Bookings

When pricing is configured:

  • Pricing is automatically applied when customers book tables of that type
  • Customers are informed about additional charges during booking
  • Pricing appears on invoices and receipts
  • Deposits are collected as per configuration

Best Practices

  • Clearly communicate pricing to customers before booking
  • Set reasonable pricing that reflects the value of the table type
  • Review pricing regularly and adjust based on demand
  • Ensure pricing is clearly displayed in booking confirmations
  • Train staff on table type pricing to answer customer queries
