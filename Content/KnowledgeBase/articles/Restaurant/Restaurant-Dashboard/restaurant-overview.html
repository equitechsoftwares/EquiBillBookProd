Restaurant Overview

The Restaurant Overview in EquiBillBook's Restaurant Dashboard provides a comprehensive view of your restaurant's current status, key metrics, and quick access to important information. This guide will help you understand and navigate the Restaurant Overview section.

Accessing Restaurant Overview

To access the Restaurant Overview:

  1. Navigate to Restaurant from the main menu
  2. Click on Restaurant Dashboard
  3. Select your branch (if multi-branch enabled)
  4. The Restaurant Overview will be displayed by default

Key Metrics Displayed

The Restaurant Overview shows important metrics at a glance:

Booking Statistics

  • Today's Bookings: Total number of bookings scheduled for today
  • Pending Bookings: Bookings awaiting confirmation
  • Confirmed Bookings: Bookings that have been confirmed
  • Active Bookings: Currently active bookings (confirmed or pending)
  • Completed Bookings: Bookings that have been completed
  • Cancelled Bookings: Bookings that were cancelled
  • Upcoming Bookings: Bookings scheduled for the next 7 days

KOT Statistics

  • Today's KOTs: Total Kitchen Order Tickets created today
  • Pending KOTs: Orders awaiting preparation
  • Preparing KOTs: Orders currently being prepared
  • Ready KOTs: Orders ready for service
  • Completed KOTs: Orders that have been served

Table Statistics

  • Total Tables: Total number of active tables in your restaurant
  • Available Tables: Tables currently available for booking
  • Occupied Tables: Tables currently in use
  • Reserved Tables: Tables reserved for future bookings
  • Booked Tables: Tables with active bookings
  • Maintenance Tables: Tables in maintenance mode

Revenue Statistics

  • Today's Revenue: Total revenue generated today from restaurant operations
  • Revenue Breakdown: Revenue by payment method or source

Quick Actions

The Restaurant Overview provides quick access to common actions:

  • Create Booking: Quickly create a new table booking
  • Create KOT: Create a new Kitchen Order Ticket
  • View Calendar: Access the booking calendar view
  • Table Status: View detailed table status information
  • Reports: Access restaurant reports

Multi-branch Support

If you have multiple branches:

  • Select the branch from the branch dropdown
  • Overview metrics will update based on the selected branch
  • Compare performance across branches by switching between them

Real-time Updates

The Restaurant Overview updates in real-time to reflect:

  • New bookings and cancellations
  • KOT status changes
  • Table status updates
  • Revenue transactions

Drill-down Capabilities

Click on any metric card to:

  • View detailed information about that metric
  • Access related reports
  • Filter and analyze data

Best Practices

  • Review the overview at the start of each day to plan operations
  • Monitor pending bookings and KOTs regularly throughout the day
  • Use quick actions for efficient workflow
  • Check table availability before creating new bookings
  • Compare metrics across different branches to identify best practices

Next Steps

After reviewing the Restaurant Overview, you can:

  • View Table Status for detailed table information
  • Access Booking Calendar to manage reservations
  • Review KOT Management for order tracking
  • Check Restaurant Statistics for detailed analytics
