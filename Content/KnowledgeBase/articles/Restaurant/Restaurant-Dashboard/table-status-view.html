Table Status View

The Table Status View in EquiBillBook's Restaurant Dashboard provides a real-time visual representation of all tables in your restaurant, their current status, and quick access to table management functions. This guide explains how to use the Table Status View effectively.

Accessing Table Status View

To access the Table Status View:

  1. Navigate to Restaurant â†’ Restaurant Dashboard
  2. Click on the Table Status tab or section
  3. Select your branch and floor (if applicable)

Understanding Table Status Colors

Tables are color-coded based on their current status:

  • Green: Available - Table is free and ready for booking
  • Red: Occupied - Table is currently in use
  • Yellow/Orange: Reserved - Table is reserved for a future booking
  • Blue: Booked - Table has an active booking
  • Gray: Maintenance - Table is unavailable due to maintenance

Floor Plan View

The Table Status View displays tables in a floor plan layout:

  • Visual Layout: Tables are positioned according to your floor plan configuration
  • Table Numbers: Each table displays its table number or name
  • Capacity Display: Table capacity (seating) is shown on each table
  • Status Indicators: Color coding and icons indicate current status

Table Information Display

Clicking on a table shows detailed information:

  • Table Details: Table number, type, capacity, floor
  • Current Status: Real-time status of the table
  • Active Booking: Details of any active booking
  • Active KOT: Current Kitchen Order Ticket information
  • Waiter Assignment: Assigned waiter (if any)
  • Customer Information: Details of customers currently at the table

Filtering and Sorting

Use filters to view specific tables:

  • By Status: Filter tables by status (Available, Occupied, Reserved, etc.)
  • By Floor: View tables on specific floors
  • By Table Type: Filter by table type (VIP, Normal, Outdoor)
  • By Capacity: Filter tables by seating capacity
  • Search: Search for specific tables by number or name

Quick Actions from Table Status View

Right-click or click on a table to access quick actions:

  • Create Booking: Create a booking for the selected table
  • View Booking: View details of active booking
  • Create KOT: Create a Kitchen Order Ticket for the table
  • Change Status: Manually change table status
  • Assign Waiter: Assign or change waiter assignment
  • Maintenance Mode: Set table to maintenance mode
  • View History: View table booking and usage history

Real-time Status Updates

The Table Status View updates automatically when:

  • New bookings are created or cancelled
  • Customers are seated or tables are cleared
  • KOTs are created or completed
  • Table status is manually updated
  • Maintenance mode is enabled or disabled

Multi-floor Navigation

For restaurants with multiple floors:

  • Use the floor selector to switch between floors
  • View floor-specific statistics
  • Compare table availability across floors
  • See aggregate statistics for all floors

Statistics Summary

The Table Status View displays summary statistics:

  • Total tables count
  • Tables by status (Available, Occupied, Reserved, etc.)
  • Occupancy percentage
  • Utilization rate

Table Status Management

You can manually update table status when needed:

  1. Select the table you want to update
  2. Choose the new status from the status options
  3. Confirm the status change
  4. The table status will update immediately in the view

Best Practices

  • Monitor table status regularly throughout service hours
  • Update table status promptly when customers are seated or leave
  • Use the visual floor plan to optimize table assignments
  • Check availability before accepting walk-in customers
  • Review table utilization statistics to identify peak times
  • Use maintenance mode for tables that need repairs or cleaning

Tips for Efficiency

  • Bookmark frequently used filters for quick access
  • Use keyboard shortcuts for common actions (if available)
  • Enable real-time notifications for status changes
  • Train staff on how to update table status accurately
  • Regularly review table utilization to optimize seating arrangements
