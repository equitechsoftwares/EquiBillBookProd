Multiple Table Booking

EquiBillBook allows you to book multiple tables for the same reservation, which is essential for large groups, events, or parties. This guide explains how to create and manage multiple table bookings.

What are Multiple Table Bookings?

Multiple table bookings allow you to reserve several tables for a single reservation. This is useful when:

  • Accommodating large groups that require multiple tables
  • Booking adjacent tables for parties or events
  • Combining tables to create a larger seating area
  • Managing group reservations that need multiple tables

Creating Multiple Table Bookings

To create a booking with multiple tables:

  1. Navigate to Restaurant â†’ Table Booking
  2. Click "Create New Booking"
  3. Fill in the basic booking information (customer, date, time, number of guests)
  4. In the Table Selection section, select multiple tables
  5. Ensure the total capacity of selected tables matches or exceeds the number of guests
  6. Review table availability for all selected tables
  7. Save the booking

Table Selection for Multiple Tables

When selecting multiple tables:

  • Table Capacity: The system shows the combined capacity of all selected tables
  • Availability Check: All selected tables must be available for the booking date and time
  • Adjacent Tables: Consider selecting tables that are close to each other for better service
  • Table Layout: Review the floor plan to ensure selected tables work well together

Capacity Management

Managing capacity for multiple table bookings:

  • Total Capacity: Check that the combined capacity of selected tables accommodates all guests
  • Guest Count: Enter the total number of guests accurately
  • Validation: The system validates that guest count doesn't exceed total table capacity
  • Flexibility: You can adjust table selection if capacity requirements change

Managing Multiple Table Bookings

After creating a multiple table booking:

  • View All Tables: The booking shows all assigned tables in the details
  • Table Status: All tables in the booking are marked as booked/reserved
  • Duration: All tables are reserved for the same duration
  • Updates: You can modify table selection if needed before the booking time

Best Practices

  • Verify that all selected tables are available before confirming the booking
  • Select tables that are physically close to each other for better service
  • Ensure total table capacity comfortably accommodates the number of guests
  • Communicate table arrangement to service staff in advance
  • Consider table layout and customer preferences when selecting multiple tables
  • Document any special arrangements or table preferences in special requests

Use Cases

Common scenarios for multiple table bookings:

  • Large Groups: Birthday parties, corporate events, or family gatherings
  • Adjacent Tables: Combining tables for a larger party
  • Events: Private events or celebrations requiring multiple tables
  • VIP Reservations: Special accommodations for VIP customers

Next Steps

After creating a multiple table booking:

  • Review the booking in the calendar view to see all assigned tables
  • Assign waiters to ensure proper service coverage
  • Update booking status when customers arrive
  • Create KOT (Kitchen Order Tickets) when customers place orders
  • Manage payments and deposits for the booking
