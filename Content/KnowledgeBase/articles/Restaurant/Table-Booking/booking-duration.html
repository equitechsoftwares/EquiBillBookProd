Booking Duration

Setting appropriate booking durations in EquiBillBook helps manage table turnover and optimize restaurant operations. This guide explains how to set and manage booking durations effectively.

What is Booking Duration?

Booking duration is the expected length of time customers will occupy their reserved tables. Setting accurate durations helps with table allocation, planning service schedules, and managing table availability.

Setting Booking Duration

To set the duration for a booking:

  1. Navigate to the booking creation or editing page
  2. Find the Duration field (usually in minutes)
  3. Enter the expected duration (e.g., 120 for 2 hours)
  4. The system calculates the end time based on booking time and duration
  5. Save the booking

Default Booking Duration

Configure default duration in Restaurant Settings:

  • Default Duration: Set a standard duration for all bookings (e.g., 90 minutes, 2 hours)
  • Apply Default: New bookings will use the default duration automatically
  • Override: You can change duration for individual bookings as needed
  • Flexibility: Adjust duration based on booking type or customer requirements

Duration Guidelines

Consider these factors when setting duration:

  • Meal Type: Lunch bookings typically shorter than dinner bookings
  • Group Size: Larger groups may need more time
  • Occasion: Special events may require extended durations
  • Restaurant Policy: Follow your restaurant's table turnover policy
  • Peak Hours: Shorter durations during peak hours to maximize turnover

Common Duration Settings

Typical duration settings:

  • Quick Service: 60-90 minutes
  • Standard Dining: 90-120 minutes (1.5-2 hours)
  • Fine Dining: 120-180 minutes (2-3 hours)
  • Events/Parties: 180+ minutes (3+ hours)
  • Business Lunch: 60-90 minutes

Calculating End Time

The system automatically calculates booking end time:

  • Start Time: The booking start time you select
  • Duration: The duration you enter (in minutes)
  • End Time: System calculates start time + duration
  • Display: End time is shown in booking details and calendar

Extended Bookings

For bookings that need more time:

  • Extended Duration: Increase duration if customers request more time
  • Special Events: Set longer durations for events or celebrations
  • VIP Bookings: Allow flexible durations for VIP customers
  • Update: Modify duration if customers request extension

Duration and Table Availability

How duration affects table availability:

  • Blocked Time: Tables are blocked for the entire duration
  • Availability Check: System checks availability for start time through end time
  • Conflicts: Prevents overlapping bookings for the same table
  • Turnover: Accurate durations help plan table turnover

Modifying Duration

To change booking duration:

  1. Open the booking details
  2. Update the duration field
  3. The system will recalculate the end time
  4. Check for availability conflicts with the new duration
  5. Save the changes

Best Practices

  • Set realistic durations based on your restaurant's typical service time
  • Use default durations for consistency
  • Adjust durations for special occasions or events
  • Communicate duration policies to customers if applicable
  • Review and adjust durations based on actual customer behavior
  • Consider peak hours when setting durations
  • Be flexible with durations for VIP or special customers

Duration Reports

Analyze booking durations:

  • Average Duration: Calculate average booking duration
  • Duration by Time: See how durations vary by time of day
  • Duration by Group Size: Analyze duration based on number of guests
  • Turnover Analysis: Use duration data to optimize table turnover

Next Steps

After setting up booking durations:

  • Configure default duration in Restaurant Settings
  • Train staff on duration guidelines
  • Review duration settings regularly
  • Adjust defaults based on actual usage patterns
  • Use duration data to optimize operations
