Creating Table Bookings

Table bookings are essential for managing restaurant reservations in EquiBillBook. This guide will walk you through the process of creating table bookings, including selecting tables, customers, dates, times, and configuring all booking details.

What are Table Bookings?

Table bookings in EquiBillBook allow you to reserve tables for customers at specific dates and times. Each booking contains information like customer details, table selection, booking time, duration, number of guests, and special requirements.

Accessing the Table Booking Module

To create a new table booking:

  1. Navigate to Restaurant from the main menu
  2. Click on Table Booking or Bookings
  3. Click the "Create New" or "New Booking" button

Basic Booking Information

Fill in the essential booking details:

  • Customer: Select an existing customer from the dropdown or create a new customer
  • Booking Date: Select the date for the reservation
  • Booking Time: Select the time for the reservation
  • Number of Guests: Enter the total number of guests for the booking
  • Duration: Set the expected duration of the booking (in minutes)
  • Booking Type: Select whether this is a Walk-in, Phone, Online, or other booking type

Table Selection

Select the tables for the booking:

  • Select Tables: Choose one or more tables from the available tables
  • Table Capacity: Ensure the selected tables can accommodate the number of guests
  • Table Availability: The system will check if tables are available for the selected date and time
  • Multiple Tables: You can select multiple tables for large groups or events

Additional Booking Details

Configure additional booking options:

  • Status: Set the booking status (Pending, Confirmed, Seated, Completed, Cancelled)
  • Waiter Assignment: Assign a waiter to the booking (optional)
  • Deposit Amount: Enter the deposit amount if required
  • Special Requests: Add any special requests, dietary requirements, or notes

Saving the Booking

After filling in all necessary details:

  1. Review all the information entered
  2. Check table availability for the selected date and time
  3. Click "Save" or "Create Booking"
  4. The booking will be saved and appear in the booking calendar

Best Practices

  • Always verify table availability before creating bookings
  • Set appropriate booking durations based on your restaurant's typical dining times
  • Assign waiters when creating bookings to improve service management
  • Record special requests to ensure customer satisfaction
  • Collect deposits for high-value or special occasion bookings
  • Update booking statuses promptly as customers arrive and are seated

Next Steps

After creating a booking, you can:

  • View the booking in the calendar view
  • Update booking status as the reservation progresses
  • Convert the booking to a sales invoice when customers order
  • Manage deposits and payments
  • Create KOT (Kitchen Order Tickets) from the booking
