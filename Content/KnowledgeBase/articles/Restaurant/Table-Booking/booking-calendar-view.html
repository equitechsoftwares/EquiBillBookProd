Booking Calendar View

The booking calendar view in EquiBillBook provides a comprehensive visual interface to view, manage, and navigate all table bookings. This guide will help you understand and use the calendar view effectively.

What is the Booking Calendar View?

The booking calendar view displays all table bookings in a calendar format, allowing you to see bookings by date, time, and status. You can navigate through dates, view booking details, and manage reservations directly from the calendar.

Accessing the Calendar View

To access the booking calendar:

  1. Navigate to Restaurant from the main menu
  2. Click on Table Booking or Bookings
  3. Select "Calendar View" or "Calendar" tab

Calendar View Features

The calendar view includes several features:

  • Date Navigation: Use arrows or date picker to navigate between days, weeks, or months
  • Time Slots: View bookings organized by time slots throughout the day
  • Table Columns: See bookings organized by table (if table view is enabled)
  • Color Coding: Different colors represent different booking statuses
  • Booking Details: Hover or click on bookings to view detailed information

Viewing Options

You can customize the calendar view:

  • Day View: See all bookings for a specific day
  • Week View: View bookings for the entire week
  • Month View: Get an overview of bookings for the month
  • Table View: View bookings organized by table
  • Status Filter: Filter bookings by status (Pending, Confirmed, Seated, etc.)

Booking Information Display

Each booking in the calendar shows:

  • Customer Name: The name of the customer
  • Time: Booking start time and duration
  • Number of Guests: Total number of guests
  • Table Numbers: Assigned tables
  • Status: Current booking status
  • Special Indicators: Icons for deposits, special requests, etc.

Managing Bookings from Calendar

You can perform various actions directly from the calendar:

  • View Details: Click on a booking to view full details
  • Edit Booking: Double-click or use context menu to edit booking
  • Change Status: Update booking status quickly
  • Cancel Booking: Cancel bookings directly from the calendar
  • Create New Booking: Click on a time slot to create a new booking

Search and Filter

Use search and filter options to find specific bookings:

  • Date Range: Filter bookings by date range
  • Customer Search: Search bookings by customer name
  • Table Filter: Show bookings for specific tables
  • Status Filter: Filter by booking status
  • Booking Type: Filter by booking type (Walk-in, Phone, Online)

Best Practices

  • Use the calendar view to get an overview of daily reservations
  • Check for overlapping bookings or potential conflicts
  • Use color coding to quickly identify booking statuses
  • Regularly update booking statuses as customers arrive
  • Use filters to focus on specific tables or time periods
  • Review the calendar at the start of each day to plan table assignments

Next Steps

After viewing bookings in the calendar, you can:

  • Edit booking details if changes are needed
  • Update booking statuses as reservations progress
  • Assign or reassign tables if needed
  • Create new bookings for available time slots
  • Export booking data for reporting purposes
