Restaurant Features Activation

EquiBillBook's Restaurant Module includes various features that can be activated based on your business needs. This guide will help you understand and activate the restaurant features you require.

Available Restaurant Features

The Restaurant Module includes the following features:

  • Table Booking: Manage customer reservations and table assignments
  • KOT System: Create and manage Kitchen Order Tickets
  • Kitchen Display System: Real-time order display for kitchen staff
  • Public Booking Portal: Allow customers to book tables online
  • QR Code Integration: Generate QR codes for tables and menus
  • Waiter Assignment: Assign waiters to tables and bookings
  • Deposit Management: Handle booking deposits and refunds
  • Recurring Bookings: Support for regular/recurring table reservations

Accessing Feature Activation

To activate restaurant features:

  1. Navigate to Settings → Restaurant Settings
  2. Click on Features or Feature Activation
  3. You'll see a list of available restaurant features

Activating Table Booking

Enable table booking functionality:

  1. Find Table Booking in the features list
  2. Toggle the switch to Enable
  3. Configure booking settings:
    • Advance booking period
    • Booking duration limits
    • Confirmation requirements
  4. Click Save

Activating KOT System

Enable Kitchen Order Ticket system:

  1. Find KOT System in the features list
  2. Toggle to Enable
  3. Configure KOT settings:
    • KOT numbering format
    • Auto-print options
    • Kitchen station assignment
  4. Set up Kitchen Stations if needed
  5. Click Save

Activating Kitchen Display System

Enable real-time kitchen display:

  1. Find Kitchen Display System in the features list
  2. Toggle to Enable
  3. Configure display settings:
    • Display refresh rate
    • Station filtering
    • Order timer settings
  4. Set up display devices if applicable
  5. Click Save

Activating Public Booking Portal

Enable online booking for customers:

  1. Find Public Booking Portal in the features list
  2. Toggle to Enable
  3. Configure portal settings:
    • Booking URL
    • Customer registration requirements
    • Payment options
    • Table visibility
  4. Customize the booking portal appearance
  5. Click Save

Activating QR Code Integration

Enable QR code functionality:

  1. Find QR Code Integration in the features list
  2. Toggle to Enable
  3. Configure QR code settings:
    • QR code format and design
    • Actions when scanned (view menu, place order, etc.)
    • Generate QR codes for tables
  4. Generate QR codes for existing tables
  5. Click Save

Activating Waiter Assignment

Enable waiter assignment to tables:

  1. Find Waiter Assignment in the features list
  2. Toggle to Enable
  3. Configure assignment settings:
    • Automatic or manual assignment
    • Assignment rules
    • Waiter capacity limits
  4. Click Save

Activating Deposit Management

Enable deposit handling for bookings:

  1. Find Deposit Management in the features list
  2. Toggle to Enable
  3. Configure deposit settings:
    • Deposit amount (fixed or percentage)
    • Payment methods
    • Refund policy
    • Deposit account
  4. Click Save

Activating Recurring Bookings

Enable support for recurring reservations:

  1. Find Recurring Bookings in the features list
  2. Toggle to Enable
  3. Configure recurring booking settings:
    • Recurrence patterns (daily, weekly, monthly)
    • End date options
    • Auto-confirmation settings
  4. Click Save

Feature Dependencies

Some features depend on others:

  • Kitchen Display System requires KOT System to be enabled
  • QR Code Integration works best with Public Booking Portal
  • Waiter Assignment enhances Table Booking functionality
  • Deposit Management integrates with Table Booking

Deactivating Features

To disable a feature:

  1. Navigate to Restaurant Settings → Features
  2. Find the feature you want to disable
  3. Toggle the switch to Disable
  4. Confirm the action
  5. Note: Disabling a feature will hide it but won't delete existing data

Best Practices

  • Activate only the features you need to avoid complexity
  • Start with core features (Table Booking, KOT) and add others gradually
  • Test each feature after activation before using it in production
  • Train staff on activated features before going live
  • Review feature dependencies before activation
  • Monitor feature usage and deactivate unused features

Testing Activated Features

After activating features, test them:

  • Create a test booking to verify Table Booking works
  • Place a test order to verify KOT system
  • Check Kitchen Display System shows orders correctly
  • Test Public Booking Portal from a customer perspective
  • Scan QR codes to verify they work as expected

Next Steps

After activating restaurant features:

  • Create Restaurant Floors
  • Set up Tables on each floor
  • Configure Table Types if needed
  • Set up Kitchen Stations if KOT is enabled
  • Test all activated features with sample data
  • Train your team on using the features
