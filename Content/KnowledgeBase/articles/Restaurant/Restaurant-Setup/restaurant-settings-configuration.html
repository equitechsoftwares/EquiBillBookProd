Restaurant Settings Configuration

Restaurant Settings allow you to configure all aspects of your restaurant operations in EquiBillBook. This comprehensive guide will help you set up booking preferences, deposit management, notifications, and other restaurant-specific configurations.

Accessing Restaurant Settings

To configure Restaurant Settings:

  1. Navigate to Settings from the main menu
  2. Click on Restaurant Settings or Restaurant Configuration
  3. You'll see various sections for different configuration options

Booking Settings

Configure how table bookings work:

  • Advance Booking Period: Set how far in advance customers can book (e.g., 30 days, 60 days)
  • Minimum Booking Duration: Set the minimum time for a booking (e.g., 1 hour, 2 hours)
  • Maximum Booking Duration: Set the maximum time allowed for a booking
  • Booking Confirmation: Enable/disable automatic confirmation or require manual approval
  • Allow Multiple Table Booking: Enable customers to book multiple tables in one booking
  • Booking Cancellation Policy: Set rules for booking cancellations and refunds

Deposit Management

Configure deposit requirements and handling:

  • Deposit Required: Enable or disable deposit requirements for bookings
  • Deposit Amount: Set fixed deposit amount or percentage of estimated bill
  • Deposit Payment Methods: Specify which payment methods accept deposits
  • Deposit Refund Policy: Configure automatic or manual deposit refunds
  • Deposit Account: Select the account where deposits are recorded

Notification Settings

Set up notifications for restaurant operations:

  • Booking Notifications: Enable email/SMS notifications for new bookings
  • Reminder Notifications: Send reminders to customers before their booking time
  • KOT Notifications: Notify kitchen staff when new orders are placed
  • Table Status Alerts: Get alerts when tables are occupied or available
  • Low Stock Alerts: Receive notifications for low inventory items

Public Booking Portal Settings

Configure the public booking portal:

  • Enable Public Booking: Allow customers to book tables through the public portal
  • Booking URL: Set the custom URL for your booking portal
  • Require Customer Registration: Require customers to create accounts before booking
  • Show Available Tables: Display available tables on the booking portal
  • Allow Online Payment: Enable payment processing through the booking portal

QR Code Settings

Configure QR code functionality:

  • Enable Table QR Codes: Generate QR codes for each table
  • QR Code Actions: Configure what happens when customers scan QR codes (view menu, place order, etc.)
  • QR Code Format: Choose QR code design and format

Operating Hours Configuration

Set up your restaurant's operating schedule:

  • Default Operating Hours: Set standard hours for each day of the week
  • Special Hours: Configure different hours for holidays or special events
  • Date Overrides: Set specific dates with different operating hours
  • Break Times: Configure break periods during the day

KOT Settings

Configure Kitchen Order Ticket settings:

  • Auto-print KOT: Automatically print KOT when orders are placed
  • KOT Numbering: Set up KOT numbering sequence and prefixes
  • Kitchen Display: Enable real-time kitchen display system
  • Station Assignment: Configure automatic or manual kitchen station assignment
  • KOT Status Tracking: Enable item-level status tracking in KOT

Table Management Settings

Configure table-related settings:

  • Table Status Updates: Automatic or manual table status changes
  • Table Capacity Rules: Set rules for table capacity and overbooking
  • Table Maintenance Mode: Configure how maintenance mode works for tables
  • Table Assignment: Set rules for automatic waiter assignment to tables

Saving Settings

After configuring all settings:

  1. Review all changes you've made
  2. Click Save or Update Settings
  3. The system will confirm that settings have been saved
  4. Settings will take effect immediately

Best Practices

  • Configure settings before creating floors and tables
  • Test booking settings with a sample booking before going live
  • Review notification settings to avoid overwhelming customers or staff
  • Set realistic advance booking periods based on your restaurant's demand
  • Configure deposit settings that balance customer convenience and business protection
  • Regularly review and update operating hours as your business evolves
