Floor Management

Effective floor management is crucial for organizing your restaurant operations. This guide covers how to manage restaurant floors, including editing details, organizing tables, viewing floor status, and maintaining floor information.

Accessing Floor Management

To manage restaurant floors:

  1. Navigate to Restaurant from the main menu
  2. Click on Floors or Restaurant Floors
  3. You'll see a list of all floors with their details

Viewing Floor List

The floor list displays:

  • Floor Name: Name of the floor
  • Floor Code: Short code identifier
  • Number of Tables: Count of tables on the floor
  • Status: Active or Inactive
  • Capacity: Total capacity of the floor
  • Actions: Edit, View, or Delete options

To modify floor information:

  2. Find the floor you want to edit
  3. Click on the floor name or the Edit icon
  4. Modify any of the following:
    • Floor Name
    • Floor Code
    • Description
    • Capacity
    • Status (Active/Inactive)
  5. Click Save to apply changes

Viewing Floor Details

To see detailed information about a floor:

  1. Click on a floor name from the list
  2. View the floor details page showing:
    • Complete floor information
    • List of tables on the floor
    • Table status summary
    • Booking statistics

Organizing Tables on Floors

Manage tables assigned to each floor:

  1. Go to the floor details page
  2. View the Tables section
  3. See all tables assigned to this floor
  4. You can:
    • Add new tables to the floor
    • Move tables to different floors
    • View table status (Available, Occupied, Reserved, etc.)
    • Edit table details

Viewing Floor Status

Monitor floor status and availability:

  • Active Floors: Operational and accepting bookings
  • Inactive Floors: Temporarily unavailable
  • Floor Utilization: See how many tables are occupied vs. available
  • Booking Status: View current and upcoming bookings for the floor

    • Changing Floor Status

    To activate or deactivate a floor:

    1. Go to the floor you want to modify
    2. Click Edit
    3. Change the Status:
      • Select Active to make the floor operational
      • Select Inactive to temporarily disable the floor
    4. Click Save
    5. Note: Inactive floors won't appear in booking options

    Floor Statistics and Reports

    View analytics for each floor:

    • Table Utilization: Percentage of tables in use
    • Booking Count: Number of bookings on the floor
    • Revenue by Floor: Sales performance per floor
    • Peak Hours: Busiest times for the floor

    Bulk Floor Operations

    Perform operations on multiple floors:

    • Bulk Status Change: Activate or deactivate multiple floors at once
    • Export Floor Data: Export floor information to Excel/CSV
    • Floor Reordering: Change the display order of floors

      • Searching and Filtering Floors

      Find floors quickly:

      • Use the Search box to find floors by name or code
      • Filter by Status (Active/Inactive)
      • Sort floors by name, number of tables, or status

      Deleting Floors

      To delete a floor:

      1. Go to the Floors list
      2. Find the floor you want to delete
      3. Click Delete or the delete icon
      4. Confirm the deletion
      5. Important: You can only delete floors with no tables. Move or delete tables first.

      Floor Maintenance

      Regular maintenance tasks:

      • Review floor information periodically
      • Update capacity as your restaurant layout changes
      • Keep floor descriptions current
      • Monitor floor utilization and adjust as needed
      • Archive or deactivate unused floors instead of deleting them

      Best Practices

      • Regularly review and update floor information
      • Keep floor names and codes consistent
      • Monitor floor utilization to optimize table distribution
      • Use inactive status for maintenance periods
      • Maintain accurate capacity information
      • Document any special characteristics in floor descriptions
      • Review floor statistics regularly to identify trends

      Common Tasks

      Renaming a Floor:

      1. Edit the floor
      2. Change the Floor Name
      3. Save changes

      Temporarily Closing a Floor:

      1. Edit the floor
      2. Change status to Inactive
      3. Add a note in description about closure reason
      4. Save changes

      Viewing Floor Performance:

      1. Go to floor details
      2. View statistics and reports
      3. Analyze utilization and booking patterns
