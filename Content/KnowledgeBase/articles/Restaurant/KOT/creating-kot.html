Creating KOT

A Kitchen Order Ticket (KOT) is a document used in restaurants to communicate customer orders from the service staff to the kitchen. In EquiBillBook, KOTs help you manage restaurant orders efficiently, track order preparation, and coordinate between the front-of-house and kitchen staff.

What is a KOT?

A KOT (Kitchen Order Ticket) contains:

  • Order Information: KOT number, table number, order time
  • Items: List of food items ordered with quantities
  • Special Instructions: Customer preferences and special requests
  • Kitchen Stations: Assignment of items to specific kitchen stations
  • Order Status: Current status of the order (Pending, Preparing, Ready, Served, Cancelled)

Accessing KOT Creation

To create a new KOT:

  1. Navigate to Restaurant from the main menu
  2. Click on KOT or Kitchen Order Ticket
  3. Click the "Create New" or "Add KOT" button

Basic KOT Information

Fill in the essential details:

  • Table (Optional): Select a table if this is a dine-in order
  • Booking (Optional): Link to an existing table booking
  • Order Type: Select Dine-In or Takeaway
  • Guest Count: Enter the number of guests (for dine-in orders)
  • Waiter: Assign a waiter to handle this order

Adding Items to KOT

To add items to your KOT:

  1. Click "Add Item" or search for items in the item selection area
  2. Select items from your item list
  3. Enter Quantity for each item
  4. Review item details including price and description
  5. Items will be added to the KOT details list

Kitchen Station Assignment

You can assign items to specific kitchen stations:

  • Select items that need to be prepared at specific stations
  • Assign stations based on item categories or preparation requirements
  • Multiple items can be assigned to the same station
  • Station assignment helps route orders to the correct kitchen area

Special Instructions

Add special instructions for the order:

  • Click on Special Instructions field
  • Enter customer preferences, dietary requirements, or preparation notes
  • Instructions will be displayed on the KOT for kitchen staff
  • Item-level instructions can also be added for specific items

Saving the KOT

After adding all items and details:

  1. Review all information entered
  2. Verify item quantities and station assignments
  3. Click "Save" or "Create KOT"
  4. The KOT will be created with status "Pending"
  5. KOT number will be automatically generated

KOT Status

After creation, KOT status changes through these stages:

  • Pending: KOT is created but not yet sent to kitchen
  • Printed: KOT has been printed
  • Preparing: Kitchen is preparing the order
  • Ready: Order is ready for service
  • Served: Order has been served to customers
  • Cancelled: Order has been cancelled

Best Practices

  • Ensure all items are correctly added before saving
  • Double-check quantities to avoid order errors
  • Add clear special instructions for kitchen staff
  • Assign stations appropriately based on item preparation needs
  • Verify table assignment for dine-in orders
  • Review KOT before printing or sending to kitchen

Next Steps

After creating a KOT, you can:

  • Print the KOT for kitchen staff
  • Track order status as it progresses
  • Update item-level status for detailed tracking
  • Create sales invoice from the KOT
  • View KOT on kitchen display system
