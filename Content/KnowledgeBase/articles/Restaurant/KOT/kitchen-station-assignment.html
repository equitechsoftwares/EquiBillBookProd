Kitchen Station Assignment

Kitchen station assignment allows you to route KOT items to specific preparation areas in your kitchen. This feature helps organize kitchen workflow, improve efficiency, and ensure items are prepared at the correct stations.

What are Kitchen Stations?

Kitchen stations are specific areas or sections in your kitchen where certain types of items are prepared:

  • Hot Kitchen: For hot food items, grills, fryers
  • Cold Kitchen: For salads, cold appetizers
  • Beverage Station: For drinks, cocktails, beverages
  • Dessert Station: For desserts and sweet items
  • Bakery Station: For bread, pastries, baked items
  • Pizza Station: For pizza preparation

Benefits of Station Assignment

Assigning items to stations provides:

  • Better Organization: Items routed to correct preparation areas
  • Improved Efficiency: Kitchen staff knows where items need to be prepared
  • Clear Workflow: Each station sees only relevant items
  • Better Tracking: Monitor preparation time by station
  • Kitchen Display: Stations can filter items on kitchen display screens

Assigning Stations During KOT Creation

When creating or editing a KOT:

  1. Add items to the KOT
  2. For each item, select the Kitchen Station from the dropdown
  3. Choose the appropriate station based on item type
  4. Multiple items can be assigned to the same station
  5. Save the KOT with station assignments

Automatic Station Assignment

Stations can be automatically assigned based on:

  • Item Categories: Items in specific categories assigned to stations
  • Item Settings: Default station for items
  • Station Type Mapping: Category-to-station relationships

Viewing Station Assignments

You can view station assignments:

  • In the KOT details view, see which station each item is assigned to
  • On kitchen display system, filter items by station
  • In KOT reports, analyze items by station
  • In station performance reports

Multiple Stations per KOT

A single KOT can have items assigned to different stations:

  • Different items can go to different stations
  • Each item is routed to its assigned station
  • Kitchen display can show items grouped by station
  • Stations work on their assigned items independently

Changing Station Assignments

You can modify station assignments:

  • Edit the KOT and change item station assignments
  • Reassign items to different stations if needed
  • Changes update in real-time on kitchen display
  • Station changes are logged for tracking

Station Filtering on Kitchen Display

Kitchen display system can filter by station:

  • Each station screen shows only items assigned to that station
  • Staff see only relevant orders for their area
  • Reduces confusion and improves focus
  • Multiple displays can show different stations

Best Practices

  • Set up stations before creating KOTs
  • Assign stations consistently based on item types
  • Use automatic assignment when possible for efficiency
  • Review station assignments before printing KOT
  • Ensure all items have appropriate station assignments
  • Train staff on station organization and workflow

Troubleshooting

Station not showing in dropdown?

  • Verify the station is created and active
  • Check station configuration in Kitchen Stations
  • Ensure stations are enabled in restaurant settings

Items not routing correctly?

  • Review item-to-station assignments
  • Check category-to-station mappings
  • Verify item categories are set correctly
