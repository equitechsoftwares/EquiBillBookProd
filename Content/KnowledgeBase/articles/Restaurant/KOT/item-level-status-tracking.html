Item-level Status Tracking

Item-level status tracking allows you to track the preparation status of individual items within a KOT order. This feature provides detailed visibility into order preparation, helping coordinate kitchen operations and provide accurate updates to customers.

What is Item-level Status Tracking?

Item-level status tracking enables:

  • Tracking status of each item independently within an order
  • Updating individual items as they are prepared
  • Better coordination when items are prepared at different stations
  • Accurate order progress reporting
  • Improved customer communication about specific items

Benefits of Item-level Tracking

Tracking items individually provides:

  • Granular Control: Know exactly which items are ready
  • Better Coordination: Serve items as they become ready
  • Accurate Updates: Provide precise information to customers
  • Station Efficiency: Track preparation by station
  • Detailed Reporting: Analyze item preparation times

Item Status Options

Each item can have its own status:

  • Pending: Item not yet started
  • Preparing: Item is being prepared
  • Ready: Item is ready for service
  • Served: Item has been served
  • Cancelled: Item has been cancelled

Viewing Item Status

Item status is visible in:

  • KOT Details View: Each item shows its status
  • Kitchen Display: Items displayed with individual status
  • KOT List: Item status summary
  • Reports: Item-level status analysis

Updating Item Status

To update an item's status:

  1. Open the KOT containing the item
  2. Locate the item in the item list
  3. Click on the item's status indicator
  4. Select the new status from the dropdown
  5. Save the changes
  6. Status updates immediately across the system

Bulk Item Status Updates

You can update multiple items at once:

  • Select multiple items in the KOT
  • Choose "Update Status" for selected items
  • Set status for all selected items
  • Useful when multiple items are ready together

Item Status vs Order Status

Understanding the relationship:

  • Order Status: Overall KOT status
  • Item Status: Individual item status within the order
  • Order status can be "Preparing" while items have different statuses
  • Order status may update based on item statuses

Kitchen Display with Item Status

Kitchen display systems show:

  • Each item with its individual status
  • Visual indicators for item status
  • Items grouped by status for easy viewing
  • Real-time updates as items change status

Partial Order Service

Item-level tracking enables:

  • Serving items as they become ready
  • Partial order fulfillment
  • Better customer service for long orders
  • Flexibility in service timing

Status Synchronization

The system handles status synchronization:

  • Order status may update based on item statuses
  • When all items are ready, order status can update to "Ready"
  • When all items are served, order status can update to "Served"
  • Manual order status updates override item status aggregation

Best Practices

  • Update item status promptly as items are prepared
  • Use item status for accurate order tracking
  • Coordinate item status updates with kitchen stations
  • Train staff on item-level status management
  • Use item status for partial order service
  • Review item status reports for efficiency improvements

Troubleshooting

Item status not updating?

  • Verify you have permissions to update item status
  • Check if KOT allows item status updates
  • Ensure item is not in a locked state
  • Refresh the KOT view

Item status not syncing with order status?

  • Check status synchronization settings
  • Verify all items have appropriate status
  • Review order status update rules

Next Steps

After implementing item-level status tracking:

  • Train kitchen staff on item status updates
  • Set up procedures for item status management
  • Monitor item status transitions
  • Analyze item preparation times
  • Optimize workflow based on item status data
