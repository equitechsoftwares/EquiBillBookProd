KOT from Booking

Creating a KOT from an existing table booking streamlines your restaurant operations by linking customer reservations directly to kitchen orders. This feature allows you to convert a table booking into a KOT, maintaining the connection between booking and order.

What is KOT from Booking?

When you create a KOT from a booking:

  • The KOT is automatically linked to the table booking
  • Table and customer information is pre-filled from the booking
  • Guest count and table details are inherited from the booking
  • Booking and KOT remain connected for tracking and reporting

When to Use KOT from Booking

Create KOT from booking when:

  • Customers have a confirmed table booking
  • You want to track orders against specific reservations
  • You need to maintain booking-to-order relationship
  • You want to streamline order creation for booked tables

Accessing Booking Details

To create a KOT from a booking:

  1. Navigate to Restaurant → Table Booking
  2. Find and open the booking you want to convert
  3. Click on "Create KOT" or "Generate KOT" button
  4. Or go to Restaurant → KOT → Create from Booking

Booking Information Pre-filled

When creating KOT from booking, the following information is automatically populated:

  • Booking ID: Linked to the original booking
  • Table: Tables assigned to the booking
  • Customer: Customer information from the booking
  • Guest Count: Number of guests from the booking
  • Booking Date/Time: Reserved date and time

Adding Items to KOT

After the booking information is loaded:

  1. Click "Add Item" to select items from your menu
  2. Enter quantities for each item
  3. Add any special instructions or dietary requirements
  4. Assign items to kitchen stations if needed
  5. Review all items before saving

Saving KOT from Booking

To complete the KOT creation:

  1. Verify all booking information is correct
  2. Review all items and quantities
  3. Check special instructions
  4. Click "Save" or "Create KOT"
  5. The KOT will be created and linked to the booking

Booking-KOT Relationship

After creating KOT from booking:

  • The booking status may update to reflect the order
  • Both booking and KOT can be viewed together
  • Reports can show booking-to-KOT conversions
  • The relationship is maintained throughout the order lifecycle

Multiple KOTs from One Booking

You can create multiple KOTs from a single booking:

  • Useful for additional orders or separate courses
  • Each KOT maintains its own status and items
  • All KOTs remain linked to the original booking
  • Helpful for tracking multiple orders per booking

Best Practices

  • Create KOT from booking when customers arrive for their reservation
  • Verify booking details before creating KOT
  • Ensure table assignment matches the booking
  • Add items promptly after creating KOT from booking
  • Keep booking and KOT status synchronized
  • Review linked bookings when viewing KOT details

Troubleshooting

Cannot create KOT from booking?

  • Ensure the booking status allows KOT creation
  • Check that tables are assigned to the booking
  • Verify booking is not cancelled or completed
  • Confirm you have permissions to create KOTs

Booking information not showing?

  • Refresh the booking details page
  • Verify booking ID is correct
  • Check that booking exists and is active

Next Steps

  • Track order status and preparation
  • Update booking status as order progresses
  • Create sales invoice when order is completed
  • Monitor booking-KOT relationship in reports
