Order Status Tracking

Order status tracking allows you to monitor the progress of KOT orders from creation to completion. Understanding and managing order statuses helps coordinate between front-of-house and kitchen staff, ensuring timely service and better customer experience.

KOT Order Statuses

KOT orders progress through the following statuses:

  • Pending: KOT is created but not yet sent to kitchen
  • Printed: KOT has been printed and sent to kitchen
  • Preparing: Kitchen is actively preparing the order
  • Ready: Order is prepared and ready for service
  • Served: Order has been served to customers
  • Cancelled: Order has been cancelled

Understanding Status Workflow

The typical order workflow is:

  1. Pending: Order created, awaiting action
  2. Printed: KOT printed and sent to kitchen
  3. Preparing: Kitchen starts preparation
  4. Ready: Items are ready for service
  5. Served: Items delivered to customers

Viewing Order Status

You can view order status in several ways:

  • KOT List: Status column shows current status
  • KOT Details: Status displayed prominently
  • Kitchen Display: Real-time status on screens
  • Dashboard: Status summary and statistics
  • Reports: Status analysis and trends

Updating Order Status

To update KOT order status:

  1. Open the KOT you want to update
  2. Click on Status or Update Status
  3. Select the new status from the dropdown
  4. Confirm the status change
  5. Status updates immediately across the system

Status Transition Rules

Status changes follow logical workflow:

  • Pending → Printed → Preparing → Ready → Served
  • Any status can transition to Cancelled
  • Some statuses may restrict editing (e.g., Ready, Served)
  • Status changes are logged for audit purposes

Automatic Status Updates

Some status updates can be automatic:

  • Status may update when KOT is printed
  • Kitchen display actions can trigger status changes
  • Time-based status updates (if configured)
  • Integration with POS or service systems

Status-Based Actions

Different statuses enable different actions:

  • Pending/Printed: Can edit, print, cancel
  • Preparing: Can update to Ready, view details
  • Ready: Can mark as Served, view details
  • Served: Can view, create invoice, view reports
  • Cancelled: View only, cannot modify

Status Tracking Benefits

Tracking order status provides:

  • Visibility: Everyone knows order progress
  • Coordination: Better communication between staff
  • Efficiency: Identify bottlenecks and delays
  • Customer Service: Inform customers about order status
  • Reporting: Analyze preparation times and efficiency

Status Notifications

Status changes can trigger notifications:

  • Kitchen display updates in real-time
  • Front-of-house staff notified when orders are ready
  • Dashboard updates with status changes
  • Reports reflect current status

Best Practices

  • Update status promptly as order progresses
  • Train staff on status workflow and importance
  • Use consistent status update procedures
  • Monitor status transitions for process improvement
  • Review status reports regularly
  • Ensure all staff understand status meanings

Troubleshooting

Cannot update status?

  • Check if status transition is allowed
  • Verify you have permissions to update status
  • Ensure KOT is not in a locked state
  • Check if status workflow rules allow the change

Status not updating on display?

  • Refresh the kitchen display screen
  • Check WebSocket connection for real-time updates
  • Verify status was saved successfully
