Real-time Order Display

The Real-time Order Display feature in EquiBillBook's Kitchen Display System shows orders from KOTs (Kitchen Order Tickets) instantly as they are created, providing kitchen staff with immediate visibility of new orders and updates.

What is Real-time Order Display?

Real-time Order Display automatically shows new orders on the Kitchen Display screen as soon as they are created from KOTs. The system updates the display automatically without requiring manual refresh, ensuring kitchen staff always see the latest orders.

How Real-time Display Works

The Real-time Order Display operates by:

  • Automatic Updates: The display refreshes automatically at set intervals
  • Order Appearance: New orders appear on the screen as soon as KOTs are created
  • Status Changes: Order status updates are reflected immediately
  • Visual Indicators: New orders are highlighted or marked to draw attention

Display Features

The Real-time Order Display shows the following information for each order:

  • Order Number: KOT number or order identifier
  • Table Information: Table number and customer details
  • Items Ordered: List of items with quantities
  • Special Instructions: Any special requests or modifications
  • Order Time: When the order was placed
  • Elapsed Time: How long the order has been in preparation
  • Order Status: Current status (Pending, Preparing, Ready, etc.)
  • Station Assignment: Which kitchen station should prepare the items

Auto-refresh Configuration

Configure auto-refresh settings for optimal real-time updates:

  1. Access Kitchen Display settings or configuration
  2. Set the refresh interval (e.g., every 5, 10, 15, or 30 seconds)
  3. Choose between automatic refresh or manual refresh mode
  4. Enable sound alerts for new orders (optional)

Order Organization

Orders on the display can be organized by:

  • Time: Chronological order (oldest first or newest first)
  • Station: Grouped by kitchen station
  • Status: Grouped by order status
  • Priority: VIP orders or high-priority items first
  • Table: Organized by table number

Visual Indicators

The display uses visual indicators to help kitchen staff identify:

  • New Orders: Highlighted or marked to draw attention
  • Urgent Orders: Special highlighting for rush orders
  • Order Age: Color coding based on how long orders have been waiting
  • Status Colors: Different colors for Pending, Preparing, Ready, Served
  • Timer Warnings: Visual alerts when orders exceed expected preparation time

Multi-station View

When viewing multiple stations:

  • Orders are grouped by assigned kitchen station
  • Each station section shows relevant orders
  • Staff can filter to view specific stations
  • Station-specific views help organize kitchen workflow

Display Layout Options

Choose a layout that works best for your kitchen:

  • Grid Layout: Orders displayed in a grid format
  • List Layout: Orders in a vertical list
  • Card Layout: Orders shown as individual cards
  • Full-screen Mode: Maximized view for better visibility

Best Practices

  • Set refresh interval based on order volume (more frequent during busy periods)
  • Use large, clear displays for better readability
  • Position displays at strategic locations in the kitchen
  • Train staff to regularly check the display for new orders
  • Use visual indicators effectively to prioritize orders
  • Keep the display clean and visible at all times
  • Monitor display performance and adjust settings as needed

Troubleshooting

If orders are not displaying in real-time:

  • Check Refresh Settings: Verify auto-refresh is enabled
  • Manual Refresh: Try refreshing the page manually (F5 or browser refresh)
  • Check KOT Creation: Verify that KOTs are being created successfully
  • Station Assignment: Ensure items in KOTs are assigned to stations
  • Browser Issues: Clear browser cache or try a different browser
  • Internet Connection: Verify stable internet connection for updates
