Status Updates

Status Updates in EquiBillBook's Kitchen Display System allow kitchen staff to update the status of orders and individual items, tracking the progress of order preparation from creation to serving. This helps coordinate kitchen workflow and keep all staff informed about order status.

What are Status Updates?

Status Updates are changes made to the status of orders or items in the Kitchen Display System. They indicate the current stage of order preparation, helping kitchen staff, servers, and managers track order progress in real-time.

Order Status Types

The Kitchen Display System supports various order statuses:

  • Pending: Order received but not yet started
  • Preparing: Order is currently being prepared
  • Ready: Order is ready for serving
  • Served: Order has been served to the customer
  • Cancelled: Order has been cancelled

Item-Level Status Updates

In addition to order-level status, items can have individual statuses:

  • Not Started: Item has not begun preparation
  • In Progress: Item is currently being prepared
  • Ready: Item is ready but order is not complete
  • Completed: Item is finished and ready to serve

Updating Order Status

To update the status of an order:

  1. On the Kitchen Display, locate the order you want to update
  2. Click on the order card or order details
  3. Look for status buttons or status dropdown menu
  4. Select the new status (e.g., "Preparing", "Ready", "Served")
  5. Confirm the status change
  6. The status update will be reflected immediately on the display

Updating Item Status

To update the status of individual items:

  1. Open the order details view
  2. Locate the specific item within the order
  3. Click on the item or item status button
  4. Select the new item status
  5. The item status will update, and order status may update accordingly

Status Update Methods

Status can be updated through various methods:

  • Kitchen Display Screen: Touch or click to update status directly
  • Tablet Interface: Use tablets for easy status updates
  • KOT Management: Update status from the KOT management page
  • Mobile App: Update status using mobile devices (if available)

Real-time Status Sync

Status updates are synchronized in real-time:

  • Immediate Updates: Status changes appear instantly on all displays
  • Multi-user View: All staff see the same status information
  • Server Notifications: Servers are notified when orders are ready
  • Manager Visibility: Managers can monitor order status remotely

Status Workflow

Typical order status workflow:

  1. Pending: Order is created and appears on the display
  2. Preparing: Kitchen staff starts working on the order
  3. Ready: All items are prepared and ready to serve
  4. Served: Server delivers the order to the customer

Benefits of Status Updates

Status Updates provide several advantages:

  • Better Coordination: All staff know the current state of each order
  • Improved Communication: Reduces verbal communication needs
  • Efficient Workflow: Helps prioritize and organize kitchen work
  • Customer Service: Servers know exactly when orders are ready
  • Performance Tracking: Monitor preparation times and efficiency

Status Update Permissions

Different users may have different permissions:

  • Kitchen Staff: Can update status to "Preparing" and "Ready"
  • Servers: Can update status to "Served"
  • Managers: Can update any status and view status history
  • System: Automatically sets initial status when orders are created

Best Practices

  • Update status promptly as work progresses
  • Use consistent status updates across all staff
  • Train staff on proper status update procedures
  • Update item status when working on individual items
  • Mark orders as "Ready" only when all items are complete
  • Update to "Served" immediately after delivery to customer
  • Review status patterns to identify workflow improvements

Status Notifications

Status updates may trigger notifications:

  • Ready Notifications: Alert servers when orders are ready
  • Delay Alerts: Notify managers of orders taking too long
  • Status Change Log: Record all status changes for reporting

Troubleshooting

If Status Updates are not working correctly:

  • Status Not Changing: Check user permissions for status updates
  • Updates Not Syncing: Verify internet connection and refresh display
  • Status Options Missing: Check status configuration settings
  • Delayed Updates: Ensure auto-refresh is enabled on displays
