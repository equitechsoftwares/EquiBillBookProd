Station Filtering

Station Filtering in EquiBillBook's Kitchen Display System allows you to filter and display orders based on assigned kitchen stations. This helps organize orders and ensures kitchen staff only see the items relevant to their station, improving efficiency and reducing confusion.

What is Station Filtering?

Station Filtering is a feature that displays only the orders assigned to a specific kitchen station. When you select a station, the Kitchen Display System shows only the items that should be prepared at that particular station, helping kitchen staff focus on their specific tasks.

How Station Filtering Works

Station Filtering operates based on:

  • Station Assignment: Items in KOTs are assigned to specific kitchen stations
  • Category Mapping: Stations are linked to item categories
  • Filter Selection: You can select one or more stations to view
  • Dynamic Updates: Filtered view updates automatically as new orders come in

Filtering Options

Station Filtering offers various options:

  • Single Station View: View orders for one specific station
  • Multiple Station View: Select multiple stations to view simultaneously
  • All Stations: View orders for all stations (default view)
  • Station Groups: Group related stations together for viewing

Setting Up Station Filtering

Before using Station Filtering, ensure:

  1. Stations are Created: All kitchen stations are set up in the system
  2. Categories are Assigned: Stations are linked to appropriate item categories
  3. KOTs Assign Stations: KOTs properly assign items to stations based on categories
  4. Stations are Active: Stations are set to active status

Using Station Filters

To filter orders by station:

  1. On the Kitchen Display page, locate the station filter control
  2. Click on the station dropdown or station selection buttons
  3. Select the desired station(s) from the list
  4. The display will refresh to show only orders for selected stations
  5. Orders are organized by the selected station(s)

Benefits of Station Filtering

Station Filtering provides several advantages:

  • Reduced Clutter: Staff only see relevant orders for their station
  • Improved Focus: Better concentration on station-specific tasks
  • Better Organization: Clear separation of work by station
  • Increased Efficiency: Faster order processing with focused views
  • Reduced Errors: Less confusion about which items belong to which station

Multi-Station Display

When multiple stations are selected:

  • Orders are grouped by station within the display
  • Each station section is clearly labeled
  • Orders within each station are organized chronologically or by status
  • Staff can easily identify which station should handle each order

Default Filter Settings

Configure default station filter settings:

  • Default View: Set which stations are shown by default
  • Save Preferences: Save frequently used filter combinations
  • Auto-apply Filters: Automatically apply filters based on user or location

Best Practices

  • Set up stations logically based on your kitchen workflow
  • Assign item categories accurately to stations
  • Train staff on which station should handle which items
  • Use station filtering to create focused work areas
  • Monitor filter usage to optimize station assignments
  • Ensure all items in KOTs are properly assigned to stations
  • Review and adjust station assignments regularly based on workload

Troubleshooting

If Station Filtering is not working correctly:

  • No Orders Showing: Verify that KOT items are assigned to stations
  • Wrong Station Assignments: Check category-to-station mappings
  • Missing Stations: Ensure all stations are created and active
  • Filter Not Applying: Refresh the page or clear browser cache
  • Items in Wrong Station: Review item category assignments in station configuration
