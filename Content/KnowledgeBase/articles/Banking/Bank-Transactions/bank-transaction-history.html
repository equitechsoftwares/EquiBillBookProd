Bank Transaction History

Bank transaction history allows you to view, search, and manage all banking transactions recorded in EquiBillBook. This guide will help you effectively navigate and utilize the bank transaction history feature.

Understanding Bank Transaction History

Bank transaction history provides:

  • Complete record of all bank transactions
  • Search and filter capabilities
  • Transaction details and audit trail
  • Export and reporting options
  • Transaction editing and management
  • Reconciliation status tracking

Accessing Bank Transaction History

To view bank transaction history:

  1. Navigate to the Banking module from the main menu
  2. Click on Transaction History, Bank Transactions, or All Transactions
  3. Select the Bank Account (or view all accounts)

Viewing Transaction History

Transaction List View

The transaction list displays:

  • Date: Transaction date
  • Type: Transaction type (Deposit, Withdrawal, Transfer)
  • Description: Transaction description or narration
  • Reference: Reference number (cheque number, transaction ID)
  • Debit: Withdrawal or outgoing amount
  • Credit: Deposit or incoming amount
  • Balance: Running balance after transaction
  • Status: Reconciliation status (Matched, Unmatched, Reconciled)

Transaction Details View

Click on a transaction to view details:

  • Full transaction information
  • Account information
  • Related transactions (if any)
  • Attachments or documents
  • Reconciliation status
  • Edit and delete options

Filtering Transactions

Date Range Filter

Filter by date range:

  • Select From Date and To Date
  • Use predefined ranges (Today, This Week, This Month, This Year)
  • Custom date range selection
  • Apply filter to view transactions in the selected period

Transaction Type Filter

Filter by transaction type:

  • All Types: View all transactions
  • Deposits: Only deposit transactions
  • Withdrawals: Only withdrawal transactions
  • Transfers: Only transfer/contra transactions

Other Filters

Additional filtering options:

  • Bank Account: Filter by specific account
  • Payment Method: Filter by payment method (Cash, Cheque, Online, etc.)
  • Amount Range: Filter by transaction amount
  • Reconciliation Status: Filter by matched/unmatched/reconciled
  • Reference Number: Search by reference number
  • Description: Search by description keywords

Searching Transactions

Quick Search

Use the search box to:

  • Search by description or narration
  • Search by reference number
  • Search by amount
  • Search by payee or customer name

Advanced Search

For complex searches:

  • Combine multiple search criteria
  • Use date ranges with other filters
  • Search across multiple accounts
  • Save search criteria for future use

Managing Transactions

Viewing Transaction Details

To view full transaction details:

  • Click on any transaction in the list
  • View all transaction information
  • See related transactions or linked records
  • View attachments or documents

Editing Transactions

To edit a transaction:

  1. Click on the transaction to view details
  2. Click "Edit" or "Modify"
  3. Make necessary changes
  4. Save the updated transaction
  5. Note: Editing reconciled transactions may require unreconciliation

Deleting Transactions

To delete a transaction:

  • Click on the transaction
  • Click "Delete" or "Remove"
  • Confirm the deletion
  • Note: Deletion may be restricted for reconciled or linked transactions

Voiding Transactions

For transactions that shouldn't be deleted:

  • Use "Void" option instead of delete
  • Voided transactions remain in history but are marked as void
  • Voided transactions don't affect account balances
  • Maintains audit trail while correcting errors

Transaction Status and Reconciliation

Reconciliation Status

Transactions show reconciliation status:

  • Unmatched: Not yet matched in reconciliation
  • Matched: Matched during reconciliation process
  • Reconciled: Finalized in completed reconciliation

Viewing Reconciliation Details

  • See which reconciliation a transaction belongs to
  • View reconciliation date
  • Check reconciliation notes

Exporting Transaction History

Export Options

Export transactions in various formats:

  • Excel (XLS/XLSX): For analysis and reporting
  • CSV: For data import to other systems
  • PDF: For printing and archiving

Export Process

  1. Apply filters to select transactions to export
  2. Click "Export" or "Download"
  3. Select export format
  4. Choose columns to include in export
  5. Download the exported file

Printing Transaction History

To print transaction history:

  • Apply filters to select transactions to print
  • Click "Print" or "Print Preview"
  • Review the print preview
  • Adjust print settings if needed
  • Print or save as PDF

Transaction Reports

Generate reports from transaction history:

  • Daily Transaction Report: Transactions for a specific day
  • Monthly Transaction Report: Transactions for a month
  • Summary Report: Totals and summaries
  • Reconciliation Report: Reconciliation status report

Best Practices

  • Review transaction history regularly for accuracy
  • Use filters to focus on specific periods or types
  • Search for specific transactions when needed
  • Export transaction history for backup and analysis
  • Keep transaction history for audit purposes
  • Document any corrections or adjustments made
  • Use reconciliation status to track matched transactions
  • Review unmatched transactions regularly

Troubleshooting

Transactions not appearing:

  • Check date range filters
  • Verify account selection
  • Clear all filters and search again
  • Check if transactions are deleted or voided

Cannot edit transaction:

  • Check if transaction is reconciled (may need to unreconcile first)
  • Verify you have edit permissions
  • Check if transaction is linked to other records

Search not finding transactions:

  • Check spelling and keywords
  • Try different search terms
  • Use partial matches
  • Clear search and use filters instead
