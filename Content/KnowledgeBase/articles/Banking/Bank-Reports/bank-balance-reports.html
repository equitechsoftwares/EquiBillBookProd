Bank Balance Reports

Bank Balance Reports provide detailed information about bank account balances at specific dates, balance trends over time, and cash position analysis in EquiBillBook. This guide will help you generate and use these reports effectively.

Understanding Bank Balance Reports

Bank Balance Reports help you:

  • View account balances at specific dates
  • Track balance trends over time
  • Analyze cash position and liquidity
  • Compare balances across accounts
  • Monitor balance changes
  • Generate balance reports for accounting and auditing

Accessing Bank Balance Reports

To generate bank balance reports:

  1. Navigate to the Banking module from the main menu
  2. Click on Reports â†’ Bank Balance Reports or Balance Reports
  3. Select report type and parameters
  4. Click "Generate Report"

Types of Balance Reports

1. Current Balance Report

Current balances for all accounts:

  • Current balance for each account
  • Account information
  • Total balances summary
  • Balance as of today or selected date

2. Balance as of Date Report

Balances at a specific date:

  • Select any historical date
  • View balances as they were on that date
  • Useful for historical analysis
  • Compare balances across different dates

3. Balance Trend Report

Balance changes over time:

  • Balance trends over selected period
  • Daily, weekly, or monthly balance snapshots
  • Visual representation of balance changes
  • Identify patterns and trends

4. Comparative Balance Report

Compare balances across periods:

  • Compare balances between two dates
  • Show balance changes
  • Percentage change calculations
  • Period-over-period comparison

5. Cash Position Report

Overall cash position analysis:

  • Total bank balances
  • Total credit card balances
  • Net cash position
  • Available cash for operations

Report Parameters and Filters

Date Selection

Select the date for balance:

  • Current Date: Today's balance
  • Specific Date: Select any date
  • End of Period: Month-end, quarter-end, year-end
  • Date Range: For trend reports

Account Selection

Filter by bank accounts:

  • All Accounts: Include all bank accounts
  • Specific Account: Select one or more accounts
  • Account Type: Filter by Bank or Credit Card
  • By Bank: Filter by bank name
  • Active Only: Only active accounts

Balance Type Selection

Select balance type to report:

  • Current Balance: Available balance
  • Book Balance: Balance per books
  • Statement Balance: Balance per bank statement
  • Cleared Balance: Cleared transactions only

Report Format and Layout

Report Structure

The report typically includes:

  • Header: Report title, date, account information
  • Summary Section: Total balances and key metrics
  • Detail Section: Individual account balances
  • Footer: Report generation date, notes

Column Options

Select columns to include:

  • Account Name
  • Bank Name
  • Account Number
  • Account Type
  • Opening Balance
  • Current Balance
  • Balance Change
  • Currency
  • Last Transaction Date

Grouping Options

Group accounts by:

  • Bank Name
  • Account Type
  • Currency (if multi-currency)
  • Branch (if multi-branch)

Generating and Viewing Reports

Generate Report

  1. Select report type
  2. Choose date or date range
  3. Select bank account(s)
  4. Apply filters
  5. Select columns and grouping
  6. Click "Generate Report"
  7. Review the generated report

Balance Summary View

The summary shows:

  • Total bank account balances
  • Total credit card balances
  • Net bank balance (bank minus credit card)
  • Number of accounts included
  • Average balance per account

Analyze cash position:

Exporting Reports

Export Formats

Export balance reports in:

  • Excel (XLS/XLSX): For analysis and calculations
  • PDF: For printing and sharing
  • CSV: For data import to other systems

Export Process

  1. Generate and preview the report
  2. Click "Export" or "Download"
  3. Select export format
  4. Choose export options
  5. Download the exported file

Printing Reports

To print balance reports:

  • Generate and preview the report
  • Click "Print" or "Print Preview"
  • Review print preview
  • Adjust print settings
  • Print or save as PDF

Troubleshooting

Balance not matching expected:

  • Check if all transactions are recorded
  • Verify the selected date
  • Review recent transactions for errors
  • Check if reconciliation is needed
  • Verify balance type selected (current vs book balance)

Account not appearing in report:

  • Check if account is active
  • Verify account filters applied
  • Check user permissions for account access
  • Verify account has transactions

Incorrect totals:

  • Verify all accounts are included
  • Check currency conversion if multi-currency
  • Review individual account balances
  • Verify balance calculations
